Executive Summary

Gilded Cage Design currently exhibits several critical cybersecurity vulnerabilities and control gaps. A recent network scan revealed Remote Desktop Protocol (RDP) exposed on 10.10.10.51, complementing an existing high-severity risk entry for RDP exposure on 10.10.10.50. This indicates a systemic issue with external access management across multiple assets. Furthermore, critical policy and training deficiencies were identified: the absence of an employee acceptable use policy and a lack of security awareness training for new hires. While the organization has implemented Multi-Factor Authentication (MFA) for key systems, these other identified gaps significantly increase the overall attack surface and the risk of successful cyberattacks. Immediate attention to RDP hardening and foundational security policy implementation is strongly recommended to enhance the organization’s defensive posture.

Recommendations

To address the identified vulnerabilities and enhance Gilded Cage Design’s cybersecurity posture, the following recommendations are provided:

  1. Secure Remote Desktop Protocol (RDP) Access:

    • For 10.10.10.51 and 10.10.10.50: Implement a VPN for all RDP access, ensuring RDP ports are not directly exposed to the internet.

    • Enforce strong, complex passwords and Multi-Factor Authentication (MFA) for all RDP user accounts.

    • Restrict RDP access to specific source IP addresses (IP Whitelisting) where VPN is not feasible.

    • Regularly review RDP logs for anomalous activity.

    • Patch systems with RDP enabled promptly to mitigate known vulnerabilities.

    • Evaluate the necessity of RDP on these systems; disable it if not required.

  2. Develop and Implement an Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that clearly outlines employee responsibilities regarding the use of company IT resources, data handling, and internet usage.

    • Ensure all employees read, understand, and sign the AUP annually.

    • Integrate AUP review into annual security awareness training.

  3. Establish Mandatory Security Awareness Training for New Employees:

    • Develop and implement a mandatory security awareness training program for all new hires during their onboarding process.

    • This training should cover topics such as phishing, social engineering, password best practices, data protection, and the organization’s AUP.

    • Regular refresher training (at least annually) should also be mandatory for all employees, as currently implemented.

  4. Holistic Vulnerability Management Program:

    • Implement a regular vulnerability scanning and penetration testing schedule across the entire IT infrastructure.

    • Establish a formal process for tracking, prioritizing, and remediating identified vulnerabilities.

    • Continuously monitor for new threats and vulnerabilities, especially those targeting commonly exposed services like RDP.
