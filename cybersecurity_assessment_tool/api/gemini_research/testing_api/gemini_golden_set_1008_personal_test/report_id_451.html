Recommendations

The following recommendations are provided to mitigate the identified risks and enhance the overall cybersecurity posture of Blackwood Industries.

  • Implement MFA for Email Access (Critical): Immediately deploy and enforce Multi-Factor Authentication (MFA) for all organizational email accounts. This significantly reduces the risk of account compromise from phishing or stolen credentials, a common initial access vector for attackers.

  • Implement MFA for Sensitive Data Systems (High): Mandate MFA for all access to systems containing sensitive organizational or customer data. This adds a crucial layer of security, protecting critical information assets from unauthorized access even if primary credentials are breached.

  • Develop and Enforce an Acceptable Use Policy (High): Create a comprehensive Acceptable Use Policy (AUP) that clearly defines expectations for employee behavior regarding company IT assets, data handling, and internet usage. Ensure all employees review, understand, and acknowledge the AUP, ideally on an annual basis, to foster a culture of responsible technology use.

  • Secure SSH Service (Medium):

    • Review the configuration of the SSH service on .

    • Ensure strong, unique passwords are used, or preferably, enforce key-based authentication exclusively.

    • Disable root login over SSH to prevent direct attacks on the highest privilege account.

    • Implement rate limiting and IP whitelisting where possible to restrict access attempts.

    • Keep the SSH server software updated to the latest stable version to patch known vulnerabilities.

    • Consider changing the default SSH port (22) to a non-standard port; while not a security control, it can reduce noise from automated scans targeting common ports.

  • Continue Security Awareness Training (Good Practice): Maintain the current excellent practice of providing security awareness training for new employees and annual refreshers for all staff. Consider incorporating specific modules on recognizing advanced phishing attempts and the critical importance of MFA.
