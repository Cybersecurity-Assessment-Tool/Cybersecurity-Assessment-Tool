Recommendations

To address the identified risks and enhance the overall cybersecurity posture of Skyward Bound, the following recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) for Email Access:

    • Enable and enforce MFA for all email accounts across the organization. This significantly reduces the risk of credential compromise leading to unauthorized access, even if passwords are stolen.

    • Educate employees on the importance and usage of MFA.

  2. Develop and Enforce an Employee Acceptable Use Policy (AUP):

    • Draft a comprehensive AUP that clearly outlines the acceptable use of all company IT resources, data, and communication systems.

    • Ensure the AUP covers data handling, internet usage, software installation, personal device usage (if applicable), and reporting security incidents.

    • Obtain acknowledgment of understanding and compliance from all employees.

  3. Implement Mandatory Security Awareness Training for New Employees:

    • Develop and integrate a mandatory security awareness training module into the onboarding process for all new hires.

    • This training should cover topics such as phishing identification, password hygiene, safe internet practices, data classification, and the company’s AUP.

    • Reinforce these concepts with regular, ongoing training for all employees, as is already being done annually.

  4. Conduct Regular External Vulnerability Scans and Penetration Tests:

    • Although the provided scan data was empty, it is crucial to routinely perform authenticated and unauthenticated vulnerability scans of all external-facing assets.

    • Consider engaging third-party cybersecurity firms for periodic penetration tests to identify potential weaknesses that automated scans might miss.

  5. Establish an Incident Response Plan (IRP):

    • Develop and document a clear Incident Response Plan to effectively prepare for, detect, respond to, and recover from cybersecurity incidents.

    • Regularly test the IRP through tabletop exercises to ensure its effectiveness and refine processes.
