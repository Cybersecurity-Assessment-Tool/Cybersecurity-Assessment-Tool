1. Executive Summary

This report details the cybersecurity posture of Blackwood Industries, derived from a network scan, an organizational security questionnaire, and pre-existing risk data. The current security posture is assessed as High Risk, requiring immediate and comprehensive remediation efforts to mitigate potential compromise and improve overall resilience against cyber threats.

6. Recommendations

Based on the analysis, the following recommendations are provided to enhance Blackwood Industriesâ€™ cybersecurity posture. These are prioritized by severity and potential impact.

  1. Secure Remote Desktop Protocol (RDP) Services (Critical Priority):

    • Restrict Access: Immediately implement strict firewall rules to limit RDP access to only necessary, trusted internal IPs. If external access is required, place RDP behind a robust Virtual Private Network (VPN) solution or a secure bastion host.

    • Mandate MFA: Implement Multi-Factor Authentication for all RDP connections, both internal and external.

    • Strong Authentication: Enforce complex, unique passwords for all RDP accounts and implement account lockout policies to thwart brute-force attacks.

    • Patch Management: Ensure all RDP servers are regularly updated with the latest security patches.

    • Auditing: Continuously monitor RDP login attempts and session activity for anomalies.

  2. Implement Multi-Factor Authentication (MFA) Across Critical Systems (Critical Priority):

    • Computer Logins: Roll out MFA for all user and administrative computer logins throughout the organization.

    • Sensitive Data Systems: Extend MFA requirements to all systems that store, process, or transmit sensitive organizational data.

  3. Develop and Enforce an Employee Acceptable Use Policy (AUP) (High Priority):

    • Policy Creation: Draft a comprehensive AUP that clearly outlines the acceptable and unacceptable use of company IT resources, data, and communication channels.

    • Dissemination & Acknowledgment: Distribute the AUP to all employees and require formal acknowledgment of understanding and compliance. Integrate it into onboarding processes.

  4. Establish and Maintain Robust Security Awareness Training (High Priority):

    • Onboarding Training: Implement mandatory security awareness training for all new employees during their onboarding process.

    • Annual Refresher Training: Conduct mandatory annual security awareness training for all employees, covering current threats such as phishing, social engineering, malware, and secure data handling practices.

    • Regular Simulations: Conduct periodic phishing and social engineering simulations to test employee vigilance and reinforce training.

  5. Implement Proactive Vulnerability Management (Ongoing):

    • Regular Scanning: Perform regular internal and external vulnerability scans to identify new exposures.

    • Patch Management Program: Establish and maintain a rigorous patch management program for all operating systems, applications, and network devices.

    • Inventory Management: Maintain an accurate inventory of all IT assets to ensure comprehensive coverage of security controls.
