Executive Summary

This report provides a comprehensive assessment of the cybersecurity posture for Iron Oak Furniture, based on a network scan, a security questionnaire, and existing risk data. The analysis identifies several significant vulnerabilities and critical control gaps that require immediate attention. Key findings include an extremely outdated FTP server (vsftpd 2.3.4) with anonymous access enabled on the internal network (10.0.0.15), and a critical lack of Multi-Factor Authentication (MFA) for computer logins. Additionally, an existing risk of outdated Windows 7 operating systems was noted. Addressing these issues is paramount to mitigate significant risks of unauthorized access, data compromise, and potential system exploitation.

Analysis of Security Controls

While many foundational controls are in place (MFA for email and sensitive data, AUP, regular security training), a significant gap was identified: the lack of Multi-Factor Authentication for computer logins. This omission dramatically increases the risk of successful phishing or credential stuffing attacks leading to internal network compromise.

Technical Analysis

The discovery of an FTP server running vsftpd 2.3.4 on 10.0.0.15 represents a critical vulnerability. vsftpd version 2.3.4 was released in 2011 and is long End-of-Life (EOL). This specific version is notoriously known for a critical backdoor vulnerability (CVE-2011-2523), which allows remote attackers to execute arbitrary commands. Recommendations

To address the identified risks and improve the overall cybersecurity posture, the following recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) for Computer Logins (High Priority):

    • Deploy and enforce MFA for all user logins to company computers, especially those accessing the internal network or sensitive resources. This significantly reduces the risk of successful credential-based attacks.

  2. Address Outdated FTP Server and Anonymous Access (Immediate Critical Action):

    • Disable Anonymous Access Immediately: Reconfigure the FTP server on 10.0.0.15 to disallow anonymous logins. Ensure that only authenticated users with strong credentials can access the service.

    • Upgrade/Replace FTP Server: vsftpd 2.3.4 is critically outdated.

      • Short-term: Upgrade vsftpd to the latest stable and supported version, ensuring all known vulnerabilities are patched.

      • Long-term: Consider replacing FTP with more secure file transfer protocols like SFTP (SSH File Transfer Protocol) or FTPS (FTP Secure), which provide encryption and better authentication mechanisms.

    • Evaluate Necessity: Determine if the FTP server is still essential for business operations. If not, it should be decommissioned and removed from the network to eliminate this attack surface.

  3. Upgrade Outdated Windows Operating Systems (High Priority):

    • Plan and execute an upgrade path for all workstations running Windows 7 to a currently supported operating system (e.g., Windows 10 or Windows 11).

    • Develop a hardware refresh strategy to ensure compatibility with newer operating systems and to support future security features.

  4. Regular Vulnerability Management:

    • Establish a continuous vulnerability scanning and patch management program to identify and remediate outdated software and misconfigurations across all systems.

    • Conduct regular penetration testing to validate security controls and uncover potential exploitation paths.
