Recommendations

To significantly enhance Stellar Pathways’ cybersecurity posture, the following recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) Widely:

    • Email Access: Immediately enforce MFA for all email accounts. This is a critical first step to protect against phishing and credential theft, which are common initial attack vectors.

    • Computer Login: Implement MFA for all user logins to company computers and endpoints. This prevents unauthorized access even if primary login credentials are compromised.

    • Sensitive Data Systems: Mandate MFA for access to all systems containing sensitive data. This provides a crucial layer of defense for critical information assets.

  2. Enhance New Employee Security Awareness Training:

    • Develop and implement a mandatory security awareness training program for all new employees as part of their onboarding process. This training should cover essential topics such as identifying phishing attempts, safe browsing habits, social engineering awareness, password hygiene, and adherence to acceptable use policies.

  3. Conduct Regular and Comprehensive Technical Scans:

    • Perform authenticated network scans and vulnerability assessments regularly to identify open ports, outdated software, misconfigurations, and other technical weaknesses that could be exploited.

    • Consider engaging in penetration testing to simulate real-world attacks and uncover exploitable vulnerabilities from an attacker’s perspective.

  4. Review and Update Security Policies:

    • Ensure that the employee acceptable use policy and other organizational security policies are regularly reviewed and updated to reflect current threat landscapes, technological changes, and best practices.
