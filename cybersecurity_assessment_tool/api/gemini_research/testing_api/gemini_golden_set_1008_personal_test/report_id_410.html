Executive Summary

This cybersecurity assessment highlights several areas requiring immediate attention for True North Travel. While some foundational controls are in place, critical gaps exist in multi-factor authentication for computer access, the establishment of an acceptable use policy, and regular security awareness training. Technical scans revealed a severely outdated FTP server allowing anonymous access, posing a significant data exposure risk. Furthermore, the organization is aware of an existing risk concerning outdated Windows operating systems. Addressing these vulnerabilities and control deficiencies is paramount to strengthening the organization’s defense against modern cyber threats.

Technical Analysis

The discovery of an FTP service (vsftpd 2.3.4) running on port 21/tcp is a significant concern. vsftpd 2.3.4 was released in 2011 and has several known critical vulnerabilities, including a backdoor vulnerability (CVE-2011-2523) that allows arbitrary command execution. This version is severely outdated and should not be used in production.

Compounding this issue, the FTP server is configured to allow anonymous logins. This means anyone can connect to the FTP server without credentials, potentially accessing or uploading files. The combination of an outdated, vulnerable service and anonymous access presents a critical risk for data breach, system compromise, and unauthorized data modification.

Recommendations

Based on the assessment, the following recommendations are provided to enhance True North Travel’s cybersecurity posture:

Security Control Enhancements

  1. Implement Multi-Factor Authentication (MFA) for Computer Login:

    • Action: Mandate and deploy MFA for all employee computer logins.

    • Benefit: Significantly reduces the risk of unauthorized access even if primary credentials are compromised.

  2. Develop and Implement an Acceptable Use Policy (AUP):

    • Action: Create a clear and comprehensive AUP outlining employee responsibilities and acceptable use of organizational IT assets. Ensure it is acknowledged by all employees.

    • Benefit: Sets clear expectations, reduces misuse, and provides a basis for disciplinary action if policies are violated.

  3. Establish Annual Security Awareness Training:

    • Action: Implement a mandatory annual security awareness training program for all employees, covering topics like phishing, password hygiene, and data handling.

    • Benefit: Educates employees on current threats, improves their ability to identify and report suspicious activities, and reinforces a security-conscious culture.

Technical Mitigation Actions

  1. Secure or Decommission Outdated FTP Service:

    • Immediate Action: Disable anonymous FTP access on immediately.

    • Short-term Action: Evaluate if FTP is truly necessary. If not, decommission the service.

    • Long-term Action: If file transfer is required, replace vsftpd 2.3.4 with a modern, secure file transfer solution (e.g., SFTP, SCP, or a cloud-based file sharing service) with strong authentication and encryption. If FTP must be used, upgrade to the latest stable and patched version of vsftpd and configure it securely.

    • Benefit: Eliminates critical vulnerabilities and unauthorized access vectors.

  2. Upgrade Outdated Windows Operating Systems:

    • Action: Develop a plan to migrate all workstations currently running Windows 7 to a currently supported operating system (e.g., Windows 10 or 11).

    • Benefit: Ensures systems receive regular security patches, reducing exposure to known vulnerabilities and improving overall system stability and performance.

Conclusion

True North Travel faces notable cybersecurity risks stemming from outdated software configurations and gaps in essential security controls. By systematically addressing the outlined recommendations, particularly those concerning MFA, security training, and the vulnerable FTP service, the organization can significantly enhance its security posture, reduce its attack surface, and protect its valuable assets from potential cyber threats. A proactive and continuous approach to cybersecurity is crucial in today’s evolving threat landscape.
