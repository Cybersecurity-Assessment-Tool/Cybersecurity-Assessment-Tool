Executive Summary

This report details the findings of a cybersecurity posture assessment for Sterling Silver. The assessment involved a review of organizational security controls via questionnaire, analysis of network scan data, and evaluation of pre-existing risk information. Key findings include critical gaps in Multi-Factor Authentication (MFA) for sensitive data systems and a lack of security awareness training for new employees. Furthermore, an open MySQL port running version 5.7.33 was identified on 172.16.50.20, aligning with an existing "Database Exposure" risk. These findings indicate areas requiring immediate attention to enhance the overall security posture and reduce potential attack vectors.

The presence of an open MySQL port (3306/tcp) running MySQL version 5.7.33 is noted. While MySQL 5.7 is still supported for critical bug fixes, it is nearing its End-of-Life (EOL) in October 2023. Exposure of this service to the network without proper access controls presents a significant attack surface, especially given its EOL status.

Recommendations

To mitigate the identified risks and enhance the overall cybersecurity posture, the following recommendations are provided:

  1. Database Exposure: Quick Fix: Firewall rules. Long Term: VPN.

  2. Lack of Multi-Factor Authentication (MFA) for Sensitive Data Systems: Implement mandatory MFA for all access to sensitive data systems. Evaluate available MFA solutions compatible with your existing systems and roll out a phased implementation.

  3. Lack of New Employee Security Awareness Training: Develop and implement a mandatory security awareness training program for all new employees, to be completed as part of their onboarding process. Cover topics such as phishing, password hygiene, data handling, and reporting suspicious activities.
