Executive Summary

This report details the findings from a cybersecurity assessment conducted for Skyward Bound. The assessment involved a review of organizational security controls via a questionnaire and a network scan of an internal host.

The primary findings indicate significant gaps in foundational security practices, particularly concerning Multi-Factor Authentication (MFA), the absence of an Acceptable Use Policy (AUP), and insufficient security awareness training. While the technical scan of the target host (192.168.1.100) did not reveal any open ports or immediately exploitable services, the absence of active threats from this specific scan does not negate the high-risk posture introduced by the policy and control deficiencies.

These identified gaps pose substantial risks to the organization, including unauthorized access, data breaches, and non-compliance with industry best practices. Immediate action is recommended to address these critical vulnerabilities to enhance the overall security posture.

Recommendations

To mitigate the identified risks and improve the overall cybersecurity posture of Skyward Bound, the following recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA):

    • Action: Immediately deploy and enforce MFA for all critical systems, including email accounts, user logins to computers and networks, and access to all sensitive data systems and applications.

    • Benefit: Significantly reduces the risk of unauthorized access even if credentials are stolen, forming a crucial layer of defense.

  2. Develop and Implement an Acceptable Use Policy (AUP):

    • Action: Create a comprehensive AUP that clearly outlines employee responsibilities regarding the use of company IT resources, data handling, and internet usage. Ensure all employees review and acknowledge the policy annually.

    • Benefit: Establishes clear guidelines, reduces misuse of resources, enhances data protection, and improves compliance.

  3. Establish Regular Security Awareness Training Programs:

    • Action: Implement mandatory annual security awareness training for all employees, supplementing the initial onboarding training. This program should be dynamic, covering current threats (e.g., phishing, ransomware), secure browsing habits, data protection best practices, and the importance of reporting suspicious activities.

    • Benefit: Enhances employee vigilance, reduces human error, and strengthens the ’human firewall’ against social engineering tactics.

  4. Conduct Regular Internal Vulnerability Scans and Penetration Tests:

    • Action: While this specific scan found no open ports, it represents a limited snapshot. Implement a schedule for regular, comprehensive internal vulnerability scans and penetration tests across the entire network infrastructure to proactively identify and address technical weaknesses in systems, applications, and configurations.

    • Benefit: Proactively identifies and remediates technical vulnerabilities before they can be exploited by malicious actors.

  5. Review and Strengthen Internal Security Policies:

    • Action: Review all existing security policies (or create them if absent) to ensure they cover critical areas such as incident response, data backup and recovery, access control, password policies, and vendor management.

    • Benefit: Creates a structured and comprehensive security framework, ensuring consistent application of security best practices throughout the organization.
