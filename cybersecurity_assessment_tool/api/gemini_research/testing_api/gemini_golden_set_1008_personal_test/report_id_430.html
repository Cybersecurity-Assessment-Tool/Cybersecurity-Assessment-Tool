6. Recommendations

To mitigate the identified risks and strengthen the cybersecurity posture of Harbor Light Foundation, the following recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) for Email Accounts (Critical):

    • Action: Mandate and enforce MFA for all email accounts across the organization.

    • Benefit: Significantly reduces the risk of account compromise from phishing or stolen credentials.

  2. Implement Multi-Factor Authentication (MFA) for Sensitive Data Systems (Critical):

    • Action: Deploy and require MFA for access to all systems containing or processing sensitive organizational data.

    • Benefit: Provides an essential layer of security, protecting critical data from unauthorized access even if primary credentials are breached.

  3. Establish Security Awareness Training for New Employees (High):

    • Action: Develop and implement a mandatory security awareness training program for all new hires during their onboarding process.

    • Benefit: Ensures new employees are aware of organizational security policies, common threats (e.g., phishing), and their role in maintaining security from day one.

  4. Implement Annual Security Awareness Training for All Employees (High):

    • Action: Conduct mandatory, regular (at least annual) security awareness training for all employees.

    • Benefit: Keeps employees updated on evolving threats, reinforces best practices, and fosters a continuous security-conscious culture.

  5. Conduct Regular Vulnerability Scanning and Penetration Testing (General):

    • Action: Implement a schedule for external and internal vulnerability scans, and consider periodic penetration testing to identify technical weaknesses.

    • Benefit: Proactively identifies technical vulnerabilities that could be exploited by attackers, providing actionable intelligence for remediation.

7. Conclusion

While Harbor Light Foundation demonstrates some foundational security practices (e.g., MFA for computer login, AUP), critical gaps exist concerning MFA for email and sensitive data, and comprehensive security awareness training. Addressing these identified high-priority items will significantly enhance the organization’s defense against common cyber threats and improve its overall resilience. Continuous monitoring and a proactive approach to cybersecurity are recommended to maintain a strong security posture in the face of evolving threats.
