Executive Summary

Recommendations

To mitigate the identified risks and improve the overall cybersecurity posture, the following recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) for Email:

    • Deploy and enforce MFA for all email accounts, preferably using strong, phishing-resistant methods (e.g., FIDO2 security keys, authenticator apps).

    • Educate employees on the importance and usage of MFA.

  2. Implement Multi-Factor Authentication (MFA) for Sensitive Data Systems:

    • Extend MFA requirements to all systems that store or process sensitive organizational or client data.

    • Ensure MFA configurations are robust and regularly reviewed.

  3. Develop and Enforce an Acceptable Use Policy (AUP):

    • Draft a clear and comprehensive AUP outlining acceptable use of organizational IT resources, data handling, and internet/email usage.

    • Ensure all employees read, understand, and acknowledge the AUP, ideally during onboarding and annually thereafter.

  4. Conduct Annual Security Awareness Training for All Employees:

    • Implement a mandatory annual security awareness training program covering topics such as phishing, social engineering, password hygiene, data handling, and incident reporting.

    • Tailor training content to be relevant to the organization’s specific risks and roles.

  5. Initiate Regular Network Vulnerability Scans:

    • Schedule and perform regular internal and external network vulnerability scans to identify and remediate technical weaknesses.

    • Integrate scan results into a continuous vulnerability management program.
