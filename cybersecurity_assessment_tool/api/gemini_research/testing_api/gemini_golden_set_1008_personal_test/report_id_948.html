6. Recommendations

Based on the analysis, the following recommendations are provided to address identified gaps and enhance the organization’s cybersecurity posture:

6.1. High Priority Recommendations

  • Implement MFA for All Computer Logins: Immediately deploy and enforce multi-factor authentication for all user logins to company computers and endpoints. This significantly reduces the risk of unauthorized access even if primary credentials are compromised.

  • Establish Mandatory New Employee Security Awareness Training: Develop and implement a mandatory cybersecurity awareness training program for all new employees as part of their onboarding process. This training should cover essential topics such as phishing, password hygiene, identifying suspicious emails, and organizational security policies.

6.2. Continuous Improvement Recommendations

  • Regular Policy Review and Update: Periodically review and update the employee acceptable use policy to ensure it remains relevant with evolving technologies and threats.

  • Enhance Existing Training: Continuously update the content of the annual security awareness training to include the latest threat intelligence and real-world examples. Consider specialized training modules for roles that handle sensitive data.

  • Expand MFA Coverage: While MFA is used for email and sensitive data, evaluate the feasibility of extending MFA to other critical internal systems and applications to create a more robust authentication posture across the board.

Implementing these recommendations will significantly strengthen Open Door’s defense mechanisms against common cyber threats and improve its overall security maturity.
