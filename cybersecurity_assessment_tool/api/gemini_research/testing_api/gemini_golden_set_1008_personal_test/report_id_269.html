Recommendations

To address the identified risks and improve the overall cybersecurity posture, the following recommendations are provided:

  1. Mitigate RDP Exposure:

    • Implement strong access controls for all RDP services. Restrict RDP access to only essential personnel and from specific, trusted source IP addresses.

    • Utilize a VPN for all remote RDP connections, ensuring RDP is not directly exposed to the internet or wide internal networks.

    • Consider alternatives to RDP, such as jump servers or bastion hosts.

    • Ensure all RDP servers are fully patched and configured with network-level authentication (NLA) enabled.

  2. Implement MFA for Sensitive Data Systems:

    • Mandate multi-factor authentication (MFA) for all sensitive data systems and applications to significantly reduce the risk of credential-based attacks.

  3. Develop and Enforce an Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that clearly defines employee responsibilities and acceptable behaviors regarding company IT resources.

    • Distribute the AUP to all employees and require acknowledgment of understanding and compliance.

  4. Establish Robust Security Awareness Training Programs:

    • Implement mandatory security awareness training for all new employees as part of their onboarding process.

    • Conduct annual security awareness training for all employees, covering topics such as phishing, social engineering, password hygiene, and incident reporting.

    • Regular simulated phishing exercises should be conducted to reinforce training.

  5. Regular Vulnerability Management:

    • Implement a regular vulnerability scanning and penetration testing program to proactively identify and remediate security weaknesses across all systems.

    • Maintain an up-to-date asset inventory to ensure all systems are accounted for in security assessments.
