1. Executive Summary

The assessment reveals a mixed security posture for Gilded Cage Design. While a recent network scan of an internal host found no open ports, indicating a potentially hardened internal system or strict firewalling, several significant gaps in organizational security controls were identified through the questionnaire. Critically, the lack of Multi-Factor Authentication (MFA) across email, computer login, and sensitive data systems poses a high risk of unauthorized access due to credential compromise. Additionally, the absence of security awareness training for new employees creates a foundational vulnerability. A previous risk entry concerning an "Unencrypted Web Server" on Port 80 exists, which contradicts the current scan results; this discrepancy requires immediate investigation. Recommendations focus on addressing these policy and control weaknesses to build a more robust defense.

6. Recommendations

Based on the assessment, the following recommendations are provided to improve Gilded Cage Design’s cybersecurity posture.

6.1. Immediate Actions

  • Implement Multi-Factor Authentication (MFA): Urgently implement MFA for all access points, including email systems, computer logins, and any sensitive data applications. This is the single most impactful immediate action to prevent credential theft.

  • Initiate New Employee Security Awareness Training: Establish mandatory security awareness training for all new hires before they gain access to company systems.

  • Verify Port 80 Status and Risk Register: Investigate the discrepancy regarding Port 80 on . Confirm if the port is truly closed or if the internal scan missed an active service. Update the risk register accordingly based on current findings. If a web server is active on another port, ensure it uses HTTPS/SSL/TLS for encryption.

6.2. Long-Term Strategies

  • Regular Security Policy Review and Updates: Periodically review and update the employee acceptable use policy and other security policies to ensure they remain relevant and effective.

  • Periodic Vulnerability Assessments: Conduct regular internal and external vulnerability scans and penetration tests to proactively identify and remediate security weaknesses.

  • Enhanced Security Awareness Program: Expand the existing annual security awareness training to cover advanced threats and specific organizational risks. Consider simulated phishing campaigns.
