Recommendations

To address the identified risks and enhance Harbor Light Foundation’s cybersecurity posture, the following recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) for Computer Login:

    • Deploy MFA solutions for all user logins to computers and workstations. This adds a crucial layer of security, making it significantly harder for attackers to gain access even if they obtain user credentials.

  2. Implement Multi-Factor Authentication (MFA) for Sensitive Data Systems:

    • Mandate MFA for all access to systems storing or processing sensitive organizational data. This is a critical control to protect against unauthorized access to high-value assets.

  3. Develop and Implement an Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that clearly outlines employee responsibilities and appropriate use of company IT resources.

    • Ensure all employees read, understand, and sign the AUP, and provide regular training on its contents.

  4. Conduct Regular Network Scans:

    • Implement a schedule for regular internal and external network vulnerability scans to identify open ports, insecure services, outdated software versions, and other technical weaknesses.

    • Promptly patch and update all operating systems and applications to their latest secure versions.

  5. Regular Security Audits:

    • Periodically engage third-party cybersecurity experts to conduct comprehensive security audits and penetration tests, providing an objective assessment of the organization’s defenses.

Conclusion

While Harbor Light Foundation demonstrates strengths in email MFA and security awareness training, critical gaps exist concerning MFA for computers and sensitive data systems, as well as the absence of an acceptable use policy. Addressing these fundamental security controls should be prioritized to mitigate significant risks. Furthermore, establishing a robust technical vulnerability management program, including regular scanning, is essential for a comprehensive security strategy.
