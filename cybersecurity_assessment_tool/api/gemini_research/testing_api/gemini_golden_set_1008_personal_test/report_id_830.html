Executive Summary

This report details the findings of a cybersecurity posture assessment for Arcane Security. The assessment involved a review of organizational security controls, an external network scan targeting 10.0.0.15, and an analysis of pre-existing risk data. Key findings include critical vulnerabilities related to an outdated FTP service with anonymous access, significant gaps in Multi-Factor Authentication (MFA) implementation, and the presence of an unsupported operating system (Windows 7) within the environment. Urgent recommendations are provided to address these high-priority risks and enhance the overall security posture.

Analysis of Security Control Gaps

The following critical gaps were identified based on the security questionnaire:

  • Lack of MFA for Email Access: Not requiring MFA for email significantly increases the risk of account takeover, phishing, and business email compromise (BEC).

  • Lack of MFA for Sensitive Data Systems: Failure to implement MFA for sensitive data systems poses a direct threat to data confidentiality and integrity.

  • Absence of Employee Acceptable Use Policy (AUP): Without an AUP, employees may not be aware of their responsibilities regarding company IT resources, leading to increased risk of misuse, data leakage, and compliance issues.

Technical Scan Results

An external network scan was conducted on the target 10.0.0.15. The scan revealed the following critical findings:

Target: 10.0.0.15

  • Open Port 21 (FTP):

    • State: Open

    • Service: FTP

    • Product: vsftpd

    • Version: 2.3.4

    • Details: The vsftpd version 2.3.4 is significantly outdated (released in 2011) and contains known critical vulnerabilities, including a notorious backdoor vulnerability (CVE-2011-2523) that allows remote code execution.

    • Configuration Issue: The scan also detected that Anonymous FTP login is allowed. This configuration permits unauthorized access to the FTP server without credentials, potentially exposing sensitive files or allowing attackers to upload malicious content, leading to data breaches or system compromise.

Recommendations

The following recommendations are provided to mitigate the identified risks and improve the overall cybersecurity posture of Arcane Security. These actions are prioritized based on their assessed severity.

  • Address Outdated FTP Service and Anonymous Access (Critical)

    • Immediate Action: Disable anonymous FTP access immediately on 10.0.0.15. Review all files and directories on the FTP server for sensitive information that may have been exposed.

    • Medium-Term: Upgrade vsftpd to the latest stable and secure version, or migrate to a more secure file transfer protocol (e.g., SFTP, SCP) if FTP functionality is still required. If FTP is no longer needed, decommission the server.

    • Long-Term: Implement robust access controls and continuous monitoring for all external-facing services to detect and prevent unauthorized access.

  • Implement Multi-Factor Authentication (MFA) (High)

    • Immediate Action: Prioritize and implement MFA for all email accounts to significantly reduce the risk of phishing and account compromise.

    • Medium-Term: Extend MFA requirements to all sensitive data systems and critical applications to protect valuable organizational data.

    • Long-Term: Explore a broader MFA rollout across the entire organization, including network and VPN access, to establish a strong identity and access management framework.

  • Develop and Implement an Acceptable Use Policy (AUP) (High)

    • Immediate Action: Draft and formally implement an AUP that clearly defines appropriate use of company IT resources, data handling procedures, and security expectations for all employees.

    • Medium-Term: Incorporate AUP review and acknowledgment into new employee onboarding processes and conduct annual refreshers during security awareness training.

    • Long-Term: Regularly review and update the AUP to align with evolving organizational needs, technological changes, and the current threat landscape.

  • Upgrade Outdated Operating Systems (High)

    • Immediate Action: Identify all workstations running Windows 7 and initiate a plan for their immediate upgrade to a currently supported operating system (e.g., Windows 10 or 11). Prioritize systems handling sensitive data.

    • Medium-Term: Implement a comprehensive patch management and lifecycle management program to ensure all systems are kept up-to-date with security patches and replaced before reaching end-of-life.

    • Long-Term: Standardize on supported operating systems and maintain an accurate inventory of all devices and their software versions to manage asset lifecycles effectively.
