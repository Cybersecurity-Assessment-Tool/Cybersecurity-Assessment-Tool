Recommendations

To enhance Urban Jungle Planning’s cybersecurity posture, the following recommendations are provided:

  1. Implement MFA for Email and Computer Login:

    • Deploy MFA solutions for all employee email accounts.

    • Enforce MFA for all user logins to company computers and internal systems.

  2. Develop and Enforce an Acceptable Use Policy (AUP):

    • Create a clear and comprehensive AUP outlining the proper use of company IT assets, data, and internet access.

    • Ensure all employees read, understand, and sign the AUP annually.

  3. Establish a Comprehensive Security Awareness Training Program:

    • Implement mandatory security awareness training for all new employees during their onboarding process.

    • Conduct annual, mandatory security awareness training for all employees, covering topics like phishing, malware, password hygiene, and data handling.

  4. Verify and Update Risk Register:

    • Investigate the discrepancy regarding the "Unencrypted Web Server" risk (Port 80). If the port remains closed, update the risk register to reflect its mitigated status.

    • If the service is intended to be available, ensure it is properly secured with HTTPS (Port 443) and strong encryption protocols.

Conclusion

Urban Jungle Planning faces significant risks primarily due to critical gaps in foundational security controls such as Multi-Factor Authentication, formal policies, and ongoing security awareness training. While the recent network scan on the specified target showed no immediately exploitable open ports, the organizational control weaknesses present substantial vectors for potential cyberattacks. Addressing these fundamental issues through the recommended actions will significantly strengthen the organization’s overall security posture.
