Recommendations

Based on the analysis, the following recommendations are provided to mitigate identified risks and enhance Firebrand Media’s overall cybersecurity posture:

  1. Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that clearly outlines expectations for employee behavior concerning company resources, data handling, internet usage, and device policies.

    • Ensure all employees read, understand, and formally acknowledge the AUP.

    • Integrate the AUP into new employee onboarding and annual review processes to ensure continuous awareness and compliance.

  2. Establish Mandatory Security Awareness Training for New Employees:

    • Implement a mandatory security awareness training program for all new hires during their onboarding process. This training should be distinct from ongoing annual training.

    • The training should cover common threats (phishing, malware), password best practices, data protection, reporting suspicious activity, and the organization’s security policies (including the new AUP).

    • Regularly update training content to reflect current threat landscapes and provide refresher modules as needed.

  3. Maintain Strong MFA Enforcement:

    • Continue and regularly review the enforcement of MFA across all critical systems, including email, computer logins, and sensitive data access.

    • Explore extending MFA to any other systems where sensitive data is accessed or high-privilege operations are performed to further minimize unauthorized access risks.

  4. Regular Network Monitoring and Scanning:

    • Although no open ports were found in this specific scan, implement regular, scheduled network vulnerability scans and penetration testing for all internet-facing assets and internal infrastructure. This proactive approach helps identify new exposures as the environment evolves.

    • Implement continuous monitoring of network traffic for anomalous behavior that could indicate compromise attempts or internal policy violations.
