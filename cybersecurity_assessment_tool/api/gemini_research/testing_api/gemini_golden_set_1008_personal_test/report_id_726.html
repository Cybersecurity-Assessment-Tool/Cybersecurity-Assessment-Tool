Executive Summary

This report details the cybersecurity posture of Skyward Bound based on a review of organizational security controls, an external network scan, and an assessment of pre-existing risks. The organization demonstrates strong foundational practices in security awareness training and acceptable use policies. However, critical gaps were identified regarding the implementation of Multi-Factor Authentication (MFA) for email access and computer logins, posing significant risks of unauthorized access. The network scan of target 192.168.0.5 revealed no open ports, which is a positive finding and potentially mitigates a previously identified "Unencrypted Web Server" risk. Recommendations focus on addressing the MFA gaps and verifying the status of pre-existing vulnerabilities.

4. Risk Assessment

This section synthesizes findings from the security control review, technical scan, and pre-existing vulnerability data to present a holistic view of the organization’s risk landscape.

Based on the security control review, the following critical gaps have been identified:

5. Recommendations

To enhance the overall cybersecurity posture of Skyward Bound, the following recommendations are provided, prioritizing the most critical findings.

  • Implement Multi-Factor Authentication (MFA) for Email Access:

    • Action: Mandate MFA for all user accounts accessing email services.

    • Benefit: Significantly reduces the risk of email account compromise, even if passwords are stolen.

  • Implement Multi-Factor Authentication (MFA) for Computer Logins:

    • Action: Deploy and enforce MFA for all user logins to company computers and endpoints.

    • Benefit: Provides an additional layer of security against unauthorized access to internal systems.

  • Verify Status of Unencrypted Web Server Risk:

    • Action: Investigate the discrepancy between the existing risk report and the current network scan. Confirm if the "Unencrypted Web Server" risk has been remediated (e.g., port 80 closed, moved to HTTPS on port 443), or if it pertains to a different asset.

    • Benefit: Ensures accurate risk reporting and confirms the effectiveness of security measures. If the server exists on another IP, ensure it uses HTTPS.

  • Maintain Security Awareness Training and Acceptable Use Policy:

    • Action: Continue robust security awareness training programs for all employees, including new hires, at least annually. Regularly review and update the acceptable use policy.

    • Benefit: Reinforces a strong security culture and educates employees on best practices, acting as a crucial human firewall.
