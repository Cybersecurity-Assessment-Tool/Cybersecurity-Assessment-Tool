Security Posture Overview

The overall security posture of Nebula Creative, as revealed by this assessment, indicates several areas of significant concern. While some critical controls like MFA for email and sensitive data access are in place, there are notable deficiencies in foundational cybersecurity practices, particularly around endpoint protection (MFA for computer logins), human element security (AUP and security awareness training), and network perimeter hardening (RDP exposure). The presence of unmitigated RDP exposure on multiple internal hosts, coupled with the lack of crucial security policies and training, collectively elevates the organization’s risk profile to a concerning level. Addressing these identified gaps is paramount for improving overall resilience against cyber threats.

Recommendations

The following recommendations are provided to address the identified risks and improve Nebula Creative’s cybersecurity posture. Immediate action is advised for Critical and High severity items.

  1. Mitigate RDP Exposure on Internal Hosts (10.10.10.50, 10.10.10.51):

    • Immediate Fix: Restrict RDP access to only essential personnel from specific, trusted IP addresses using firewall rules. Disable RDP entirely if not strictly necessary.

    • Long-Term Fix: Implement a Virtual Private Network (VPN) solution to secure RDP access, ensuring all connections are authenticated and encrypted before reaching internal RDP servers. Enforce strong, complex passwords and, ideally, MFA for RDP itself if feasible.

    • Ensure all RDP sessions are monitored and logged.

  2. Implement Multi-Factor Authentication (MFA) for Computer Logins:

    • Mandate MFA for all user logins to company computers, especially those accessing sensitive data or network resources. This significantly reduces the risk of unauthorized access even if credentials are compromised.

  3. Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Create a clear and comprehensive AUP outlining acceptable and unacceptable use of company IT resources, data, and internet access.

    • Ensure all employees read, understand, and sign the AUP upon hiring and periodically thereafter.

  4. Establish Comprehensive Security Awareness Training Programs:

    • For New Employees: Implement mandatory security awareness training as part of the onboarding process. This should cover company policies, common threats (phishing, social engineering), and safe computing practices.

    • For All Employees (Annual): Conduct mandatory annual security awareness training for all employees to reinforce concepts, update them on new threats, and maintain a strong security culture. Consider regular phishing simulations to test awareness.
