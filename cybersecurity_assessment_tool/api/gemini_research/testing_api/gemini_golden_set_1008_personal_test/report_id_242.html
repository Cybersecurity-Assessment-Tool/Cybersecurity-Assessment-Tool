Analysis of Questionnaire Findings

The review indicates strong adoption of Multi-Factor Authentication (MFA) across critical access points (email, computers, sensitive data systems), which is commendable. However, a significant gap exists with the absence of an employee acceptable use policy. This policy is fundamental for setting clear expectations for employee behavior, protecting organizational assets, and reducing legal and operational risks.

Analysis of Technical Findings

The scan revealed an active FTP service running vsftpd version 2.3.4, which is a significantly outdated and End-of-Life (EOL) version. This version is known to have critical vulnerabilities, including a backdoor (CVE-2011-2523) that could allow arbitrary command execution. More critically, the service is configured to allow anonymous FTP login. This configuration permits unauthorized users to connect to the FTP server without credentials, potentially allowing them to view, download, or upload files, leading to data breaches, unauthorized modifications, or even the hosting of malicious content.

Recommendations

The following recommendations are provided to address the identified risks and enhance the cybersecurity posture of Firebrand Media:

Immediate Actions (Critical Risks)

  • Disable Anonymous FTP & Update/Remove FTP Service:

    • Immediately disable anonymous FTP access on the server .

    • If FTP is required, upgrade vsftpd to the latest stable version or replace it with a more secure file transfer protocol (e.g., SFTP, SCP) that uses strong authentication and encryption.

    • If FTP is not required, uninstall the service entirely to remove the attack surface.

High Priority Actions

  • Develop and Implement an Acceptable Use Policy (AUP):

    • Draft a comprehensive AUP that clearly outlines employee responsibilities for using company IT resources, internet, email, and data.

    • Ensure the AUP covers security best practices, data handling, and consequences of policy violations.

    • Communicate the AUP to all employees and obtain signed acknowledgements. Integrate it into new employee onboarding.

Medium Priority Actions

  • Address Outdated Operating Systems:

    • Prioritize upgrading all workstations running Windows 7 to a currently supported operating system, such as Windows 10 or 11.

    • Implement a regular patching and update schedule for all operating systems and applications.

    • Consider a hardware refresh program to support newer OS versions where current hardware is incompatible.

Conclusion

Firebrand Media has demonstrated a good commitment to MFA implementation, which is a strong foundation for access control. However, the presence of critical vulnerabilities, particularly the anonymous FTP service on an outdated platform, coupled with fundamental policy gaps, presents significant risks. By diligently addressing the recommendations outlined in this report, Firebrand Media can significantly improve its security posture, reduce its attack surface, and better protect its assets and data.
