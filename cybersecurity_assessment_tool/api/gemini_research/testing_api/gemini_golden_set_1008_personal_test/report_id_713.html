1. Executive Summary

This report details the cybersecurity posture of Stone Arch Masonry, based on an analysis of network scan data, organizational security questionnaire responses, and pre-existing risk information. The assessment identified critical security gaps related to Remote Desktop Protocol (RDP) exposure on multiple hosts, and significant deficiencies in organizational security policies and employee training programs. While some foundational controls are in place (e.g., MFA for sensitive data systems), substantial improvements are required to mitigate high-severity risks and enhance the overall security resilience.

6. Recommendations

To address the identified risks and significantly improve Stone Arch Masonry’s overall cybersecurity posture, the following actionable recommendations are provided:

  1. Secure Remote Desktop Protocol (RDP) Exposure:

    • Immediate Action (Easy Fix): Implement strict firewall rules to restrict RDP access (port 3389) on 10.10.10.51 and 10.10.10.50 to only trusted internal IP addresses. Block all external direct RDP access.

    • Long-Term Solution: For any necessary external RDP access, mandate the use of a Virtual Private Network (VPN) for all connections. Implement Multi-Factor Authentication (MFA) for all RDP logins. Ensure all RDP accounts use strong, unique passwords and regularly patch and update RDP servers and client software.

    • Benefit: Reduces the attack surface and significantly lowers the risk of brute-force attacks, credential theft, and unauthorized access.

  2. Develop and Implement an Acceptable Use Policy (AUP):

    • Action: Draft, approve, and formally disseminate a comprehensive AUP to all employees. The AUP should clearly define expectations for appropriate technology usage, data handling, internet browsing, and email practices. Ensure all employees acknowledge understanding and compliance, preferably through a signed agreement.

    • Benefit: Establishes clear guidelines, reduces misuse of company resources, minimizes legal risks, and enhances data security by promoting responsible employee behavior.

  3. Establish Mandatory Annual Security Awareness Training Program:

    • Action: Implement a recurring program that provides mandatory security awareness training to all employees at least once per year. Training should cover crucial topics such as identifying phishing and social engineering attempts, best practices for password hygiene, safe internet browsing, incident reporting procedures, and data protection principles.

    • Benefit: Empowers employees to become the first line of defense, significantly reduces the risk of human-factor cyber incidents, and fosters a strong security culture within the organization.

7. Conclusion

Stone Arch Masonry has some foundational security controls in place, particularly regarding Multi-Factor Authentication for sensitive systems. However, the analysis highlights the critical nature of exposed RDP services and the high-severity risks stemming from gaps in essential organizational policies and a lack of consistent security awareness training. Proactively addressing these identified issues through the recommended actions is paramount to enhancing the organization’s overall resilience against the evolving landscape of cyber threats and protecting its valuable assets.
