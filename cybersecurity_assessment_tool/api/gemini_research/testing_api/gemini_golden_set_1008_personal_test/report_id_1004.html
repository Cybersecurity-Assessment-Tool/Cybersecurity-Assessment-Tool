6. Recommendations

Based on the assessment, the following recommendations are provided to enhance the organization’s cybersecurity posture:

  1. Ensure all systems containing sensitive data are protected by mandatory MFA.

  2. Establish and conduct annual, mandatory security awareness training for all employees to keep them informed about evolving threats and best practices.

  3. Implement mandatory Multi-Factor Authentication (MFA) for all email accounts to significantly reduce the risk of unauthorized access.

  4. Implement mandatory security awareness training for all new employees during their onboarding process.

  5. Mandate MFA for all computer logins, integrating with existing directory services where possible.

  6. Review the necessity of exposing Port 80 (HTTP). If required, ensure the service running is patched, hardened, and secured. Consider implementing a Web Application Firewall (WAF) or restricting access.
