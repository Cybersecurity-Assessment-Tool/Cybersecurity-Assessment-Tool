Executive Summary

This report details the findings of a cybersecurity posture assessment for True Grit. The assessment comprised a review of organizational security controls via a questionnaire, and a technical network scan of specified assets. While the organization demonstrates some positive security practices, such as requiring MFA for email and sensitive data, and conducting security awareness training, several critical gaps and high-risk vulnerabilities have been identified. Key concerns include the lack of Multi-Factor Authentication for computer logins, the absence of an Employee Acceptable Use Policy, and the presence of an outdated web server software. Addressing these areas is crucial for enhancing the overall security posture and reducing exposure to potential cyber threats.

Recommendations

To mitigate the identified risks and enhance the overall cybersecurity posture of True Grit, the following recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) for Computer Logins: Immediately deploy MFA for all employee computer logins to significantly reduce the risk of unauthorized access due to compromised credentials.

  2. Develop and Implement an Employee Acceptable Use Policy (AUP): Create a clear and comprehensive AUP that defines acceptable use of organizational IT resources and employee responsibilities. Ensure all employees acknowledge and understand this policy.

  3. Update Nginx Web Server: Upgrade the Nginx web server on 192.168.10.5 to the latest stable version (e.g., 1.24.x or later) to mitigate known vulnerabilities. Thorough testing should precede deployment in a production environment.

  4. Conduct Regular Vulnerability Scanning and Penetration Testing: Schedule periodic external and internal vulnerability scans and penetration tests to proactively identify and address security weaknesses.

  5. Enhance Security Awareness Training: While training is performed, ensure it covers the latest threat landscape, phishing techniques, social engineering, and emphasizes the importance of new policy implementations (e.g., AUP) and technologies (e.g., MFA).
