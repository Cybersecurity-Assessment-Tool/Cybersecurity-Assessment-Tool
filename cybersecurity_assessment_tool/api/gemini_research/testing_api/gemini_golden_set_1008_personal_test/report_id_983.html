Recommendations

To mitigate the identified risks and enhance the overall cybersecurity posture of Oasis Wellness, the following actionable recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) for Email: Mandate and enforce MFA for all organizational email accounts. This is a critical step to significantly reduce the risk of email account compromise, which is a common initial vector for cyberattacks.

  2. Implement Multi-Factor Authentication (MFA) for Sensitive Data Systems: Extend MFA requirements to all systems that store, process, or transmit sensitive organizational data. This adds a crucial layer of security to protect the most valuable assets.

  3. Develop and Enforce an Acceptable Use Policy (AUP): Create a comprehensive AUP outlining the appropriate and inappropriate use of company IT resources, data, and internet access. Ensure all employees acknowledge, understand, and adhere to this policy annually.

  4. Establish Annual Security Awareness Training: Implement a mandatory, engaging, and up-to-date annual cybersecurity awareness training program for all employees. Training should cover current threats (e.g., phishing), secure computing practices, data handling best practices, and AUP compliance.

  5. Review and Secure Localhost Exposure and SSH Service: Conduct an immediate investigation into the ‘Localhost Exposed‘ risk and the open SSH service on . If the service is not required, disable it. If required, ensure it is configured with strong security measures, including:

    • Disabling root login directly via SSH.

    • Enforcing key-based authentication over password-based authentication.

    • Restricting access to specific IP addresses or user groups.

    • Ensuring the SSH server and client software are regularly patched and up-to-date.

  6. Conduct Regular Vulnerability Assessments and Penetration Tests: Implement a schedule for regular internal and external vulnerability scanning and penetration testing. This proactive approach helps to continuously identify and address security weaknesses across the IT infrastructure.

  7. Develop and Test an Incident Response Plan: Create a formal incident response plan that outlines procedures for detecting, responding to, containing, eradicating, recovering from, and learning from cybersecurity incidents. Regularly test this plan through tabletop exercises to ensure effectiveness.

Conclusion

Oasis Wellness demonstrates some foundational security practices, such as MFA for computer logins and initial employee security training. However, significant gaps exist, particularly concerning MFA adoption for critical systems (email, sensitive data), formal policy enforcement (AUP), and ongoing employee training. The technical scan, combined with pre-existing risks, highlights specific configuration weaknesses that require immediate attention. Addressing these areas through the provided recommendations will substantially strengthen the organization’s defenses against current and emerging cyber threats, fostering a more robust and resilient cybersecurity posture.
