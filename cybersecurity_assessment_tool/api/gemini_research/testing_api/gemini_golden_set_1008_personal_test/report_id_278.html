Executive Summary

This report provides a comprehensive cybersecurity posture assessment for Falcon Heavy. The analysis integrates findings from a technical network scan, a self-assessment questionnaire, and a review of pre-existing organizational risks. The assessment highlights several critical security vulnerabilities and control gaps. Notably, the organization is running an extremely outdated and vulnerable FTP service (vsftpd 2.3.4) with anonymous login enabled, posing an immediate and severe risk of compromise. Additionally, significant policy and control gaps were identified, particularly the lack of Multi-Factor Authentication (MFA) for email and sensitive data systems, and insufficient security awareness training. Immediate action is required to remediate these critical findings and implement foundational security controls to protect Falcon Heavy from prevailing cyber threats.

Detailed Technical Findings

Outdated VSFTPD 2.3.4 on Port 21

The FTP service running on port 21 is vsftpd 2.3.4. This version is extremely old (released in 2011) and contains a critical backdoor vulnerability (CVE-2011-2523) along with other numerous security flaws. Continued use of this version poses a severe risk of system compromise and data exfiltration. Additionally, anonymous FTP login is permitted on this service, further exacerbating the risk by allowing unauthorized access to the FTP server without credentials.

Recommendations

Based on the comprehensive assessment, the following actionable recommendations are provided to improve Falcon Heavy’s cybersecurity posture and mitigate identified risks:

  1. Disable anonymous FTP login on 10.0.0.15 port 21. All FTP access should require strong authentication, and access should be restricted to authorized users only.

  2. Develop and enforce a comprehensive employee acceptable use policy (AUP) to define acceptable behavior and usage of organizational IT resources, reducing insider threats and misuse.

  3. Establish regular security awareness training for all employees, including new hires, covering common threats like phishing, malware, and social engineering. This builds a human firewall.

  4. Immediately implement Multi-Factor Authentication (MFA) for access email. This is a critical security control to prevent unauthorized access, even if credentials are compromised.

  5. Immediately implement Multi-Factor Authentication (MFA) for access sensitive data systems. This is a critical security control to prevent unauthorized access, even if credentials are compromised.

  6. Immediately upgrade vsftpd on 10.0.0.15 port 21 to the latest stable version or replace it with a more secure file transfer solution (e.g., SFTP/SCP). Ensure all patches are applied.

  7. Refit hardware (for Outdated Windows Policy)

  8. Upgrade to Win 10 (for Outdated Windows Policy)
