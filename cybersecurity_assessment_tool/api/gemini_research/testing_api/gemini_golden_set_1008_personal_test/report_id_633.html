Executive Summary

This report provides a comprehensive cybersecurity posture assessment for Fable & Lore, based on an organizational questionnaire, a technical network scan, and a review of existing risk data. Our analysis reveals a mixed security posture, with notable strengths in some areas but critical gaps that require immediate attention.

Key findings include a significant lack of Multi-Factor Authentication (MFA) for email and computer access, and an absence of regular security awareness training. Technically, an open port 8080 was identified on an internal system, exposing a service titled "TOP SECRET DB." This directly contradicts existing risk documentation that classified this port as secure and a false positive, indicating a critical discrepancy in the organization’s risk management and asset inventory. Recommendations focus on addressing these critical gaps and discrepancies to enhance the overall security posture.

Recommendations

Based on the assessment, the following recommendations are provided to improve Fable & Lore’s cybersecurity posture. These are prioritized to address the most critical identified risks.

Immediate Actions (High Priority)

  1. Investigate and Secure Port 8080 on 10.5.5.5:

    • Immediately verify the purpose and legitimacy of the service running on port 8080 displaying "TOP SECRET DB."

    • If unauthorized, shut down the service and port immediately.

    • If legitimate, ensure it is properly secured, access-controlled, and documented. Implement strong authentication and encryption.

    • Reconcile this finding with existing risk documentation and update the risk register accordingly. Investigate why this discrepancy was not previously identified or remediated.

  2. Implement Multi-Factor Authentication (MFA) for Email Access:

    • Deploy and enforce MFA for all employee email accounts (e.g., using authenticator apps, hardware tokens).

    • Communicate changes and provide training to users.

  3. Implement Multi-Factor Authentication (MFA) for Computer Login:

    • Evaluate and deploy MFA solutions for logging into organizational computers and critical servers.

    • Phased implementation may be necessary, starting with administrative accounts and critical systems.

Strategic Actions (Medium Priority)

  1. Establish and Conduct Regular Security Awareness Training:

    • Develop a comprehensive security awareness program.

    • Provide mandatory security awareness training for all new employees during onboarding.

    • Implement annual (at minimum) security awareness training for all existing employees, covering topics like phishing, social engineering, password hygiene, and data handling.

  2. Regular Vulnerability Scanning and Penetration Testing:

    • Implement a schedule for regular internal and external vulnerability scans.

    • Consider engaging third-party penetration testers periodically to identify weaknesses from an attacker’s perspective.

  3. Review and Update Risk Management Process:

    • Enhance the process for identifying, assessing, and documenting risks to ensure findings from technical scans are integrated and discrepancies are addressed promptly.

    • Ensure regular reviews of the risk register and associated mitigations.
