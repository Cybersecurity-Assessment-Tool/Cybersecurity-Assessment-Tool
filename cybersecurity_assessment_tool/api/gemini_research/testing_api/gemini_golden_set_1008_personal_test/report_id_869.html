5. Recommendations

To address the identified risks and enhance the overall cybersecurity posture, the following recommendations are provided:

5.1. Implement Multi-Factor Authentication (MFA)

  • MFA for Email Access: Implement and enforce MFA for all email access methods, including webmail, desktop clients, and mobile applications. This significantly reduces the risk of account compromise due to stolen or weak passwords.

  • MFA for Computer Login: Deploy and enforce MFA for all computer logins, especially for domain-joined machines and any remote access solutions (e.g., VPNs, remote desktop gateways).

5.2. Enhance Security Awareness Training

  • New Employee Training: Develop and implement a mandatory security awareness training program for all new employees during their onboarding process.

  • Annual Recurrent Training: Conduct mandatory, organization-wide security awareness training at least once per year for all employees. This training should cover current threats such as phishing, social engineering, malware, and best practices for data handling.

5.3. Mitigate RDP Exposure

  • Immediate Action - Restrict Access: Immediately restrict direct RDP access to (and ) to only trusted administrative subnets or through a dedicated, hardened jump host. RDP should not be directly accessible from untrusted network segments.

  • Long-Term Solution - Secure Remote Access: Implement a Virtual Private Network (VPN) solution for all remote administrative access. Ensure that RDP services are only accessible once a secure VPN tunnel has been established.

  • Security Configuration:

    • Enable Network Level Authentication (NLA) on all RDP instances to add an authentication layer before session establishment.

    • Enforce strong, complex passwords and account lockout policies for all accounts with RDP access. Consider implementing smart card authentication where feasible.

    • Regularly patch and update operating systems hosting RDP services to protect against known vulnerabilities.

    • Log RDP connection attempts and review logs for suspicious activity.
