Recommendations

The following recommendations are provided to address the identified risks and enhance Copperhead Cables’ cybersecurity posture:

Immediate Actions

  • Implement MFA for Email (RA-01): Enable and enforce Multi-Factor Authentication for all email accounts across the organization. This is a critical first step to prevent phishing and credential stuffing attacks.

  • Implement MFA for Sensitive Data Systems (RA-02): Deploy MFA for all systems that store or process sensitive data. This adds an essential layer of security for high-value assets.

  • Secure Exposed RDP Services (RA-04, RA-05):

    • Restrict RDP access on both and (and any other internally exposed RDP services) to trusted internal networks only.

    • Ideally, RDP should only be accessible via a VPN connection.

    • Ensure strong, unique passwords or certificate-based authentication is enforced for all RDP accounts.

    • Implement network-level access control (e.g., firewall rules) to limit source IPs.

Long-Term Strategic Initiatives

  • Develop and Implement an Acceptable Use Policy (RA-03): Create a comprehensive Employee Acceptable Use Policy (AUP) that clearly defines acceptable and unacceptable use of company IT resources. Ensure all employees are trained on and acknowledge this policy.

  • Regular Vulnerability Scanning: Implement a schedule for regular internal and external vulnerability scans to proactively identify and address exposed services and misconfigurations.

  • Enhance Security Awareness Program: While training is conducted, ensure that topics like MFA importance, recognizing phishing attempts, and safe RDP usage are emphasized, especially given the current MFA and RDP-related gaps.

Conclusion

Copperhead Cables demonstrates a foundational commitment to security through regular security awareness training. However, critical vulnerabilities exist related to the lack of Multi-Factor Authentication for key systems, the absence of an Employee Acceptable Use Policy, and the exposure of Remote Desktop Protocol services. Addressing these findings with the provided recommendations will significantly strengthen the organization’s defenses against common cyber threats and improve its overall security posture.
