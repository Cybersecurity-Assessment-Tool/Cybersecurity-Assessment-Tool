Overview

This report provides a cybersecurity posture assessment for Oasis Wellness, based on a review of their organizational security controls, a network scan of a target IP, and an assessment of existing risks. The analysis reveals several critical areas requiring immediate attention, particularly concerning Multi-Factor Authentication (MFA) implementation and an outdated web server version. While some foundational security controls are in place, the identified gaps introduce significant risk to the organization’s data and systems.

Technical Analysis

The scan revealed an Nginx web server running version 1.18.0 on port 443 (HTTPS). Nginx 1.18.0 was released in April 2020. As of the time of this assessment, this version is significantly outdated. Running outdated software can expose the system to a multitude of known vulnerabilities that have been patched in newer releases. This poses a high risk, as unpatched vulnerabilities can be exploited for various attacks, including data breaches, denial of service, and system compromise. The certificate common name www.acme-corp.com was observed, which may indicate a mismatch or an inherited configuration.

Recommendations

To mitigate the identified risks and improve the overall cybersecurity posture, the following recommendations are provided:

  1. Implement MFA for Email Access:

    • Action: Immediately implement and enforce Multi-Factor Authentication for all email accounts across the organization.

    • Benefit: This significantly reduces the risk of email account compromise, even if passwords are stolen.

  2. Implement MFA for Computer Login:

    • Action: Deploy and enforce MFA for all employee computer logins, especially for devices with access to sensitive data or internal networks.

    • Benefit: Enhances endpoint security and prevents unauthorized access to workstations and internal resources.

  3. Update Nginx Web Server:

    • Action: Plan and execute an upgrade of the Nginx web server on 192.168.10.5 (and any other instances) to the latest stable version. Prioritize this update due to the age of the current version.

    • Consideration: Ensure proper testing of applications hosted on the server before and after the upgrade to prevent service disruption.

    • Action: Investigate the SSL certificate common name mismatch (www.acme-corp.com vs. OasisWellness.com) and update it to reflect the correct organizational domain.

    • Benefit: Addresses known vulnerabilities, improves performance, and ensures compatibility with modern security features.

  4. Establish Annual Security Awareness Training:

    • Action: Develop and implement a mandatory annual security awareness training program for all employees, covering topics such as phishing, social engineering, password hygiene, and data handling.

    • Benefit: Strengthens the human firewall, reduces the likelihood of successful social engineering attacks, and fosters a proactive security culture.

  5. Regular Software Patching and Updates:

    • Action: Implement a robust patch management policy for all operating systems, applications, and network devices.

    • Benefit: Ensures that all systems are protected against the latest known vulnerabilities.

Conclusion

While Oasis Wellness has some foundational security measures in place, the identified critical gaps in Multi-Factor Authentication, outdated software, and insufficient ongoing security awareness training represent significant vulnerabilities. Addressing these issues with the recommended actions will substantially enhance the organization’s defense against cyber threats and protect its valuable assets. A proactive approach to cybersecurity, including regular assessments and continuous improvement, is vital in today’s threat landscape.
