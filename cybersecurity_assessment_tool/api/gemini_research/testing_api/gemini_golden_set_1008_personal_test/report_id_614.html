Technical Observations

  • Critical Vulnerable FTP Service: An FTP service (vsftpd 2.3.4) was found running on port 21. This specific version is notoriously vulnerable to a critical backdoor exploit (CVE-2011-2523), allowing remote code execution. This represents an extremely high risk to the targeted host and potentially the wider network.

  • Anonymous FTP Access: The FTP service is configured to allow anonymous logins. This, combined with the critical vulnerability, exacerbates the risk, allowing unauthenticated attackers to potentially gain access and exploit the system.

Recommendations

To mitigate the identified risks and enhance the overall cybersecurity posture, the following recommendations are provided:

Immediate Critical Actions

  • Patch/Upgrade/Disable FTP Service: Immediately patch, upgrade, or disable the vsftpd service (version 2.3.4) on 10.0.0.15. Given its critical vulnerability, this service should be taken offline or secured without delay. If FTP is essential, replace it with a secure, up-to-date alternative (e.g., SFTP).

  • Disable Anonymous FTP: Configure the FTP server to disallow anonymous login. Ensure that all necessary FTP users authenticate with strong credentials.

High Priority Actions

  • Implement MFA for Computer Logins: Mandate and implement Multi-Factor Authentication for all computer logins across the organization.

  • Implement MFA for Sensitive Data Access: Extend MFA requirements to all systems storing or processing sensitive data.

  • Upgrade End-of-Life Operating Systems: Develop and execute a plan to upgrade all workstations running Windows 7 to a currently supported and patched operating system (e.g., Windows 10/11 or a secure Linux distribution).

Medium Priority Actions

  • Introduce New Employee Security Awareness Training: Develop and implement mandatory security awareness training as part of the onboarding process for all new employees. This should cover common threats like phishing, social engineering, and safe internet practices.

Conclusion

While Zenith Point demonstrates some foundational security controls (e.g., AUP, annual SAT, MFA for email), significant vulnerabilities and gaps exist. The critical FTP vulnerability and lack of comprehensive MFA are immediate concerns that require urgent attention. By systematically addressing the recommendations outlined in this report, Zenith Point can significantly strengthen its defenses against potential cyberattacks and improve its overall security posture.
