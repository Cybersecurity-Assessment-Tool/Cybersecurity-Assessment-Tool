Overview

This report provides a comprehensive cybersecurity assessment based on an organizational data questionnaire, a network scan, and an existing risk register. The analysis reveals several key areas of concern, primarily revolving around the lack of foundational security controls and the exposure of critical services.

The organization, Granite Shield, currently exhibits significant gaps in security awareness training and multi-factor authentication (MFA) implementation for critical systems, including computer logins and employee acceptable use policy enforcement. Technically, an internal MySQL database server is exposed, correlating with an identified high-severity risk. Addressing these issues is crucial for enhancing the overall security posture and reducing the attack surface.

Recommendations

Based on the assessment, the following recommendations are provided to mitigate identified risks and improve the overall cybersecurity posture:

  1. Implement Multi-Factor Authentication (MFA) for Computer Logins:

    • Action: Mandate MFA for all user accounts logging into organizational computers and systems to significantly reduce the risk of credential-based attacks.

    • Benefit: Protects against unauthorized access even if primary credentials are compromised.

  2. Restrict Database Access (Port 3306):

    • Action (Easy Fix): Implement strict firewall rules to limit access to MySQL port 3306 () only to authorized systems or applications.

    • Action (Long-term Fix): Implement a VPN or secure network segmentation to ensure database access is only possible over an encrypted and authenticated tunnel.

    • Benefit: Prevents unauthorized direct access to the database, reducing the attack surface.

  3. Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Action: Create a clear and comprehensive AUP outlining the proper use of company IT resources, data, and internet access.

    • Action: Ensure all employees read and acknowledge the AUP, ideally as part of onboarding and periodically thereafter.

    • Benefit: Establishes clear guidelines, reduces misuse, and helps in compliance efforts.

  4. Initiate and Maintain Comprehensive Security Awareness Training:

    • Action: Develop and implement mandatory security awareness training for all new hires during onboarding.

    • Action: Conduct annual refresher security awareness training for all employees, covering topics such as phishing, social engineering, password hygiene, and data handling best practices.

    • Benefit: Educates employees on common threats, turning them into a strong line of defense against cyberattacks.
