Recommendations

The following recommendations are provided to address the identified risks and improve Blackwood Industries’ cybersecurity posture.

Multi-Factor Authentication (MFA)

  • Implement MFA for Computer Logins: Deploy and enforce Multi-Factor Authentication for all user logins to company computers and workstations. This adds a crucial layer of security beyond passwords and significantly reduces the risk of credential theft leading to unauthorized access.

  • Implement MFA for Sensitive Data Systems: Extend MFA requirements to all systems that store or process sensitive organizational data. This protects critical information even if primary credentials are compromised, aligning with best practices for data protection.

Policy and Governance

  • Develop and Implement an Employee Acceptable Use Policy (AUP): Create a comprehensive AUP that clearly outlines expected employee behavior regarding company IT resources, data handling, internet usage, and security protocols. Ensure all employees review, understand, and formally acknowledge this policy to establish clear guidelines and expectations.

Security Awareness Training

  • Introduce Mandatory Security Awareness Training for New Employees: Establish a mandatory security awareness training program for all new hires as part of their onboarding process. This should cover common threats, phishing, password hygiene, social engineering tactics, and the importance of adhering to company security policies from day one. Early training is crucial for building a security-conscious culture.

  • Review Existing Training Effectiveness: While annual training for all employees is in place, consider reviewing its content, delivery methods, and frequency to ensure it remains relevant, engaging, and effective in addressing current threat landscapes and employee knowledge gaps.

Future Technical Assessments

  • Conduct Comprehensive Network Scans: Regular and comprehensive network vulnerability scans (both external and internal) should be performed to proactively identify open ports, vulnerable services, and misconfigurations. Ensure that scan targets are properly defined and comprehensive for a complete assessment.
