Recommendations

To address the identified risks and enhance the overall cybersecurity posture, the following recommendations are provided:

Immediate Actions (High Priority)

  1. Implement MFA for Sensitive Data Systems: Mandate Multi-Factor Authentication (MFA) for all access to systems containing sensitive data. This significantly reduces the risk of unauthorized access even if credentials are compromised.

  2. Develop and Implement an Acceptable Use Policy (AUP): Create a comprehensive AUP that clearly outlines employee responsibilities, acceptable use of company IT resources, and prohibited activities. Ensure all employees acknowledge and understand this policy.

  3. Mandatory Security Awareness Training for New Hires: Implement a mandatory security awareness training program for all new employees during their onboarding process. This training should cover phishing, malware, data handling, and company security policies.

  4. Investigate and Rectify SSL Certificate Mismatch: Immediately investigate why the SSL certificate on port 443 has a common name (www.acme-corp.com) that does not align with the organization’s domain (www.CalyxBotany.com). This could indicate a misconfiguration, an expired certificate, or a security issue. Replace or reconfigure the certificate to match the correct domain.

Strategic Improvements (Medium Priority)

  1. Update Nginx Web Server: Upgrade the Nginx web server from version 1.18.0 to the latest stable release. Regularly updating server software ensures that the system benefits from the latest security patches, bug fixes, and performance improvements, mitigating risks from known vulnerabilities.

Conclusion

While Calyx Botany has implemented essential security controls such as MFA for email and computer logins, this assessment highlights several critical vulnerabilities and security gaps that require immediate attention. The absence of MFA for sensitive data, a formal AUP, and initial security awareness training for new employees, coupled with technical misconfigurations like the SSL certificate mismatch and an outdated web server, pose significant risks. By diligently addressing these recommendations, Calyx Botany can substantially strengthen its security posture and better protect its assets from evolving cyber threats.
