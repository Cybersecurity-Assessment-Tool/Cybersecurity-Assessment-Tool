Executive Summary

This report details the findings from a cybersecurity posture assessment conducted for True Grit. The assessment involved analyzing organizational security questionnaire responses, an external network scan, and existing risk data.

While the external network scan revealed no open ports or immediately exploitable services, indicating a potentially hardened external perimeter, critical gaps were identified in the organization’s internal security controls and employee-centric policies. Specifically, the absence of an employee acceptable use policy and the lack of mandatory security awareness training for new employees pose significant risks. These foundational policy and training deficiencies can lead to increased human error, insider threats, and compliance challenges.

Recommendations provided aim to address these high-priority risks by implementing essential policies and training programs to bolster the overall security posture of True Grit.

Recommendations

To mitigate the identified risks and enhance the overall cybersecurity posture of True Grit, the following recommendations are provided:

  1. Develop and Implement a Comprehensive Acceptable Use Policy (AUP):

    • Create a clear and concise AUP that outlines the rules for using company IT assets, networks, and data.

    • The policy should cover topics such as internet usage, email, social media, data handling, mobile device usage, and privacy expectations.

    • Ensure all employees read, understand, and formally acknowledge the AUP, ideally during onboarding and annually thereafter.

    • Integrate AUP principles into existing and new security awareness training programs.

  2. Establish Mandatory Security Awareness Training for New Employees:

    • Implement a mandatory security awareness training module as a critical component of the new employee onboarding process.

    • This training should cover fundamental security concepts, common threats (e.g., phishing, malware), password best practices, data protection, and reporting suspicious activities.

    • Ensure the training is engaging and relevant, providing practical examples applicable to their roles.

  3. Review and Enhance Existing Annual Security Awareness Training:

    • While annual training is already in place, review its content and effectiveness, especially in light of the identified policy and new-hire training gaps.

    • Ensure the annual training covers evolving threats, reinforces AUP principles, and includes refresher information on key security practices.

    • Consider phishing simulations and other interactive exercises to test and reinforce employee understanding.
