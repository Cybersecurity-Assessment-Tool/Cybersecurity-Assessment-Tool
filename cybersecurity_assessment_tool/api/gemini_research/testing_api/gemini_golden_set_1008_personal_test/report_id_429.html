Executive Summary

This report details the findings of a cybersecurity posture assessment for Borealis Tech, incorporating organizational security controls review, network scan results, and existing risk data. The assessment identified several critical gaps in multi-factor authentication (MFA) implementation, organizational policy adherence, and ongoing security awareness training. While a network scan revealed an open port 8080 with a potentially sensitive title, existing risk data indicates this particular finding has been previously evaluated and categorized as a mitigated false positive (CVSS 0.0). The primary areas of concern revolve around fundamental security hygiene and employee training, which are crucial for establishing a robust defense against common cyber threats. Immediate attention to the recommendations outlined in this report is advised to enhance the overall security posture.

Recommendations

To address the identified risks and enhance the overall cybersecurity posture of Borealis Tech, the following recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) for Email Access:

    • Immediately enable and enforce MFA for all employee email accounts. This significantly reduces the risk of email account compromise, even if passwords are stolen.

  2. Implement Multi-Factor Authentication (MFA) for Computer Logins:

    • Deploy and require MFA for all user logins to company computers and endpoints, especially those with access to sensitive data. This adds a critical layer of security against unauthorized access to internal systems.

  3. Develop and Implement an Acceptable Use Policy (AUP):

    • Create a comprehensive AUP outlining the proper use of company IT resources, data, and internet access.

    • Ensure all employees read, understand, and sign the AUP upon hiring and annually thereafter.

  4. Conduct Annual Security Awareness Training for All Employees:

    • Establish a mandatory, annual security awareness training program for all employees, covering topics such as phishing, social engineering, password hygiene, data handling, and company policies.

    • Supplement annual training with regular, short-form reminders or simulated phishing exercises.

  5. Continuous Monitoring and Re-validation of Port 8080:

    • Although currently classified as a mitigated false positive, regularly re-assess the security of port 8080 to ensure its configuration remains secure and that the "TOP SECRET DB" title does not indicate an overlooked vulnerability or exposure.

    • Maintain clear documentation regarding the justification for its ’secured’ status.
