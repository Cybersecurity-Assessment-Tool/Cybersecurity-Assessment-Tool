Executive Summary

This report provides a comprehensive cybersecurity assessment for Structure & Form, based on network scan data, organizational security posture questionnaire responses, and pre-existing risk information. The analysis reveals several key areas requiring immediate attention. Critical findings include the exposure of Remote Desktop Protocol (RDP) on internal network assets and a significant procedural gap with the absence of an Employee Acceptable Use Policy. While Structure & Form demonstrates strong adoption of Multi-Factor Authentication (MFA) and Security Awareness Training, these positive aspects are undermined by fundamental vulnerabilities that could lead to unauthorized access, data breaches, or compliance issues. Immediate recommendations focus on securing RDP services and implementing essential security policies to bolster overall resilience.

Recommendations

The following recommendations are provided to address the identified risks and improve the overall cybersecurity posture of Structure & Form.

High-Priority Technical Recommendations

  1. Secure Remote Desktop Protocol (RDP) Services:

    • Immediately: Evaluate the necessity of RDP on 10.10.10.51. If not essential, disable the service or block access to port 3389.

    • If RDP is required:

      • Restrict access to RDP through a Virtual Private Network (VPN) or a dedicated jump box.

      • Ensure Network Level Authentication (NLA) is enabled.

      • Implement strong, unique passwords for all RDP accounts.

      • Consider implementing multi-factor authentication for RDP access, if feasible.

      • Apply all available security patches and updates for RDP services.

    • Systemic Review: Conduct a comprehensive audit of all internal systems to identify and secure other instances of RDP exposure (as highlighted by the pre-existing risk on 10.10.10.50).

  2. Address other technical vulnerabilities

    • Follow the easy fix recommendation for "RDP Exposure" on 10.10.10.50: "Close Port".

    • Implement the long-term fix recommendation for "RDP Exposure" on 10.10.10.50: "VPN".

High-Priority Policy & Procedural Recommendations

  1. Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Create a clear and comprehensive AUP document outlining acceptable and unacceptable use of company IT resources, data, and internet.

    • Ensure the AUP covers topics such as password hygiene, data handling, email etiquette, use of personal devices, and reporting security incidents.

    • Distribute the AUP to all employees, obtain signed acknowledgments, and integrate it into new employee onboarding and annual security awareness training.

    • Regularly review and update the AUP to reflect changes in technology, threats, and organizational policy.
