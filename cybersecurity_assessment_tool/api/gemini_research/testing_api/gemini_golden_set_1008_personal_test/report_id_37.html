Recommendations

To address the identified risks and improve Nebula Creative’s cybersecurity posture, the following recommendations are provided:

  1. Implement MFA for Computer Logins:

    • Action: Mandate Multi-Factor Authentication (MFA) for all user logins to company computers and endpoints.

    • Benefit: Significantly reduces the risk of unauthorized access even if primary credentials are compromised.

  2. Implement MFA for Sensitive Data Systems:

    • Action: Extend MFA requirements to all systems storing or processing sensitive organizational or client data.

    • Benefit: Provides an additional layer of security for critical assets, protecting against data breaches and compliance violations.

  3. Develop and Enforce an Acceptable Use Policy (AUP):

    • Action: Draft, disseminate, and ensure all employees acknowledge an AUP that clearly outlines the acceptable use of company IT resources, including internet, email, and hardware.

    • Benefit: Sets clear expectations for employees, reduces misuse, and provides a framework for accountability.

  4. Establish Annual Security Awareness Training:

    • Action: Implement a mandatory annual security awareness training program for all employees, covering current threats, phishing, password hygiene, and company policies.

    • Benefit: Enhances the "human firewall," making employees more resilient to social engineering attacks and improving overall security culture.

  5. Upgrade Nginx Web Server:

    • Action: Plan and execute an upgrade of the Nginx web server on to the latest stable version (e.g., Nginx 1.26.x or newer). Ensure proper testing is conducted before deployment.

    • Benefit: Addresses potential security vulnerabilities in older versions, improves performance, and ensures access to the latest features and patches.

  6. Review and Update SSL Certificates:

    • Action: Investigate the discrepancy between the SSL certificate common name (www.acme-corp.com) and the organization’s identity. Ensure all public-facing services use certificates that accurately reflect Nebula Creative and are properly configured.

    • Benefit: Prevents user trust warnings, ensures proper identity verification, and adheres to best practices for web security.

  7. Regular Vulnerability Scanning and Patch Management:

    • Action: Implement a routine schedule for vulnerability scanning of internal and external assets. Establish a robust patch management program to ensure all systems and applications are kept up-to-date.

    • Benefit: Proactively identifies and remediates vulnerabilities before they can be exploited.

These recommendations, when implemented, will significantly strengthen Nebula Creative’s cybersecurity defenses and reduce its attack surface. A phased approach is advised, prioritizing critical and high-severity risks first.
