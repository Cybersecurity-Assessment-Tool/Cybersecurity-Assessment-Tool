Recommendations

Based on the assessment, the following recommendations are provided to enhance Sterling Silver’s cybersecurity posture:

  1. Implement Multi-Factor Authentication (MFA) for Email Access: Immediately enable and enforce MFA for all employee email accounts to significantly reduce the risk of unauthorized access due to compromised credentials.

  2. Implement Multi-Factor Authentication (MFA) for Sensitive Data Systems: Extend MFA implementation to all systems containing or accessing sensitive organizational data. This is a critical control against data breaches.

  3. Mandatory Security Awareness Training for New Employees: Develop and implement a mandatory security awareness program for all new hires, covering topics such as phishing, social engineering, password hygiene, and data handling best practices.

  4. Conduct Annual Security Awareness Training: Establish a recurring annual security awareness training program for all employees to keep them informed about current threats and best practices.

  5. Investigate and Remediate ’Unencrypted Web Server’ Risk: Validate the existence and scope of the ’Unencrypted Web Server’ risk within the organization’s infrastructure. If port 80 is indeed open on any public-facing or internal servers handling sensitive data, ensure it is either closed, redirected to HTTPS, or secured with appropriate encryption. Prioritize migration to HTTPS for all web services.

  6. Regular Vulnerability Scanning: Conduct regular internal and external vulnerability scans to proactively identify and remediate open ports, vulnerable services, and outdated software across all assets.

  7. Maintain an Up-to-Date Risk Register: Regularly review and update the organization’s risk register to reflect current findings from scans, audits, and policy reviews, ensuring accuracy and relevance.

  8. Policy Communication and Enforcement: Ensure that existing policies, such as the Employee Acceptable Use Policy, are clearly communicated to all employees and consistently enforced throughout the organization.
