6. Recommendations

Addressing the identified risks is crucial for improving Vertex Solutions’s overall cybersecurity posture. The following recommendations prioritize actions based on their potential impact and severity:

6.1. Immediate Critical Actions

  1. Implement Multi-Factor Authentication (MFA) for Sensitive Data Systems: Immediately deploy MFA for all systems containing or processing sensitive data. This is the highest priority to prevent unauthorized access to critical assets.

  2. Implement Multi-Factor Authentication (MFA) for Email Access: Deploy MFA for all corporate email accounts. This protects against phishing and credential stuffing attacks, which are common initial access vectors.

  3. Implement Multi-Factor Authentication (MFA) for Computer Login: Require MFA for all user logins to company computers and endpoints. This adds a crucial layer of defense against compromised user accounts.

6.2. High Priority Actions

  1. Establish Annual Security Awareness Training: Develop and implement a mandatory annual security awareness training program for all employees. This training should cover current threats such as phishing, social engineering, malware, and data handling best practices.

6.3. Ongoing Security Practices

  1. Regular Network Scanning and Penetration Testing: Conduct regular external and internal network scans and periodic penetration tests to identify new vulnerabilities and ensure security controls are effective.

  2. Continuous Security Monitoring: Implement systems for continuous monitoring of network traffic, system logs, and user activity to detect and respond to suspicious behavior promptly.

  3. Incident Response Plan Review: Regularly review and update the organization’s incident response plan, including conducting tabletop exercises, to ensure readiness for security incidents.

By systematically addressing these recommendations, Vertex Solutions can significantly reduce its risk exposure and build a more resilient cybersecurity defense.
