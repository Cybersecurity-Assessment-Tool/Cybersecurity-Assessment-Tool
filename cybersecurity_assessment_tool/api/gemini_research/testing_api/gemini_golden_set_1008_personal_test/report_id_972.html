3.1. Analysis of Questionnaire Findings

The review highlights critical deficiencies in foundational security practices. The lack of Multi-Factor Authentication (MFA) for email access and computer logins significantly increases the risk of unauthorized access and account compromise, as stolen or guessed credentials would be sufficient for entry. While MFA is used for sensitive data systems, the broader omission creates vulnerable entry points. Furthermore, the absence of annual security awareness training for all employees means the workforce may not be adequately prepared to identify and resist evolving social engineering threats, phishing attempts, and other common attack vectors.

6. Recommendations

Based on the analysis, the following recommendations are provided to improve Skyward Bound’s cybersecurity posture:

  1. Implement MFA for Email Access: Immediately deploy and enforce Multi-Factor Authentication for all employee email accounts. This significantly reduces the risk of account compromise even if passwords are stolen.

  2. Implement MFA for Computer Logins: Extend MFA requirements to all computer login processes, particularly for domain-joined machines and critical servers.

  3. Establish Annual Security Awareness Training: Develop and implement a mandatory, annual security awareness training program for all employees. This training should cover common threats such as phishing, social engineering, malware, and safe browsing practices.

  4. Conduct Comprehensive Network Scan: Arrange for a complete and professional network vulnerability scan of all external and internal assets. This will identify open ports, vulnerable services, and outdated software versions that were unable to be assessed in this report.

  5. Review and Document Pre-existing Risks: Conduct a thorough review of any internal risk registers or vulnerability management systems to identify and prioritize pre-existing risks not covered in this report.

  6. Develop Incident Response Plan: Ensure an up-to-date and tested incident response plan is in place to effectively manage and mitigate the impact of any security incidents.
