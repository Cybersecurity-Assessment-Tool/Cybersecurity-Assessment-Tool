Executive Summary

This report details the findings of a cybersecurity assessment conducted for Structure & Form. The assessment involved an analysis of network scan data, an organizational security questionnaire, and pre-existing risk information. The findings indicate a significant number of security vulnerabilities and control gaps, leading to an overall high-risk posture.

Key findings include:

  • Critical Software Vulnerability: An End-of-Life (EOL) MySQL database (version 5.7.33) is exposed on port 3306, lacking essential security updates.

  • Inadequate Multi-Factor Authentication (MFA): MFA is not required for access to sensitive data systems, leaving critical data vulnerable.

  • Lack of Essential Policies and Training: The organization lacks an Employee Acceptable Use Policy (AUP) and comprehensive security awareness training for both new hires and existing staff.

These combined issues present a substantial attack surface and elevate the risk of data breaches, unauthorized access, and operational disruption. Immediate and decisive action is recommended to mitigate these identified risks.

Analysis:

The presence of an open MySQL port (3306) running version 5.7.33 is a significant concern. MySQL 5.7 reached End-of-Life (EOL) in October 2023. This means it no longer receives security updates, making it highly susceptible to known vulnerabilities and exploits. Exposure of such an outdated and unpatched service significantly increases the risk of unauthorized database access, data compromise, and system exploitation.

Recommendations

To address the identified risks and improve the overall cybersecurity posture of Structure & Form, the following recommendations are provided. These recommendations are prioritized based on their potential impact and severity.

Immediate Actions (High Priority)

  1. Address EOL MySQL Database:

    • Upgrade/Migrate MySQL: Immediately plan and execute an upgrade of the MySQL database to a currently supported version (e.g., MySQL 8.0) or migrate to a more secure database solution.

    • Implement Network Segmentation/Firewall Rules: Restrict access to port 3306 (MySQL) from external networks. Implement strict firewall rules to only allow connections from trusted internal systems or via a secure VPN.

    • Review Database Configurations: Ensure the MySQL instance is configured with strong passwords, least privilege principles, and disabled unnecessary features.

  2. Implement MFA for Sensitive Data Systems:

    • Deploy and enforce Multi-Factor Authentication for all systems storing or processing sensitive organizational or client data. This significantly reduces the risk of unauthorized access due to compromised credentials.

Strategic Actions (Medium Priority)

  1. Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that clearly outlines expectations for employee behavior regarding company IT resources, data handling, and internet usage.

    • Ensure all employees read, understand, and acknowledge the AUP.

  2. Establish Comprehensive Security Awareness Training Programs:

    • New Employee Training: Implement mandatory security awareness training for all new hires during their onboarding process.

    • Annual Training: Conduct mandatory annual security awareness training for all employees to keep them informed about current threats, best practices, and organizational policies.

    • Phishing Simulations: Regularly conduct simulated phishing attacks to test employee vigilance and reinforce training.

General Best Practices (Ongoing)

  1. Regular Vulnerability Scanning and Penetration Testing: Implement a schedule for routine vulnerability assessments and periodic penetration testing to identify new weaknesses and validate the effectiveness of implemented controls.

  2. Patch Management Program: Establish and maintain a robust patch management program for all operating systems, applications, and network devices.

  3. Incident Response Plan: Develop and regularly test a comprehensive incident response plan to ensure the organization can effectively detect, respond to, and recover from cybersecurity incidents.
