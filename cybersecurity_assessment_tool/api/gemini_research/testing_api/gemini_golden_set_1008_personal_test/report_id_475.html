Recommendations

To mitigate the identified risks and enhance the overall cybersecurity posture of Maple Leaf Logistics, the following recommendations are provided:

  • Implement Multi-Factor Authentication (MFA) for Computer Logins:

    • Deploy and enforce MFA for all employee computer logins, especially for devices accessing internal networks and sensitive systems.

    • Evaluate suitable MFA solutions (e.g., hardware tokens, authenticator apps) that integrate seamlessly with existing infrastructure.

  • Establish and Enforce Annual Security Awareness Training:

    • Develop and implement a mandatory annual security awareness training program for all employees.

    • The training should cover current threat landscapes, phishing, social engineering, password best practices, data handling, and incident reporting procedures.

    • Conduct regular phishing simulations to test employee awareness and reinforce training.

  • Secure Open SSH Service:

    • Restrict Access: Limit SSH access to only necessary IP addresses or networks using firewall rules.

    • Implement Key-Based Authentication: Disable password-based authentication for SSH and enforce the use of strong SSH keys.

    • Disable Root Login: Configure the SSH server to disallow direct root login.

    • Regular Patching: Ensure the SSH server and operating system are kept up-to-date with the latest security patches to prevent exploitation of known vulnerabilities.

    • Logging and Monitoring: Implement robust logging for SSH access attempts and monitor logs for unusual activity.

    • Change Default Port (Optional): Consider moving SSH to a non-standard port to reduce automated scanning attempts, though this is a security-by-obscurity measure and not a primary control.

  • Regular Security Reviews:

    • Conduct periodic internal reviews of security policies and controls to ensure they remain effective and aligned with evolving threats and organizational changes.

    • Regularly review and update the acceptable use policy.

Implementing these recommendations will significantly strengthen the organization’s defense mechanisms and reduce its exposure to common cyber threats.
