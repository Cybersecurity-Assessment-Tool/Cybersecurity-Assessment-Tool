Executive Summary

This report details the findings of a cybersecurity assessment conducted for Falcon Heavy. The assessment combined a technical network scan, a review of existing risks, and an analysis of organizational security controls based on a questionnaire.

The assessment identified several critical and high-risk vulnerabilities that expose the organization to significant threats, including unauthorized access, data breaches, and malware infections. Key findings include:

  • Critical FTP Vulnerability: An externally facing FTP server is running a severely outdated version of vsftpd (2.3.4), which is known to contain a critical backdoor vulnerability. Furthermore, the server is configured to allow anonymous logins, permitting unauthenticated access.

  • Critical Lack of Multi-Factor Authentication (MFA): MFA is not enforced for accessing email, which is a primary target for account takeover and phishing attacks.

  • High-Risk Procedural Gaps: MFA is also not required for computer logins, and new employees do not receive mandatory security awareness training, leaving the organization vulnerable to credential theft and social engineering.

The overall security posture is considered weak due to these fundamental control gaps. Immediate remediation of the identified critical risks is strongly recommended to reduce the likelihood of a security incident.

Technical Analysis

The scan revealed a critical finding:

  1. Outdated and Vulnerable Service: The FTP service is running vsftpd 2.3.4. This version was released in 2011 and is widely known to be vulnerable to a backdoor (CVE-2011-2523), which allows for remote command execution.

  2. Insecure Configuration: The service permits anonymous FTP logins. This allows any external party to connect to the server and potentially access, upload, or download files without authentication, posing a severe risk of data leakage or malware implantation.

Recommendations

The following actions are recommended to mitigate the identified risks. They are prioritized by severity.

Critical Priority Recommendations

RISK-001/002: Remediate Vulnerable FTP Server

  • Immediate Action: Take the FTP server offline immediately to prevent exploitation.

  • Short-Term: If FTP is a business necessity, upgrade the vsftpd software to the latest stable version and disable anonymous access.

  • Long-Term: Migrate from FTP to a more secure file transfer protocol, such as SFTP (SSH File Transfer Protocol) or FTPS (FTP over SSL/TLS).

RISK-003: Implement MFA for Email

  • Immediate Action: Enforce MFA for all users, especially administrators and executives, on the FalconHeavy.com email system.

High Priority Recommendations

RISK-004: Implement MFA for Endpoints

  • Action: Deploy an MFA solution for all workstation and server logins to protect against unauthorized access from compromised credentials.

RISK-005: Mandate New Hire Security Training

  • Action: Integrate mandatory security awareness training into the standard onboarding process for all new employees before they are granted system access.

Medium Priority Recommendations

RISK-006: Upgrade End-of-Life Operating Systems

  • Action: Continue with the planned project to upgrade all Windows 7 workstations to a supported operating system, such as Windows 10 or 11, to ensure they receive security patches.
