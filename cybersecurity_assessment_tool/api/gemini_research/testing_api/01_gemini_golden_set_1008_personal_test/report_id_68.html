Executive Summary

This report provides a comprehensive cybersecurity posture assessment for Stone Arch Masonry. The analysis is based on a synthesis of network scan data, a review of organizational security controls, and an evaluation of pre-existing risk information.

The assessment identified several critical and high-risk vulnerabilities that require immediate attention. Key findings include the absence of Multi-Factor Authentication (MFA) for primary access vectors such as email and computer logins, a lack of mandatory annual security awareness training for all staff, and the presence of an unencrypted web service (HTTP) on the network. These deficiencies create significant exposure to account compromise, data breaches, and social engineering attacks.

Recommendations

The following actions are recommended to mitigate the identified risks. They are prioritized based on severity and potential impact.

Immediate Priority (Critical Risks)

  1. Enforce Multi-Factor Authentication (MFA):

    • Immediately deploy and enforce MFA for all user access to the email system (StoneArchMasonry.net).

    • Implement MFA for all workstation and laptop logins across the organization. This should apply to both local and remote access.

  2. Secure Web Communications:

    • Obtain and install a TLS/SSL certificate on the web server at 172.16.0.1.

    • Reconfigure the server to use HTTPS (port 443) for all traffic.

    • Implement a permanent redirect (HTTP 301) to automatically forward all HTTP requests to HTTPS, ensuring all connections are encrypted.

High Priority

  1. Establish Annual Security Awareness Training:

    • Develop or procure a mandatory security awareness training program for all employees.

    • Conduct this training annually to ensure that knowledge of current threats (e.g., phishing, ransomware, social engineering) remains up-to-date.

    • Track completion to ensure 100% participation.

Conclusion

The assessment reveals critical gaps in the security posture of Stone Arch Masonry. While some foundational controls are in place, the absence of MFA on key systems and the use of unencrypted protocols present an unacceptable level of risk.

By implementing the prioritized recommendations outlined in this report, the organization can significantly reduce its attack surface, enhance its resilience against common cyber threats, and build a stronger security foundation.
