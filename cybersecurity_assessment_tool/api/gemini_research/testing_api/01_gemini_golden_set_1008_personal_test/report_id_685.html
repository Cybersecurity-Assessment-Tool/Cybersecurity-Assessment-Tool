Executive Summary

This report provides a comprehensive cybersecurity assessment for Oasis Wellness, conducted on November 22, 2025. The analysis correlates organizational security controls, technical network scan data, and pre-existing risk information.

The assessment reveals a mixed security posture. The organization demonstrates strong identity and access management controls, with Multi-Factor Authentication (MFA) consistently enforced across email, endpoints, and sensitive systems. This significantly reduces the risk of unauthorized access through compromised credentials.

However, critical deficiencies were identified in foundational security practices. The absence of an employee Acceptable Use Policy (AUP) and a formal security awareness training program creates significant exposure to insider threats, social engineering, and phishing attacks.

Furthermore, technical scanning of the external-facing web server at 192.168.10.5 revealed an outdated version of Nginx (1.18.0), which is known to have multiple security vulnerabilities. This presents a high risk of system compromise.

Immediate remediation should focus on establishing a security training program, developing an AUP, and patching the vulnerable web server software to mitigate these high-impact risks.

The following table synthesizes findings from the security control review and technical scan into a prioritized list of risks.

RISK-001 Lack of Security Policies & Training Critical The absence of an AUP and security training program makes the organization highly susceptible to phishing, social engineering, and insider threats, which are primary vectors for data breaches.

Recommendations

Based on the risk assessment, the following actions are recommended to improve the cybersecurity posture of Oasis Wellness.

  1. Develop and Implement Foundational Policies (RISK-001):

    • Immediate: Draft and implement a comprehensive Acceptable Use Policy (AUP) that all employees must read and acknowledge.

    • Short-Term: Establish a mandatory security awareness training program for all new hires.

    • Ongoing: Conduct annual security awareness training for all staff to keep them informed about evolving threats like new phishing techniques.

  2. Remediate Web Server Vulnerabilities (RISK-002):

    • Immediate: Upgrade the Nginx server on host 192.168.10.5 from version 1.18.0 to the latest stable version.

    • Ongoing: Implement a formal patch management policy to ensure all internet-facing systems are regularly updated to protect against newly discovered vulnerabilities.

  3. Correct SSL Certificate Configuration (RISK-003):

    • Immediate: Replace the current SSL certificate on the web server with a valid certificate issued for the domain www.OasisWellness.net.
