Executive Summary

This report provides a comprehensive analysis of the cybersecurity posture for Midnight Oil Studios, based on a synthesis of network scan data, organizational security controls, and pre-existing risk assessments. The evaluation was conducted on 2026-01-31.

The assessment has identified a critical, immediate risk related to an externally facing FTP server. This server is running a dangerously outdated version of vsftpd (2.3.4) with a known remote code execution vulnerability (CVE-2011-2523). The server is further misconfigured to allow anonymous access, compounding the risk of a severe data breach or system compromise.

Furthermore, significant gaps were identified in the organization’s administrative controls. The absence of a formal Acceptable Use Policy and the lack of security awareness training for new employees represent high-risk deficiencies that weaken the human firewall and increase susceptibility to social engineering and insider threats.

While the organization has implemented strong Multi-Factor Authentication (MFA) controls, the identified technical and administrative vulnerabilities require immediate attention to mitigate the substantial risk of a security incident. Recommendations are prioritized to address the most critical findings first.

Analysis of Technical Findings

The presence of vsftpd version 2.3.4 is a severe security risk. This specific version contains a well-known, malicious backdoor that was inserted into the source code. An attacker can gain a command shell on the server by sending a specific string as the username. Combined with the Anonymous FTP login misconfiguration, this system is trivial to compromise, providing a foothold for attackers to pivot into the internal network.

Recommendations

The following actions are recommended to mitigate the identified risks. They are prioritized based on severity and potential impact.

Immediate Priority (Critical Risk)

  1. Isolate and Remediate the FTP Server (10.0.0.15):

    • Immediately take the server offline or apply a firewall rule to block all access to port 21.

    • Investigate the server for signs of existing compromise.

    • Upgrade the vsftpd service to the latest stable version.

    • Disable anonymous FTP access unless there is an explicit and documented business requirement.

    • Long-Term: Replace FTP with a secure alternative, such as SFTP (SSH File Transfer Protocol), which encrypts both credentials and data in transit.

High Priority

  1. Develop and Implement an Acceptable Use Policy (AUP):

    • Draft a clear AUP that defines the rules for using company IT assets, data, and networks.

    • Require all current and new employees to read and formally acknowledge the policy.

  2. Establish Onboarding Security Training:

    • Create a mandatory security awareness training module for all new employees.

    • This training should cover key topics such as phishing, password security, data handling, and the new AUP.

Medium Priority

  1. Execute Operating System Upgrade Plan:

    • Prioritize and execute the existing recommendation to upgrade all Windows 7 workstations to a supported operating system (e.g., Windows 10/11) to ensure they receive critical security patches.
