Executive Summary

This report details the findings of a cybersecurity assessment conducted for Maple Leaf Logistics. The assessment combined a review of organizational security controls, an automated network scan of internal assets, and a correlation with pre-existing risk documentation.

The analysis revealed several high-impact security gaps that require immediate attention. Key findings include:

  • Critical Gaps in Access Control: Multi-Factor Authentication (MFA) is not enforced for employee email or computer logins. This exposes the organization to significant risks, including business email compromise, ransomware, and unauthorized data access.

  • Significant Information Disclosure: An internal network scan of target 10.5.5.5 discovered an open service on port 8080 with the title "TOP SECRET DB". This suggests a high-value data source is exposed. This finding directly contradicts a pre-existing risk assessment entry which incorrectly labeled this port as secure.

  • Insufficient Security Training: The organization does not provide mandatory, annual security awareness training for all employees, increasing susceptibility to phishing and other social engineering attacks.

Urgent remediation of these issues is recommended to reduce the organization’s risk exposure and strengthen its overall security posture. Detailed recommendations are provided in Section 6.

Recommendations

The following actionable recommendations are provided to address the identified risks. They are prioritized to focus on the most critical vulnerabilities first.

Immediate Actions (0-7 Days)

  1. Investigate and Secure Port 8080:

    • Immediately investigate the service running on 10.5.5.5:8080 to identify its function and the data it contains.

    • If the service is critical, implement strong authentication (e.g., username/password and MFA) and apply network-level access controls (firewall rules) to restrict access to only authorized personnel.

    • If the service is non-essential or a development remnant, it should be disabled and removed from the server.

  2. Update Risk Register: The risk entry for "Port 8080 Secured" must be immediately updated to reflect the new finding. The severity should be re-classified as Critical.

Short-Term Actions (1-3 Months)

  1. Deploy Multi-Factor Authentication (MFA):

    • Prioritize the rollout of MFA for all user accounts on the primary email system (MapleLeafLogistics.org).

    • Plan and execute the deployment of MFA for all employee computer (workstation and laptop) logins.

  2. Implement Annual Security Training:

    • Procure and schedule mandatory security awareness training for all current employees.

    • Formalize a policy requiring all employees to complete this training on an annual basis.

Long-Term Actions (3+ Months)

  1. Enhance Vulnerability Management: Review and improve the internal vulnerability scanning and risk assessment process to ensure that findings are accurately verified and not prematurely dismissed as false positives.

  2. Conduct Regular Assessments: Schedule regular, independent security assessments (including penetration testing) to proactively identify and remediate vulnerabilities.
