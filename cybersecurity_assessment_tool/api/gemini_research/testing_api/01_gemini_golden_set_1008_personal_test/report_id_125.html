Executive Summary

This report provides a cybersecurity assessment for Hearth & Home. The analysis is based on a network scan, a review of organizational security controls, and an evaluation of pre-existing risk data.

The assessment identified several critical-risk findings that require immediate attention. The primary concerns are the widespread lack of Multi-Factor Authentication (MFA) across all critical services (email, computer logins, and sensitive data systems) and the continued exposure of Remote Desktop Protocol (RDP) on internal network assets. The technical scan confirmed a new instance of an open RDP port on host 10.10.10.51, compounding a previously identified risk.

The absence of MFA drastically lowers the barrier for unauthorized access, especially when combined with exposed services like RDP, a common vector for ransomware attacks. Additionally, foundational governance gaps, such as the lack of an employee acceptable use policy, were noted.

Immediate remediation should focus on securing all exposed RDP services and implementing MFA across the organization.

Recommendations

The following prioritized recommendations are provided to mitigate the identified risks.

  1. Immediate Priority: Secure Exposed RDP Services

    • Immediately disable or firewall port 3389 on 10.10.10.51 and any other systems where RDP is not strictly required for business operations.

    • For systems requiring remote access, implement a secure solution such as a Virtual Private Network (VPN) with MFA. Do not expose RDP directly to any untrusted network.

  2. High Priority: Implement Multi-Factor Authentication

    • Prioritize the deployment of MFA for all remote access solutions (VPN), email access (e.g., Office 365, G Suite), and access to systems containing sensitive data.

    • Develop a roadmap to enforce MFA for all employee computer logins.

  3. Medium Priority: Develop and Implement Governance Policies

    • Create a formal Acceptable Use Policy (AUP) that defines rules for the use of company technology and data.

    • Require all employees to read and acknowledge the AUP as part of the annual security awareness training program.
