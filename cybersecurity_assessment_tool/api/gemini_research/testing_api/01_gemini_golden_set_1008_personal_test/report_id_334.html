Recommendations

The following actions are recommended to mitigate the identified risks and improve the overall security posture of Terraform Global.

  1. [Critical] Implement MFA for Email Access: Immediately enforce mandatory Multi-Factor Authentication (MFA) for all user access to the email system. This is the single most effective control to prevent account takeovers resulting from compromised credentials.

  2. [High] Develop and Enforce an Acceptable Use Policy (AUP): Create a formal AUP that clearly defines the rules and expectations for employees when using company networks, devices, and data. This policy should be a mandatory part of the employee handbook and require signed acknowledgment from all staff.

  3. [High] Integrate Security Training into Onboarding: Develop a security awareness training module specifically for new hires. This training should be a mandatory part of the onboarding process and cover key topics such as phishing identification, password hygiene, and the new AUP.

  4. [Informational] Maintain and Audit Firewall Configuration: Continue to maintain the current restrictive firewall policy. It is recommended to perform periodic (e.g., quarterly) reviews of firewall rules to ensure they remain aligned with business needs and the principle of least privilege.
