Executive Summary

The security posture of Gilded Cage Design was evaluated through a combination of technical scanning and a review of administrative security controls. The assessment revealed significant gaps that expose the organization to substantial risk.

Key findings include:

  • Critical Gaps in Access Control: Multi-Factor Authentication (MFA) is not enforced for employee email or computer logins. This represents a critical vulnerability, as a single compromised password could lead to a widespread breach of communications and internal systems.

  • Vulnerable External Service: The public-facing web server is running an outdated version of nginx (1.18.0), which has known security vulnerabilities. This service could serve as an initial entry point for an attacker.

  • Administrative Control Deficiencies: The organization lacks a formal Acceptable Use Policy (AUP), a foundational document for guiding employee behavior and enforcing security standards.

  • Configuration Weakness: The SSL certificate presented by the web server does not match the organization’s domain, which can erode user trust and may indicate a server misconfiguration.

While the organization has implemented security awareness training and protects sensitive data systems with MFA, the identified weaknesses require immediate attention. This report outlines actionable recommendations to address each finding.

Analysis of Control Gaps

The questionnaire reveals critical deficiencies in fundamental security practices:

  • Lack of MFA on Email/Computers: The absence of MFA for email and computer access is the most severe finding. Email is a primary target for phishing and account takeover attacks. Without MFA, a compromised password gives an attacker full access. Similarly, unprotected computer logins allow for trivial lateral movement within the network.

  • No Acceptable Use Policy (AUP): An AUP is a crucial administrative control that defines the rules for using company IT assets. Its absence creates ambiguity regarding security responsibilities and acceptable behavior, complicating policy enforcement and incident response.

Recommendations

To address the identified risks, the following actions are recommended in order of priority.

  1. Implement Comprehensive MFA (Priority: Immediate):

    • Deploy a robust Multi-Factor Authentication solution across the entire organization.

    • Prioritize the immediate enforcement of MFA for all email access (e.g., O365, Google Workspace) and all workstation/laptop logins (e.g., Windows Hello, Duo).

  2. Remediate Web Server Vulnerabilities (Priority: Urgent):

    • Upgrade the nginx service on server to the latest stable version to patch known vulnerabilities.

    • Establish a formal patch management policy and schedule for all internet-facing systems to ensure they are updated in a timely manner.

  3. Correct SSL Certificate Configuration (Priority: High):

    • Investigate the certificate mismatch on the web server.

    • Procure and correctly install a valid SSL/TLS certificate that corresponds to the organization’s domain, www.GildedCageDesign.org.

  4. Develop and Implement an AUP (Priority: Medium):

    • Draft a comprehensive Acceptable Use Policy that clearly defines rules for employee use of company technology, data handling, and security responsibilities.

    • Require all current and new employees to read and formally acknowledge the policy.
