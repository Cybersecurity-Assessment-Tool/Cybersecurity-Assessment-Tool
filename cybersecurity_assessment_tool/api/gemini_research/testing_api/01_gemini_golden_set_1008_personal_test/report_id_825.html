Executive Summary

This report provides a cybersecurity posture assessment for Oasis Wellness, conducted on 2026-01-31. The analysis is based on a synthesis of network scan data, organizational security control questionnaires, and a review of pre-existing risk documentation.

The assessment identified a Critical risk finding. An internal network scan of host 10.5.5.5 revealed an open service on port 8080 with the title “TOP SECRET DB”. This suggests a potentially exposed and highly sensitive database. This finding directly contradicts a pre-existing risk assessment which incorrectly labeled the port as secure.

Furthermore, a High risk gap was identified in the organization’s security program: the absence of mandatory security awareness training for new employees during their onboarding process. This gap can lead to unintentional security breaches and misconfigurations, potentially contributing to situations like the one observed with the exposed database.

Immediate remediation is required to investigate and secure the service on port 8080. Recommendations are also provided to strengthen the employee security training program and improve the accuracy of the ongoing risk management process.

Analysis of Control Gaps

The review identified one significant control gap:

  • Lack of Onboarding Security Training: New employees are not receiving security awareness training upon joining the organization. This is a critical period where employees are unfamiliar with corporate policies and are more susceptible to social engineering or making configuration errors. This gap is classified as a High risk.

Recommendations

Based on the correlated findings, the following actions are recommended, prioritized by severity.

Immediate Actions (Critical)

  1. Secure Exposed Database on 10.5.5.5:8080:

    • Immediately investigate the service running on port 8080 to identify the data owner and system purpose.

    • If the service is not essential, take it offline.

    • If essential, implement strict firewall rules and require strong, MFA-based authentication for all access.

Short-Term Actions (High)

  1. Implement Onboarding Security Training:

    • Develop or procure a mandatory security awareness training module for all new hires.

    • Integrate this training into the formal employee onboarding process, to be completed within the first week of employment.

Long-Term Actions (Medium)

  1. Improve Risk Management Process:

    • Conduct a full review of the current risk register to validate all existing entries against current technical data.

    • Establish a formal process for periodic review and re-validation of all identified risks to ensure the register remains an accurate and reliable tool.
