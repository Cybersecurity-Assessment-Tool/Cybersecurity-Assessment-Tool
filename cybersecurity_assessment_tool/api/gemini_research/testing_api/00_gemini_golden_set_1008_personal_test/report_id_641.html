Recommendations

The following actionable recommendations are provided to address the identified risks and significantly improve the overall cybersecurity posture of Binary Star.

  1. Implement Strong Access Controls for RDP:

    • Restrict inbound RDP access for and to only trusted source IP addresses or specific network segments via host-based and perimeter firewalls.

    • Deploy a Virtual Private Network (VPN) solution as the sole method for remote RDP access, requiring users to authenticate to the VPN prior to RDP connection.

    • Ensure all RDP accounts utilize strong, unique passwords and, where feasible, integrate with Multi-Factor Authentication.

    • Conduct an audit to determine if RDP is absolutely necessary on these hosts. If not, disable the service immediately.

  2. Deploy Multi-Factor Authentication (MFA) Systematically Across the Organization:

    • Implement mandatory MFA for all email accounts to provide a critical layer of defense against credential theft and phishing attacks.

    • Enforce MFA for all user logins to computers, network resources, and operating systems to prevent unauthorized access via compromised credentials.

    • Extend MFA protection to all sensitive data systems and applications, safeguarding critical information.

  3. Develop and Enforce a Comprehensive Acceptable Use Policy (AUP):

    • Draft a formal AUP that clearly outlines employee responsibilities, acceptable conduct when using company IT resources, internet usage, and data handling procedures.

    • Ensure all new and existing employees formally review, understand, and acknowledge the AUP, with particular emphasis during onboarding and annual reviews.

  4. Establish Robust and Regular Security Awareness Training Programs:

    • Implement mandatory annual security awareness training for all employees, covering essential topics such as phishing, social engineering, password hygiene, data handling, and incident reporting.

    • Conduct periodic simulated phishing campaigns to test employee vigilance and reinforce the effectiveness of security awareness training.

    • Provide refresher training or micro-learning modules throughout the year to keep security top-of-mind.
