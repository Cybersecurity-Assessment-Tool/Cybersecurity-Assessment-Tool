4.1. Technical Analysis

The Nginx web server running on port 443 (version 1.18.0) is significantly outdated. Nginx 1.18.0 was released in 2020 and is considered End-of-Life (EOL), meaning it no longer receives security patches or updates. Running EOL software exposes the system to numerous known vulnerabilities that can be easily exploited by attackers. This is a critical security concern.

Additionally, the SSL certificate’s common name (www.acme-corp.com) does not align with the organization’s domain (BlueMarble.net). While the target IP is a private address, this mismatch could indicate a misconfiguration, a generic certificate, or a server that is not properly managed or intended for public-facing services under the Blue Marble brand. Further investigation is recommended to ensure proper configuration and ownership.

6. Recommendations

To mitigate the identified risks and enhance Blue Marble’s cybersecurity posture, the following recommendations are provided:

  1. Implement Mandatory MFA:

    • Immediately implement and enforce Multi-Factor Authentication for all email accounts.

    • Implement and enforce Multi-Factor Authentication for all user logins to internal computers and systems.

  2. Update and Patch Software:

    • Promptly update the Nginx web server on to the latest stable, supported version (e.g., Nginx 1.26.x or 1.24.x legacy stable).

    • Establish a robust patch management program to ensure all operating systems, applications, and network devices are regularly updated and patched.

  3. Enhance Security Awareness Training:

    • Develop and implement a mandatory, annual security awareness training program for all employees, not just new hires. This training should cover topics such as phishing, social engineering, data handling, and reporting security incidents.

  4. Review SSL Certificate Configuration:

    • Investigate the SSL certificate on to understand why the common name www.acme-corp.com is present instead of the organization’s domain. Ensure all internal and external certificates are properly configured, valid, and aligned with the intended usage.
