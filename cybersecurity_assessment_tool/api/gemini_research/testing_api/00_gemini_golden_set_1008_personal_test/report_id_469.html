Executive Summary

This report details the cybersecurity posture assessment for Iron Oak Furniture, based on a combination of organizational data, a security questionnaire, and technical network scan results. The assessment reveals a mixed security landscape. While the organization has implemented foundational security controls such as Multi-Factor Authentication (MFA) for email and computer login, an acceptable use policy, and initial security awareness training for new employees, several critical gaps and vulnerabilities have been identified.

Key findings include:

  • Organizational Gaps: Lack of MFA for sensitive data systems and no annual security awareness training for all employees pose significant risks related to unauthorized access and human-factor vulnerabilities.

  • Technical Vulnerabilities: A network scan identified an internal web server running an outdated version of Nginx (1.18.0), which is susceptible to known security flaws. Furthermore, a critical SSL certificate misconfiguration was found, displaying a common name (www.acme-corp.com) that does not match the organization’s domain (IronOakFurniture.com), leading to potential trust issues and weakened TLS security.

These issues require immediate attention to enhance the overall security posture and mitigate potential threats. Comprehensive recommendations are provided in Section 6.

Risk Assessment

R-01 Lack of MFA for Sensitive Data Systems Critical The organization does not enforce Multi-Factor Authentication (MFA) for access to systems containing sensitive data. This significantly increases the risk of unauthorized access in the event of credential compromise, potentially leading to data breaches or system infiltration.
R-02 Inadequate Annual Security Awareness Training High The organization does not conduct mandatory security awareness training for all employees at least once per year. This oversight can lead to a degradation of security posture over time, making employees more susceptible to phishing, social engineering, and other human-factor attacks.

Recommendations

The following actionable recommendations are provided to address the identified risks and enhance the overall cybersecurity posture of Iron Oak Furniture:

  • R-01: Implement MFA for Sensitive Data Systems

    • Action: Immediately implement and enforce Multi-Factor Authentication (MFA) for all systems that store, process, or transmit sensitive organizational or customer data.

    • Detail: Conduct an inventory of all sensitive data systems and ensure that appropriate MFA solutions are integrated and mandatory for all user accounts accessing these systems.

  • R-02: Establish Annual Security Awareness Training

    • Action: Develop and implement a mandatory annual security awareness training program for all employees.

    • Detail: The training should cover current threats (e.g., phishing, ransomware), best practices for data handling, password hygiene, and incident reporting procedures. Regularly update training content to reflect evolving threat landscapes.

  • R-03: Update Nginx Web Server Software

    • Action: Update the Nginx web server on to the latest stable version (e.g., Nginx 1.24.x or newer mainline).

    • Detail: Before deployment, thoroughly test the update in a staging environment to ensure compatibility with existing applications. Establish a regular patch management process for all server software.

  • R-04: Correct SSL Certificate Misconfiguration

    • Action: Replace the SSL certificate on the Nginx server () with one that correctly reflects the organization’s domain.

    • Detail: Generate a new SSL certificate with a Common Name (CN) matching (or appropriate subdomain if it’s an internal-only service with an internal CA). Ensure proper certificate chain installation and validate its functionality to eliminate browser warnings and restore trust. If the server’s purpose is different and unrelated to , ensure its configuration is documented and secured according to its intended use.
