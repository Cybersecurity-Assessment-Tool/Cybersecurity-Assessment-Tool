Executive Summary

This report details the findings from a cybersecurity posture assessment for Urban Jungle Planning. The assessment involved a comprehensive review of organizational security controls through a questionnaire, analysis of a network scan, and consideration of pre-existing vulnerabilities.

The overall security posture is assessed as Weak. Several critical vulnerabilities and significant control gaps were identified, including the presence of an outdated and vulnerable FTP server allowing anonymous access, a lack of Multi-Factor Authentication (MFA) for email access, and the continued use of End-of-Life (EOL) operating systems. These issues collectively represent a substantial risk to the organization’s data integrity, confidentiality, and availability. Immediate action is recommended to address the identified critical and high-severity risks.

Analysis of Questionnaire Gaps

The questionnaire highlights critical gaps in the organization’s security posture:

  • No MFA for Email Access: Email is a primary vector for cyberattacks. The absence of MFA significantly increases the risk of account compromise through phishing or credential stuffing attacks. This is a Critical finding.

  • No Employee Acceptable Use Policy: Without a formal policy, employees may be unaware of their responsibilities regarding IT asset usage, potentially leading to security incidents or non-compliance. This is a High finding.

  • No Security Awareness Training for New Employees: New hires are often vulnerable targets. Lack of initial training leaves them unprepared to recognize and respond to common cybersecurity threats. This is a High finding.

Analysis of Technical Vulnerabilities

The network scan revealed two severe vulnerabilities associated with the FTP service:

  • Outdated and Vulnerable FTP Service: The vsftpd version 2.3.4 is known to contain a critical backdoor vulnerability (CVE-2011-2523), allowing unauthenticated remote code execution. This is an extremely Critical risk finding.

  • Anonymous FTP Login Allowed: The FTP server is configured to allow anonymous logins. This can lead to unauthorized access to files, potential data exfiltration, or even the upload of malicious content, depending on the server configuration. This is also a Critical risk finding.

Recommendations

To mitigate the identified risks and improve Urban Jungle Planning’s cybersecurity posture, the following recommendations are provided. Priority should be given to critical and high-severity items.

  1. Disable Anonymous FTP Access: Immediately reconfigure the FTP server to disable anonymous login. If FTP is required, ensure strong authentication is enforced and access is strictly controlled and logged. Consider replacing FTP with SFTP or SCP for secure file transfers.

  2. Upgrade/Replace Outdated vsftpd Service: Urgently upgrade the vsftpd service to a currently supported and secure version, or replace it with a more modern, secure file transfer solution. If the service is not essential, consider decommissioning it entirely to eliminate the risk.

  3. Implement Mandatory MFA for Email: Enforce Multi-Factor Authentication (MFA) for all organizational email accounts. This provides a crucial layer of defense against credential theft and unauthorized access.

  4. Upgrade End-of-Life Operating Systems: Plan and execute an upgrade strategy for all workstations running Windows 7 to a currently supported operating system (e.g., Windows 10/11). Ensure systems are regularly patched and updated.

  5. Develop and Implement an Acceptable Use Policy (AUP): Create a comprehensive AUP that clearly outlines employee responsibilities and acceptable behaviors for using company IT resources. Distribute the policy to all employees and obtain signed acknowledgments.

  6. Establish Security Awareness Training for New Hires: Implement a mandatory security awareness training program as part of the onboarding process for all new employees. This training should cover common threats, company policies, and best practices.
