Executive Summary

This report details the cybersecurity posture of Willow Creek Health based on an organizational questionnaire, an external network scan, and a review of existing known risks. Our analysis reveals several critical vulnerabilities and significant control gaps, particularly concerning multi-factor authentication (MFA), employee policies, security awareness training, and an exposed, outdated, and insecure FTP service. Urgent attention is required to address the identified critical risks, including the anonymous access to an outdated FTP server, the lack of an Acceptable Use Policy, and insufficient MFA deployment.

5. Recommendations

Based on the assessment, the following recommendations are provided to enhance Willow Creek Health’s cybersecurity posture.

  • Immediate Action: Secure FTP Service

    • Disable Anonymous Access: Immediately disable anonymous login for the FTP service on .

    • Upgrade/Replace FTP Service: Urgently upgrade vsftpd to the latest stable and secure version, or migrate to a more secure file transfer solution (e.g., SFTP, SCP) if FTP is still required. If FTP is not essential for business operations, disable the service entirely.

  • Enhance Multi-Factor Authentication (MFA)

    • Implement MFA for Email: Mandate MFA for all email accounts across the organization to prevent unauthorized access and credential compromise.

    • Implement MFA for Sensitive Data Systems: Extend MFA requirements to all systems handling sensitive organizational or patient data to protect against data breaches.

  • Strengthen Organizational Policies and Training

    • Develop and Enforce an Acceptable Use Policy: Create and disseminate a clear AUP for all employees, ensuring they understand their responsibilities regarding IT asset usage, data handling, and security best practices. Conduct mandatory acknowledgment.

    • Conduct Annual Security Awareness Training: Implement mandatory annual security awareness training for all employees, covering current threats, phishing, social engineering, and organizational security policies.

  • Address Outdated Systems

    • Upgrade/Migrate Windows 7 Systems: Plan and execute an immediate upgrade of all Windows 7 workstations to a currently supported operating system (e.g., Windows 10/11) to mitigate known vulnerabilities and maintain security patching. Consider hardware refresh if necessary.
