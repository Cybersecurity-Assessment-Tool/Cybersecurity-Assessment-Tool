Running EOL software poses a significant security risk due to the lack of ongoing security patches and support for newly discovered vulnerabilities.

Recommendations

To address the identified risks and enhance the overall cybersecurity posture of True Grit, the following recommendations are provided:

  • Implement MFA for Computer Logins (High Priority):

    • Mandate Multi-Factor Authentication (MFA) for all computer login processes across the organization. This significantly reduces the risk of unauthorized access even if primary credentials are stolen, aligning with industry best practices.

  • Establish New Employee Security Awareness Training (High Priority):

    • Develop and implement a mandatory security awareness training program for all new employees during their onboarding process. This should cover acceptable use policies, phishing recognition, password best practices, social engineering awareness, and data handling procedures.

  • Upgrade Nginx Web Server (Medium Priority):

    • Plan and execute an upgrade of the Nginx web server from version 1.18.0 to the latest stable or mainline version (e.g., Nginx 1.24.x or 1.25.x). Ensure thorough testing in a staging environment before deploying to production to minimize service disruption.

    • Establish a regular patching and update schedule for all critical server software and operating systems.

  • Address SSL Certificate Mismatch (Medium Priority):

    • Investigate the SSL certificate configuration on the web server. Replace the current certificate with one that correctly matches the organization’s domain name ().

    • Implement a robust certificate lifecycle management process to ensure certificates are renewed before expiry, correctly configured, and properly installed across all relevant services.

  • Regular Security Audits:

    • Conduct periodic internal and external vulnerability assessments and penetration tests (VAPT) to proactively identify and address security weaknesses across the IT infrastructure.

Conclusion

True Grit has a foundational understanding of some security controls, but critical gaps in MFA for computer logins and new employee security training, combined with an outdated web server and an SSL certificate mismatch, present significant risks. By diligently implementing the recommendations outlined in this report, True Grit can substantially strengthen its cybersecurity defenses and reduce its attack surface, thereby protecting its assets and maintaining trust with its users and stakeholders.
