3.1 Analysis of Questionnaire Findings

While Neon Pulse Entertainment demonstrates good practices in several areas, particularly concerning general MFA adoption for email and computer login, and security awareness training, the lack of MFA for sensitive data systems stands out as a high-priority risk. This gap could allow an attacker with stolen credentials to gain unauthorized access to critical organizational data.

4. Technical Scan Results

A targeted network scan was performed on 10.0.0.15. The scan revealed the following critical findings:

  • Scan Target: 10.0.0.15

  • Open Port: 21/tcp (FTP)

  • Service/Product: FTP / vsftpd 2.3.4

  • Critical Finding: Anonymous FTP login is allowed.

  • Vulnerability Details:

    • The vsftpd 2.3.4 service is significantly outdated. This version was released in 2011 and has several known vulnerabilities, including critical remote code execution flaws (e.g., CVE-2011-2523). Running such an old version poses a severe security risk.

    • The allowance of anonymous FTP login means that any user can connect to the FTP server without providing credentials. This can lead to unauthorized access, data leakage, and potential for an attacker to upload malicious files or exploit the outdated service more easily.

6. Recommendations

The following recommendations are provided to address the identified risks and improve the overall cybersecurity posture of Neon Pulse Entertainment.

6.1 Immediate Technical Recommendations

  • Disable Anonymous FTP: Immediately disable anonymous FTP login on 10.0.0.15. If FTP access is required, ensure it is restricted to authenticated users with strong, unique credentials.

  • Upgrade/Replace FTP Service: Urgently upgrade vsftpd to the latest stable version, or preferably, replace it with a more secure file transfer protocol like SFTP (SSH File Transfer Protocol) or SCP (Secure Copy Protocol) which encrypt data in transit and require strong authentication.

  • Vulnerability Patching: Conduct a thorough vulnerability scan on 10.0.0.15 and other internal systems to identify and patch all outdated software and operating systems.

  • Firewall Configuration Review: Review firewall rules for 10.0.0.15 to ensure only necessary and authorized traffic can reach the system.

6.2 Policy and Procedure Recommendations

  • Implement MFA for Sensitive Data Systems: Mandate and implement Multi-Factor Authentication (MFA) for all systems that handle sensitive organizational data. This significantly reduces the risk of credential-based attacks.

  • Operating System Upgrade Plan: Develop and execute a plan to upgrade all computers currently running Windows 7 to a supported operating system (e.g., Windows 10 or 11) to mitigate risks associated with end-of-life software.

  • Regular Security Audits: Implement a schedule for regular internal and external cybersecurity audits and penetration testing to proactively identify and address vulnerabilities.

  • Software Lifecycle Management: Establish a policy for regular review and update of all software and services to ensure they remain supported and patched.

7. Conclusion

Neon Pulse Entertainment faces significant cybersecurity risks primarily due to an exposed and outdated FTP service with anonymous access, and a critical organizational control gap regarding MFA for sensitive data systems. Addressing these issues immediately and implementing the provided recommendations will substantially enhance the organization’s defense against potential cyber threats and improve its overall security posture.
