Executive Summary

This report details the cybersecurity posture assessment for Borealis Tech, based on a review of organizational data, a recent network scan, and existing risk documentation. The assessment reveals several critical gaps in security controls and identifies a significant discrepancy regarding a potentially exposed sensitive data system.

Key findings include:

  • Critical Gaps in Security Controls: A lack of Multi-Factor Authentication (MFA) for email access and sensitive data systems, absence of an Acceptable Use Policy, and insufficient security awareness training for employees pose significant risks.

  • Exposed Sensitive Service: A network scan identified an open port 8080 on target 10.5.5.5 exposing a service with the title "TOP SECRET DB". This finding critically contradicts existing internal risk documentation that classifies this port as "secure" and a "false positive".

  • Risk Register Inaccuracy: The discrepancy regarding Port 8080 indicates a potential flaw in the current risk management process or a failure to update documentation following live scan findings.

Immediate action is recommended to address the identified control gaps and critically re-evaluate the status of the exposed service to prevent unauthorized access to sensitive information.

Recommendations

To enhance Borealis Tech’s cybersecurity posture and mitigate the identified risks, the following recommendations are provided:

Immediate Actions

  • Investigate and Secure Port 8080 ():

    • Immediately verify the nature and contents of the service exposed on :8080.

    • If it contains sensitive data, implement stringent access controls (e.g., firewall rules, network segmentation, IP whitelisting).

    • Implement strong authentication mechanisms, including Multi-Factor Authentication (MFA), for access to this service.

    • If the service is not intended to be operational or accessible, shut it down or decommission it securely.

    • Update the organizational risk register to accurately reflect the live status and associated risks of this port and service.

Security Control Enhancements

  • Implement Multi-Factor Authentication (MFA):

    • Deploy MFA for all email accounts to prevent unauthorized access via compromised credentials.

    • Extend MFA implementation to all sensitive data systems and applications to protect critical information assets.

  • Develop and Implement an Acceptable Use Policy (AUP):

    • Create a clear and comprehensive AUP that outlines acceptable and unacceptable use of company IT resources, data, and networks.

    • Ensure all employees read, understand, and formally acknowledge the AUP.

  • Establish Comprehensive Security Awareness Training:

    • Implement mandatory security awareness training for all new employees during their onboarding process.

    • Conduct annual, mandatory security awareness training for all employees, covering current threats such as phishing, social engineering, and safe data handling practices.

    • Implement regular simulated phishing campaigns to test employee vigilance and provide targeted training.

General Security Best Practices

  • Regular Vulnerability Scanning: Conduct regular internal and external vulnerability scans to identify and remediate security weaknesses proactively.

  • Penetration Testing: Schedule periodic penetration tests by independent third parties to simulate real-world attacks and uncover exploitable vulnerabilities.

  • Risk Register Management: Ensure the risk register is regularly reviewed and updated with the latest assessment findings and remediation statuses to maintain an accurate view of the organization’s risk landscape.
