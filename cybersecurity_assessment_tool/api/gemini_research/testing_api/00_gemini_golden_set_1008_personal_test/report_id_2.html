Recommendations

To mitigate the identified risks and improve the overall cybersecurity posture of , the following recommendations are provided. These are prioritized by severity.

Critical Recommendations

  • Decommission or Secure FTP Service: Immediately shut down the 21product 21version service. If FTP functionality is absolutely required, migrate to a modern, secure file transfer solution (e.g., SFTP, FTPS with a current, patched server software) and ensure it is properly configured with strong authentication, encryption, and strict access controls.

  • Disable Anonymous FTP Access: If the FTP service is temporarily retained (not recommended), disable anonymous login immediately. Ensure no sensitive information is stored in publicly accessible FTP directories.

High Recommendations

  • Implement MFA for Sensitive Data Systems: Deploy and enforce Multi-Factor Authentication (MFA) for all systems that handle or store sensitive organizational data. This significantly reduces the risk of credential compromise leading to data breaches.

  • Mandatory Security Awareness Training for New Employees: Establish a mandatory and comprehensive security awareness training program for all new hires. This training should cover essential topics such as phishing, social engineering, password hygiene, and company security policies, and should be completed prior to granting access to IT resources.

  • Annual Security Awareness Training for All Employees: Conduct comprehensive and engaging security awareness training for all employees at least once annually. This training should be interactive, reflect the current threat landscape, and be tailored to the organization’s specific risks and roles.

Medium Recommendations

  • Upgrade End-of-Life Operating Systems: Initiate a project to upgrade all workstations currently running Windows 7 to a fully supported and patched operating system, such as Windows 10 or 11. Implement a robust patch management program for all systems.

  • Develop and Implement an Acceptable Use Policy (AUP): Create a formal Employee Acceptable Use Policy (AUP) that clearly defines the rules and expectations for employee conduct when using company IT assets and networks. Ensure all employees acknowledge, understand, and adhere to this policy.

Conclusion

The assessment highlights critical areas where ’s cybersecurity defenses are weak. By prioritizing and implementing the recommendations outlined in this report, the organization can significantly enhance its security posture, reduce its exposure to known vulnerabilities, and build a more resilient defense against evolving cyber threats. Continuous monitoring, regular security assessments, and ongoing employee training are essential for maintaining a strong and adaptive security posture.
