Analysis of Security Control Review

The organization demonstrates a strong commitment to security awareness training and policy enforcement, which are foundational elements of a robust security program. However, significant vulnerabilities exist concerning Multi-Factor Authentication:

  • MFA for Email Access: The absence of MFA for email access is a critical risk. Email is a primary vector for phishing, business email compromise (BEC), and credential theft. Compromised email accounts can lead to widespread organizational compromise.

  • MFA for Computer Logins: The lack of MFA for computer logins also represents a high risk. Without a second factor, a stolen or guessed password could grant an attacker direct access to internal systems.

While MFA is in place for sensitive data systems, the broader lack of MFA for everyday access points significantly increases the attack surface.

Recommendations

The following actionable recommendations are provided to enhance the cybersecurity posture of Astraeus Aerospace:

  1. Implement Multi-Factor Authentication (MFA) for Email Access:

    • Action: Immediately implement MFA for all user accounts accessing organizational email.

    • Benefit: Significantly reduces the risk of email account compromise, even if passwords are stolen or guessed.

    • Priority: Critical.

  2. Implement Multi-Factor Authentication (MFA) for Computer Logins:

    • Action: Deploy MFA for all user logins to company computers, both on-premises and remote.

    • Benefit: Protects against unauthorized access to workstations and internal systems, even if user passwords are compromised.

    • Priority: Critical.

  3. Conduct a Comprehensive External Network Scan:

    • Action: Perform a thorough Nmap scan or similar reconnaissance against the organization’s external IP addresses and domains.

    • Benefit: Identifies open ports, running services, and potential vulnerabilities exposed to the internet. This will allow for proper assessment and remediation of technical risks.

    • Priority: High.

  4. Regularly Review and Update Security Policies:

    • Action: Although an AUP is in place, ensure all security policies (e.g., password policy, data handling) are reviewed and updated annually, or whenever significant changes occur.

    • Benefit: Keeps security practices aligned with evolving threats and organizational needs.

    • Priority: Medium.

  5. Maintain Consistent Security Awareness Training:

    • Action: Continue with and expand existing security awareness training programs, especially incorporating topics like phishing recognition and the importance of MFA.

    • Benefit: Empowers employees to be the first line of defense against cyber threats.

    • Priority: Ongoing.

Conclusion

Astraeus Aerospace has a foundational commitment to security awareness and policy. However, the lack of MFA for critical access points (email and computer logins) presents immediate and high-priority risks that must be addressed. Furthermore, a detailed technical assessment of the external network perimeter is crucial to identify and mitigate unknown exposures. Addressing these areas will significantly strengthen the organization’s overall cybersecurity posture against common attack vectors.
