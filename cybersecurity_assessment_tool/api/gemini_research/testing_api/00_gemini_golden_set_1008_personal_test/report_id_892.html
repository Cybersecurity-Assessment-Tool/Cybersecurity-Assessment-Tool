Recommendations

To mitigate the identified risks and enhance the organization’s cybersecurity posture, the following recommendations are provided:

  1. Implement MFA for Computer Logins:

    • Mandate and enforce Multi-Factor Authentication for all user accounts logging into company computers and workstations. This significantly reduces the risk of successful credential-based attacks.

    • Evaluate suitable MFA solutions that integrate with existing identity providers and operating systems.

  2. Establish Annual Security Awareness Training Program:

    • Develop and implement a mandatory annual (or more frequent) security awareness training program for all employees, not just new hires.

    • Training should cover topics such as phishing, social engineering, password hygiene, data handling best practices, and incident reporting procedures.

    • Utilize engaging and up-to-date content to ensure effectiveness.

  3. Conduct Comprehensive Vulnerability Scanning:

    • Perform regular, authenticated vulnerability scans (both internal and external) to identify potential weaknesses in network devices, servers, and applications.

    • Include deeper penetration testing to simulate real-world attacks.

  4. Regularly Review and Update Security Policies:

    • Periodically review and update all security policies, including the acceptable use policy, to ensure they remain current with evolving threats and organizational changes.

    • Ensure policies are communicated effectively to all employees.

Conclusion

While Vertex Solutions Vertex Solutions demonstrates foundational security practices, the identified high risks related to MFA for computer logins and recurring security awareness training require immediate attention. Addressing these critical gaps will significantly strengthen the organization’s defense against common cyber threats. Continuous monitoring, regular assessments, and a proactive approach to cybersecurity are crucial for maintaining a resilient security posture.
