Risk Assessment

This section consolidates risks identified from the security control review and pre-existing organizational vulnerability data. Each risk is categorized by its severity.

Risk Name Overview Severity
Lack of MFA for Email Access Multi-factor authentication (MFA) is not required for accessing organizational email accounts. This significantly increases the risk of account compromise through phishing, brute-force attacks, or stolen credentials.
Lack of MFA for Computer Login Multi-factor authentication (MFA) is not required for logging into organizational computers. This leaves internal systems vulnerable to unauthorized access if user credentials are stolen or guessed.
Insufficient Annual Security Awareness Training Security awareness training is not conducted for all employees at least once per year. This can lead to a gradual decay in security hygiene and increased susceptibility to social engineering attacks over time.
Unencrypted Web Server An existing organizational risk identifies an unencrypted web server where port 80 is considered open. While the latest scan data indicates port 80 is currently closed on the scanned target, the organization has identified this as a standing risk that warrants review if services were to be re-enabled or if other unencrypted web services exist.

Recommendations

Based on the assessment findings, the following recommendations are provided to improve Hearth & Home’s cybersecurity posture:

Critical Recommendations

  • Implement MFA for Email Access:

    • Immediately enable and enforce Multi-Factor Authentication (MFA) for all organizational email accounts. This is a fundamental control against credential theft.

  • Implement MFA for Computer Login:

    • Deploy and require MFA for all user logins to company computers and workstations. This adds a critical layer of defense against unauthorized access to internal systems.

  • Mandate Annual Security Awareness Training:

    • Establish a mandatory program for annual security awareness training for all employees, not just new hires. Regular training ensures that employees remain vigilant against evolving threats.

    • Consider regular phishing simulations to test and reinforce training effectiveness.

High and Medium Priority Recommendations

  • Review and Address Unencrypted Web Services:

    • Investigate the existing "Unencrypted Web Server" risk. If the service is intentionally offline, verify its status and ensure it remains so unless secured.

    • For any web service that needs to be publicly accessible, ensure it is configured to use HTTPS with strong encryption protocols (TLS 1.2 or higher) and valid certificates. Avoid serving content over plain HTTP.

    • Regularly scan for unencrypted web services across the network to prevent accidental deployment or misconfiguration.

Conclusion

Hearth & Home demonstrates a foundational understanding of some security practices, as evidenced by an acceptable use policy and initial security awareness training. However, critical vulnerabilities exist primarily in the widespread adoption of Multi-Factor Authentication and consistent, annual security awareness training. Addressing these gaps, alongside a proactive review of legacy risks like unencrypted web servers, will significantly bolster the organization’s defense against common cyber threats. Continued vigilance and adherence to these recommendations are crucial for maintaining a strong and resilient security posture.
