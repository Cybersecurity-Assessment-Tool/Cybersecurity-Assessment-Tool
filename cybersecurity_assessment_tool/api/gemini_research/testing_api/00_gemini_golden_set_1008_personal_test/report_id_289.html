1. Executive Summary

This report details the findings of a cybersecurity posture assessment for Ember Glow Hospitality. The assessment involved a review of organizational security controls through a questionnaire, a technical network scan, and an analysis of existing risk data.

Key strengths identified include robust security awareness training, an acceptable use policy, and the implementation of Multi-Factor Authentication (MFA) for email and sensitive data access. However, critical gaps were identified regarding the lack of MFA for computer logins and the use of outdated web server software. The SSL certificate common name also indicates a potential misconfiguration. These issues pose significant risks to the organization’s security posture and require prompt attention.

6. Recommendations

The following recommendations are provided to mitigate the identified risks and improve the overall security posture of Ember Glow Hospitality.

  1. Implement MFA for Computer Logins:

    • Action: Immediately implement and enforce Multi-Factor Authentication (MFA) for all user accounts logging into company computers and workstations. This should cover all operating systems and critical applications.

    • Rationale: MFA provides an essential layer of security, significantly reducing the risk of successful credential stuffing, phishing, or brute-force attacks by requiring an additional verification factor beyond a password.

    • Priority: Critical

  2. Update Web Server Software:

    • Action: Upgrade the Nginx web server from version 1.18.0 to the latest stable release (e.g., Nginx 1.24.x or 1.26.x series, depending on the desired long-term support). Ensure all security patches are applied during the upgrade process. Plan for regular updates going forward.

    • Rationale: Running outdated software, especially a public-facing web server, exposes the organization to numerous known vulnerabilities that attackers can exploit. Regular updates ensure access to the latest security fixes, performance improvements, and features.

    • Priority: High

  3. Review and Correct SSL Certificate Configuration:

    • Action: Investigate why the SSL certificate for the HTTPS service on port 443 is for instead of . Replace the certificate with one issued specifically for Ember Glow Hospitality’s domain to ensure proper trust and avoid browser warnings.

    • Rationale: An incorrect SSL certificate common name leads to browser warnings for users, which can erode trust and indicate a potential security risk or misconfiguration. It is crucial for maintaining a professional and secure online presence.

    • Priority: Medium

7. Conclusion

Ember Glow Hospitality demonstrates a foundational commitment to cybersecurity through its security awareness training and acceptable use policies. However, the assessment has identified critical vulnerabilities related to MFA implementation for computer logins, outdated web server software, and an SSL certificate mismatch. Addressing these recommendations promptly and diligently will substantially enhance the organization’s resilience against cyber threats and bolster its overall security posture.
