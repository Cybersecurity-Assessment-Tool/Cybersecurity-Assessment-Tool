Executive Summary

This report provides a comprehensive cybersecurity posture assessment for Terraform Global, based on an organizational security questionnaire review and an initial network scan. The assessment identified several critical gaps in security controls, particularly concerning multi-factor authentication for computer logins, the absence of an employee acceptable use policy, and a lack of security awareness training for new employees. While the initial network scan of a specified internal host revealed no open ports, indicating a baseline level of network hygiene for that particular system, the identified policy and training deficiencies represent significant human and process-related risks that require immediate attention. Remedial recommendations are provided to enhance the overall security posture.

Recommendations

To address the identified risks and enhance the overall cybersecurity posture, the following recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) for Computer Logins:

    • Action: Mandate MFA for all user logins to company computers and endpoints.

    • Benefit: Significantly reduces the risk of unauthorized access even if primary credentials are compromised.

    • Priority: Critical

  2. Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Action: Create a clear, comprehensive AUP that outlines acceptable use of company IT resources, data handling procedures, and security expectations. Disseminate it to all employees and obtain signed acknowledgments.

    • Benefit: Establishes clear guidelines, reduces misuse, and strengthens the legal and compliance framework.

    • Priority: High

  3. Initiate Security Awareness Training for New Employees:

    • Action: Implement mandatory security awareness training as part of the onboarding process for all new hires. This should cover common threats (e.g., phishing, social engineering), reporting procedures, and the importance of security.

    • Benefit: Equips new employees with essential security knowledge from day one, reducing the human factor risk.

    • Priority: High

  4. Conduct Comprehensive Internal Network Scans:

    • Action: Extend network scanning efforts to cover all internal network segments and devices. Perform authenticated scans where possible to identify software versions, configurations, and vulnerabilities.

    • Benefit: Provides a more complete picture of the internal attack surface and helps identify misconfigurations or outdated software.

    • Priority: Medium

Conclusion

While Terraform Global demonstrates some positive security practices (e.g., MFA for email and sensitive systems, annual security training), significant vulnerabilities exist in key areas of identity and access management, policy enforcement, and initial employee training. Addressing these identified risks with the provided recommendations will substantially enhance the organization’s cybersecurity resilience against common threats. Continuous monitoring, regular security assessments, and ongoing employee education are vital for maintaining a strong security posture.
