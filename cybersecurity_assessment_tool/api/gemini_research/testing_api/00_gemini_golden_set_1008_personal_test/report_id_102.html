Recommendations

The following recommendations are provided to address the identified risks and strengthen the organization’s cybersecurity posture.

Immediate Actions (High Priority)

  • Implement MFA for Email Access: Mandate multi-factor authentication for all email accounts across the organization. This is a critical control against credential compromise.

  • Implement MFA for Sensitive Data Systems: Deploy and enforce multi-factor authentication for all systems and applications that store or process sensitive organizational data.

  • Introduce Security Awareness Training for New Employees: Establish a mandatory security awareness training program for all new hires, to be completed as part of their onboarding process. This should cover common threats like phishing, social engineering, and safe online practices.

Ongoing and Long-Term Initiatives

  • Regular Review of Security Controls: Periodically review and update security control questions and answers to ensure they reflect the current security posture and evolving threat landscape.

  • Continuous Monitoring: Implement continuous monitoring solutions to detect changes in network exposure (e.g., newly opened ports, services) and promptly identify potential vulnerabilities.

  • Incident Response Planning: Develop or refine an incident response plan to ensure the organization can effectively respond to and recover from cybersecurity incidents.

Conclusion

Oasis Wellness demonstrates a foundational understanding of cybersecurity through existing policies and annual training. However, the identified gaps in multi-factor authentication and new employee security training present significant risks that require urgent attention. By implementing the recommended actions, Oasis Wellness can substantially reduce its attack surface and improve its resilience against common cyber threats.
