1. Executive Summary

This report provides a comprehensive cybersecurity posture assessment for Nexus Dynamics, based on an analysis of network scan data, organizational security questionnaire responses, and a review of existing identified risks. The assessment reveals several critical vulnerabilities and significant security control gaps that require immediate attention. Key findings include the presence of an outdated and vulnerable FTP server allowing anonymous access, a lack of multi-factor authentication for computer logins, and the continued use of unsupported operating systems. Without addressing these issues, the organization faces a high risk of data breaches, unauthorized access, and operational disruption. Recommendations are provided to mitigate these risks and strengthen the overall security posture.

6. Recommendations

Based on the identified risks, the following recommendations are provided to improve Nexus Dynamicsâ€™ cybersecurity posture.

  1. Address Critical FTP Vulnerabilities:

    • Immediately upgrade the vsftpd service to the latest stable version or replace it with a more secure alternative if FTP is still required.

    • Disable anonymous FTP login. If file sharing is necessary, implement authenticated access with strong passwords and access controls.

    • Review the necessity of running an FTP server on an internal network segment, especially with direct exposure. Consider more secure alternatives like SFTP or cloud storage solutions.

  2. Implement Multi-Factor Authentication (MFA) for All Computer Logins:

    • Deploy and enforce MFA for all user logins to company computers and endpoints. This significantly reduces the risk of unauthorized access due to compromised credentials.

  3. Upgrade Outdated Operating Systems:

    • Prioritize the upgrade of all Windows 7 workstations to a currently supported operating system (e.g., Windows 10 or 11).

    • Develop a plan for regular OS patching and lifecycle management to prevent similar issues in the future.

  4. Develop and Enforce an Employee Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that clearly outlines acceptable and unacceptable use of company IT resources, data, and internet access.

    • Ensure all employees review and acknowledge the AUP, and integrate it into onboarding processes.

  5. Establish Regular Security Awareness Training:

    • Implement mandatory annual security awareness training for all employees, in addition to new employee training.

    • Training should cover topics such as phishing, social engineering, password hygiene, data handling, and company policies.

  6. Regular Network Scanning and Vulnerability Management:

    • Implement a routine vulnerability scanning program for both internal and external assets.

    • Establish a formal vulnerability management process to track, prioritize, and remediate identified vulnerabilities.

7. Conclusion

Nexus Dynamics exhibits a security posture that requires immediate and significant improvements. The identified critical technical vulnerabilities, coupled with crucial gaps in organizational security controls, expose the organization to a high level of risk. By systematically addressing the recommendations outlined in this report, particularly the FTP server issues and the lack of MFA for computer access, Nexus Dynamics can substantially enhance its resilience against cyber threats and protect its valuable assets.
