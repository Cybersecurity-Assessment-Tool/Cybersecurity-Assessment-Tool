Executive Summary

This report details the findings of a cybersecurity posture assessment for Harbor Light Foundation. The assessment involved a review of organizational security controls through a questionnaire, an external network scan, and an analysis of pre-existing risk data. Our analysis reveals several critical vulnerabilities and significant control deficiencies that collectively pose a substantial risk to the organization’s data integrity, confidentiality, and availability. Key findings include the presence of an outdated FTP server allowing anonymous access, a severe lack of multi-factor authentication for critical systems, and a complete absence of security awareness training and acceptable use policies. Urgent remediation efforts across all identified areas are strongly recommended to mitigate these risks and enhance the overall security posture.

Analysis of Control Gaps

The questionnaire results indicate severe deficiencies in fundamental security practices:

  • Multi-Factor Authentication (MFA): The absence of MFA for email and sensitive data systems leaves these critical assets highly vulnerable to credential stuffing, phishing, and brute-force attacks. While MFA for computer login is a positive step, its omission for other high-value targets creates significant weak points.

  • Acceptable Use Policy (AUP): The lack of an AUP means employees may not have clear guidelines regarding the appropriate use of IT assets, potentially leading to misuse, unauthorized data handling, and compliance risks.

  • Security Awareness Training (SAT): A complete absence of SAT for both new and existing employees is a critical vulnerability. Employees are often the primary target for cyberattacks, and without proper training, they are highly susceptible to social engineering, phishing campaigns, and malware, which can compromise the entire organization.

Technical Analysis

  • Port 21 (FTP) - Anonymous Access and Outdated Software:

    • Outdated Software: The FTP service is running vsftpd version 2.3.4. This version was released in 2011 and is End-of-Life (EOL). It has multiple known critical vulnerabilities, most notably CVE-2011-2523, a severe backdoor vulnerability that allows arbitrary command execution. Running such an old and unpatched version poses an extreme risk.

    • Anonymous Access: The FTP server is configured to allow anonymous logins. This configuration permits unauthenticated users to access, and potentially modify or upload files, on the server. This is a severe security misconfiguration that can lead to unauthorized data exposure, data leakage, and potential for malicious file distribution.

Recommendations

Based on the identified risks, the following prioritized and actionable recommendations are provided to enhance the cybersecurity posture of Harbor Light Foundation.

Critical/High Priority Recommendations

  1. Remediate Anonymous FTP Access and Outdated Server (Critical/High):

    • Immediate Action: Disable anonymous FTP access immediately. Evaluate the necessity of the FTP service; if not strictly required, disable or remove it completely.

    • Long-term Solution: If FTP is essential, upgrade to a current, supported version of vsftpd or replace it with a more secure file transfer protocol (e.g., SFTP, SCP) over SSH. Ensure proper authentication, authorization, and encryption are consistently in place.

  2. Implement Multi-Factor Authentication (MFA) Broadly (High):

    • Email Systems: Mandate MFA for all email accounts to prevent unauthorized access via compromised credentials.

    • Sensitive Data Systems: Implement MFA for all access to systems containing or processing sensitive data.

  3. Establish Comprehensive Security Awareness Training (SAT) (High):

    • Develop and implement mandatory security awareness training for all new employees as part of their onboarding process.

    • Conduct annual mandatory security awareness training for all employees to keep them informed about current threats (e.g., phishing, ransomware) and best practices.

Medium Priority Recommendations

  1. Develop and Enforce an Acceptable Use Policy (AUP) (Medium):

    • Create a clear AUP that outlines employee responsibilities and permissible use of organizational IT assets, data, and networks.

    • Ensure all employees read, understand, and formally acknowledge the AUP, integrating it into onboarding and annual policy reviews.

  2. Upgrade Outdated Operating Systems (Medium):

    • Migrate all Windows 7 workstations to a currently supported operating system (e.g., Windows 10 or 11) to ensure receipt of critical security updates and patches.

    • Develop a hardware refresh strategy if current hardware does not support newer OS versions, or explore virtualization options.

Conclusion

The current cybersecurity posture of Harbor Light Foundation is significantly compromised by critical vulnerabilities and substantial gaps in foundational security controls. Addressing the identified issues, particularly the anonymous FTP access, severely outdated software, and the absence of Multi-Factor Authentication and robust security awareness training, is paramount to safeguarding organizational assets, data, and reputation. Proactive and diligent implementation of these recommendations will substantially improve the organization’s resilience against cyber threats and build a more secure operational environment.
