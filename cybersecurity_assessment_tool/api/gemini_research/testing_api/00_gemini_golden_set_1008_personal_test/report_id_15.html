Executive Summary

This report details the findings of a cybersecurity posture assessment for Sovereign Trust. The assessment combined a network scan, a review of organizational security controls via a questionnaire, and an analysis of pre-existing risk data.

Key findings indicate several areas requiring immediate attention:

  • An open MySQL database port (3306) was identified on the target system (172.16.50.20), running an older version of MySQL (5.7.33). This is a significant security risk, as databases often contain sensitive information. This finding directly correlates with a pre-existing "Database Exposure" risk.

  • Critical gaps in security controls were noted, specifically the lack of Multi-Factor Authentication (MFA) for email access and the absence of mandatory annual security awareness training for all employees. These gaps increase the organization’s susceptibility to phishing, account compromise, and human-factor vulnerabilities.

Immediate recommendations include hardening database access, implementing MFA for email, and establishing a robust annual security awareness training program. Adhering to these recommendations will significantly enhance Sovereign Trust’s overall security posture.

Analysis of Technical Findings

The presence of an open MySQL port (3306) running version 5.7.33 is a significant concern. While MySQL 5.7.x is a mature branch, version 5.7.33 is not the latest release of the 5.7 series, and the entire 5.7 series reached End-of-Life (EOL) in October 2023. This means it no longer receives public security updates, making it vulnerable to known and newly discovered exploits. Exposing this service directly to the network without proper segmentation or access controls presents a high risk of database compromise.

Recommendations

The following recommendations are provided to address the identified risks and improve Sovereign Trust’s overall cybersecurity posture.

1. Database Security Hardening

  • Immediate Action (Firewall Rules): Implement strict firewall rules to restrict access to TCP port 3306 on 172.16.50.20 to only trusted internal IP addresses or specific hosts that absolutely require database connectivity.

  • Upgrade MySQL: Given that MySQL 5.7.x is End-of-Life (EOL), prioritize upgrading to a currently supported version (e.g., MySQL 8.0). This ensures ongoing security patches, bug fixes, and new features.

  • Network Segmentation: If direct exposure is unavoidable for specific applications, consider implementing a segmented network (e.g., a DMZ) for the database server and using a Virtual Private Network (VPN) for remote administrative access.

  • Regular Patching: Ensure all database servers are regularly patched and updated, even if an upgrade is not immediately feasible.

  • Secure Configuration: Review and harden MySQL configuration, including strong password policies, least privilege access, and disabling unnecessary features.

2. Implement Multi-Factor Authentication (MFA) for Email Access

  • Mandatory MFA: Enforce Multi-Factor Authentication (MFA) for all user accounts accessing email within the SovereignTrust.org domain. This significantly reduces the risk of unauthorized access even if passwords are compromised.

  • User Education: Provide clear instructions and support to employees on how to enroll and use MFA.

  • Policy Enforcement: Integrate MFA requirements into existing access policies and ensure compliance.

3. Establish Annual Security Awareness Training Program

  • Comprehensive Training: Develop and implement a mandatory annual security awareness training program for all employees, covering topics such as:

    • Phishing and social engineering recognition

    • Strong password practices and password managers

    • Data handling and classification

    • Incident reporting procedures

    • Acceptable Use Policy (AUP) refresh

  • Regular Refreshers: In addition to annual training, consider quarterly or bi-annual short refresher courses on specific high-risk topics.

  • Phishing Simulations: Conduct regular simulated phishing exercises to test employee awareness and identify areas for further training.
