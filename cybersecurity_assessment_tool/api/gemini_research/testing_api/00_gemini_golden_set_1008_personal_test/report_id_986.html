Executive Summary

This report details the cybersecurity posture of Iron Oak Furniture based on a recent network scan, organizational security questionnaire responses, and a review of pre-existing risk data. The assessment reveals several critical vulnerabilities and security gaps that require immediate attention. Key findings include the presence of an anonymous FTP server with an outdated version on a network host, critical Multi-Factor Authentication (MFA) implementation gaps for computer logins and sensitive data systems, and a significant number of workstations running an end-of-life operating system (Windows 7). Addressing these issues is paramount to enhancing the organization’s defensive capabilities against prevalent cyber threats and mitigating the risk of data breaches and system compromise.

Risk Assessment

This section consolidates all identified risks from the network scan, questionnaire analysis, and pre-existing vulnerability data. Risks are categorized by severity based on potential impact and likelihood.

Identified Risks
Risk Name Overview Severity
Anonymous FTP Access & Outdated vsftpd An FTP server (vsftpd 2.3.4, released 2011) is running on 10.0.0.15 with anonymous login allowed. This is an extremely old version with numerous known vulnerabilities. Anonymous FTP access poses a critical risk for unauthorized data access, potential data leakage, and system compromise. Critical
Lack of MFA for Computer Logins The organization does not require Multi-Factor Authentication (MFA) for employees to log into computers. This significantly increases the risk of unauthorized access, credential theft, and lateral movement within the network if passwords are compromised. High
Lack of MFA for Sensitive Data Systems The organization does not require Multi-Factor Authentication (MFA) for accessing sensitive data systems. This directly exposes critical data to a higher risk of breach, unauthorized viewing, or modification in the event of credential compromise. High
Outdated Windows Operating Systems Workstations are identified as running Windows 7, which is an end-of-life operating system. This means it no longer receives security updates, leaving systems highly vulnerable to modern exploits, malware, and compliance issues. High

Recommendations

Based on the identified risks, the following prioritized and actionable recommendations are provided to improve the cybersecurity posture of Iron Oak Furniture.

  1. Immediately address Anonymous FTP and Outdated vsftpd (Critical Risk):

    • Disable Anonymous Login: Configure the FTP server on 10.0.0.15 to disallow anonymous login immediately.

    • Evaluate Necessity: Determine if an FTP server is truly required for business operations. If secure file transfer is needed, migrate to a modern, encrypted protocol such as SFTP (SSH File Transfer Protocol) or FTPS (FTP Secure).

    • Upgrade/Replace: If FTP remains a necessity, upgrade the vsftpd service to a current, supported, and patched version, or replace it with a more secure and maintained FTP server solution.

  2. Implement Multi-Factor Authentication (MFA) for all computer logins (High Risk):

    • Solution Selection: Research and deploy an MFA solution that integrates seamlessly with the organization’s existing identity management infrastructure (e.g., Active Directory, Azure AD).

    • Mandatory Enforcement: Mandate and enforce MFA for all user accounts logging into company computers, workstations, and network devices.

  3. Implement Multi-Factor Authentication (MFA) for sensitive data systems (High Risk):

    • System Identification: Conduct an inventory of all systems and applications that store, process, or transmit sensitive data.

    • MFA Configuration: Configure and enforce MFA for access to these identified sensitive data systems, leveraging existing MFA solutions or deploying new ones as necessary.

  4. Upgrade Outdated Windows Operating Systems (High Risk):

    • Operating System Upgrade Plan: Initiate a comprehensive project to upgrade all workstations currently running Windows 7 to a currently supported operating system, such as Windows 10 or Windows 11.

    • Prioritization: Prioritize the upgrade of critical systems and those handling sensitive data for immediate action.

    • Hardware Refresh: Develop a hardware refresh plan if existing workstation hardware does not meet the minimum requirements for newer operating system versions.

  5. Establish Regular Security Audits and Vulnerability Management:

    • Vulnerability Scanning: Implement a regular schedule for both internal and external vulnerability scans to proactively identify and address security weaknesses.

    • Patch Management: Establish and maintain a robust patch management program to ensure all operating systems, applications, and network devices are kept up-to-date with the latest security patches.

    • Penetration Testing: Conduct periodic penetration testing by qualified third parties to validate the effectiveness of security controls and identify exploitable vulnerabilities.
