Recommendations

To mitigate the identified risks and improve the overall cybersecurity posture of Blackwood Industries, the following recommendations are provided:

1. Implement MFA for Sensitive Data Systems

  • Action: Immediately deploy and enforce Multi-Factor Authentication for all systems and applications that handle sensitive organizational data. Prioritize systems based on the criticality and sensitivity of the information they process.

  • Benefit: Significantly reduces the risk of credential compromise leading to unauthorized access, even if a primary password is stolen or guessed.

2. Develop and Enforce an Acceptable Use Policy (AUP)

  • Action: Create a clear, comprehensive Employee Acceptable Use Policy (AUP) that outlines employee responsibilities, acceptable use of company IT resources (e.g., internet, email, hardware, software), data handling procedures, and consequences for non-compliance. Distribute the AUP to all employees and obtain signed acknowledgements. Regularly review and update the policy.

  • Benefit: Establishes clear guidelines for employee conduct, reduces misuse of organizational resources, mitigates internal security risks, and provides a legal framework for disciplinary actions.

3. Establish Mandatory Annual Security Awareness Training

  • Action: Implement a mandatory annual security awareness training program for all employees, supplementing the existing new hire training. Content should be engaging and cover current threats (e.g., advanced phishing techniques, ransomware, social engineering tactics), secure browsing habits, safe data handling practices, and clear procedures for reporting suspicious activities.

  • Benefit: Reinforces security best practices, keeps employees informed about evolving cyber threats, fosters a security-conscious culture, and reduces the human element of risk.

4. Conduct Comprehensive Network Vulnerability Scans

  • Action: As the provided network scan data was unavailable, it is critical to perform both external and internal vulnerability scans and, ideally, penetration tests on a regular basis (e.g., quarterly or semi-annually). These assessments should cover all internet-facing assets and internal network infrastructure.

  • Benefit: Identifies misconfigurations, open ports, outdated software, unpatched systems, and other technical vulnerabilities that could be exploited by attackers, providing crucial insights into the organization’s attack surface.
