Executive Summary

This report presents a cybersecurity posture assessment for Verve & Vigor. The assessment involved a review of organizational security controls via questionnaire, analysis of network scan data, and correlation with existing identified risks. The overall security posture is evaluated as Weak, with several critical vulnerabilities and significant control gaps identified.

Key findings include an exposed anonymous FTP service, outdated software, a critical lack of Multi-Factor Authentication (MFA) for email access, absence of an Acceptable Use Policy, and insufficient employee security awareness training. These issues collectively expose the organization to a high risk of unauthorized access, data breaches, and compliance violations.

Immediate actions are recommended to mitigate the most critical risks, followed by a structured approach to strengthen overall security controls and employee awareness.

Recommendations

Based on the assessment, the following recommendations are provided to improve Verve & Vigorâ€™s cybersecurity posture:

  1. Disable Anonymous FTP Access: Immediately disable anonymous login on the FTP server running on 10.0.0.15 port 21. If FTP is required, ensure strong authentication and authorization controls are in place, or migrate to a more secure protocol like SFTP.

  2. Update or Remove Outdated FTP Server: The vsftpd 2.3.4 version is severely outdated. It is critical to either upgrade the FTP server to a current, supported version or remove it if not absolutely necessary. This mitigates known vulnerabilities associated with older software.

  3. Implement Multi-Factor Authentication (MFA) for Email: Enforce MFA for all email accounts without delay. This is a critical control to prevent unauthorized access even if credentials are stolen.

  4. Develop and Implement an Acceptable Use Policy (AUP): Create a comprehensive AUP that outlines employee responsibilities regarding the use of company IT resources. Ensure all employees read, understand, and acknowledge the policy.

  5. Mandate Annual Security Awareness Training: Establish and enforce a program for mandatory annual security awareness training for all employees. This helps to keep employees informed about evolving threats such as phishing, social engineering, and data handling best practices.

  6. Upgrade Outdated Operating Systems: Prioritize the migration of all workstations running Windows 7 to a currently supported operating system (e.g., Windows 10 or 11). Operating systems past their end-of-life receive no security updates, leaving them highly vulnerable.

  7. Regular Security Audits and Scans: Implement a schedule for regular internal and external vulnerability scans and penetration testing to proactively identify and address security weaknesses.
