1. Executive Summary

This report details the findings from a cybersecurity assessment conducted for Fable & Lore. The assessment involved reviewing organizational security controls via a questionnaire, analyzing network scan data, and integrating existing risk information.

Overall, Fable & Lore demonstrates a reasonable foundation in several key security areas, including MFA for computer logins and sensitive data, employee acceptable use policies, and security awareness training. However, a critical vulnerability was identified concerning the lack of Multi-Factor Authentication (MFA) for email access, which poses a significant risk. An open SSH port was also detected, requiring proper securing to prevent unauthorized access. Recommendations are provided to mitigate identified risks and enhance the organization’s security posture.

6. Recommendations

To mitigate the identified risks and enhance the overall cybersecurity posture, the following recommendations are provided:

6.1. High-Priority Recommendations

  • Implement MFA for Email Access: Immediately enable and enforce Multi-Factor Authentication for all email accounts. This is a critical control to prevent unauthorized access even if user credentials are compromised.

6.2. General Security Enhancements

  • Secure SSH Configuration: Review and harden the configuration for the open SSH service on 2001:db8::1.

    • Authentication: Prioritize and enforce strong, key-based authentication. Disable password-based authentication if feasible.

    • Rate Limiting: Implement tools like Fail2Ban to prevent brute-force attacks by temporarily blocking suspicious IP addresses.

    • Software Updates: Ensure the SSH server software is kept fully patched and up-to-date to protect against known vulnerabilities.

    • Access Restrictions: Consider restricting SSH access to only specific trusted IP addresses or networks, if possible, to reduce the attack surface.

    • Port Change (Optional): While not a security panacea, changing the default SSH port (22) can reduce automated scanning attempts.

  • Maintain Security Awareness Training: Continue the current practice of regular security awareness training for all employees, especially emphasizing phishing recognition and password hygiene.

  • Review Acceptable Use Policy (AUP): Periodically review and update the employee acceptable use policy to ensure it reflects current technologies and threats.

7. Conclusion

Fable & Lore has a solid foundation in several areas of cybersecurity best practices. Addressing the critical gap in MFA for email and thoroughly securing the open SSH port will significantly strengthen its defense against common cyber threats. Consistent adherence to security policies and ongoing vigilance are essential for maintaining a robust security posture.
