1. Executive Summary

This assessment provides a snapshot of Ember Glow Hospitality’s cybersecurity landscape. Our analysis reveals a mixed security posture. Strengths include the implementation of Multi-Factor Authentication (MFA) for email access and an established Acceptable Use Policy. However, critical vulnerabilities exist due to the lack of MFA for computer logins and sensitive data systems, as well as an absence of security awareness training for both new and existing employees. A network scan of the target 192.168.1.100 indicated no open ports, which is a positive technical finding, suggesting good network hygiene for that specific host from an external perspective. The primary risks identified are procedural and policy-based, requiring immediate attention to strengthen the human and internal system security layers.

5. Risk Assessment

Identified Risks
Risk Name Overview Severity
Lack of Multi-Factor Authentication for Computer Logins Without MFA for computer logins, an attacker who compromises user credentials (e.g., via phishing) can gain direct access to internal systems, increasing the risk of data breaches and unauthorized access. High

6. Recommendations

To mitigate the identified risks and strengthen the overall cybersecurity posture of Ember Glow Hospitality, the following recommendations are provided:

  • Implement Multi-Factor Authentication (MFA) for Computer Logins:

    • Deploy and enforce MFA for all employee computer logins, particularly for domain-joined machines and remote access solutions. This significantly reduces the risk of successful credential theft attacks.

  • Implement Multi-Factor Authentication (MFA) for Sensitive Data Systems:

    • Prioritize the implementation of MFA for all systems that store, process, or transmit sensitive organizational or customer data. This adds a critical layer of security to the most valuable assets.

  • Develop and Enforce Security Awareness Training for New Employees:

    • Establish a mandatory cybersecurity awareness training program for all new hires during their onboarding process. This should cover common threats, organizational policies, and best practices.

  • Implement Annual Security Awareness Training for All Employees:

    • Conduct mandatory, comprehensive cybersecurity awareness training for all employees at least once per year. This training should be updated regularly to reflect current threat landscapes and emerging attack vectors.

  • Maintain Strong Network Hygiene for External-Facing Systems:

    • Continue to monitor and ensure that public-facing systems and internal hosts that are scanned remain without unnecessary open ports, as observed in the technical scan results. Regularly review firewall rules and network configurations.
