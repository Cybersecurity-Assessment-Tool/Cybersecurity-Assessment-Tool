Executive Summary

This report details the cybersecurity posture assessment for Fable & Lore, based on a review of organizational data, a recent network scan, and existing risk information.

The assessment identified critical security gaps, primarily concerning Multi-Factor Authentication (MFA). MFA is currently not required for accessing email, logging into computers, or accessing sensitive data systems. Additionally, while new employees receive security awareness training, annual recurring training for all employees is not conducted. These deficiencies significantly increase the organization’s attack surface and risk profile.

A recent network scan targeting 192.168.0.5 identified no open ports, which is a positive finding indicating a minimal direct network exposure for the scanned target. However, this finding contradicts a previously identified internal risk pertaining to an "Unencrypted Web Server" on Port 80, necessitating further investigation and reconciliation.

Immediate recommendations focus on the implementation of MFA across all critical access points and establishing a comprehensive, recurring security awareness training program.

Recommendations

Based on the analysis, the following actionable recommendations are provided to enhance Fable & Lore’s cybersecurity posture:

Immediate Actions (High Priority)

  1. Implement Multi-Factor Authentication (MFA):

    • Deploy MFA for all email accounts (e.g., using a TOTP app, FIDO2 key, or push notifications).

    • Enforce MFA for all computer logins, especially for administrative accounts and systems.

    • Integrate MFA for access to all sensitive data systems and applications.

  2. Establish Annual Security Awareness Training:

    • Develop and implement a mandatory annual security awareness training program for all employees.

    • Include topics such as phishing, social engineering, password hygiene, data handling, and incident reporting.

    • Conduct periodic simulated phishing campaigns to test and reinforce training.

Mid-Term Actions (Medium Priority)

  1. Re-verify and Reconcile Port 80 Discrepancy:

    • Conduct a targeted re-scan of 192.168.0.5 to definitively confirm the state of Port 80 and other common web ports.

    • If Port 80 is indeed closed, update or archive the "Unencrypted Web Server" risk in the organization’s risk register.

    • If an unencrypted web server is found on a different IP or re-emerges, ensure it is either closed or secured with SSL/TLS (Port 443).

  2. Comprehensive Vulnerability Scanning:

    • Perform regular, authenticated internal and external vulnerability scans across the entire network to identify potential misconfigurations, outdated software, and other vulnerabilities.

Long-Term Strategic Actions

  1. Security Policy Review and Enhancement:

    • Periodically review and update all security policies, including the Acceptable Use Policy, to reflect current threats and best practices.

    • Develop policies specifically addressing MFA usage, incident response, and data classification.

  2. Security Information and Event Management (SIEM):

    • Consider implementing a SIEM solution to centralize log collection, enable real-time threat detection, and facilitate incident response.

By addressing these recommendations, Fable & Lore can significantly improve its overall cybersecurity posture, reduce its attack surface, and better protect its valuable assets from modern cyber threats.
