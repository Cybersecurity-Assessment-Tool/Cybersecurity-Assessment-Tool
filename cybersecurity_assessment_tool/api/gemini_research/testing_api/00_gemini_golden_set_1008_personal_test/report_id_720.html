Analysis of Security Control Gaps

Based on the questionnaire, the following significant gaps were identified:

  • Lack of MFA for Email Access: Not requiring MFA for email significantly increases the risk of account compromise through phishing or credential stuffing attacks. Email is often a primary vector for initial access and business email compromise (BEC).

  • Absence of an Employee Acceptable Use Policy (AUP): Without a formal AUP, employees may not be aware of their responsibilities regarding the use of company IT resources, data handling, and secure computing practices. This can lead to increased risks of data loss, malware infections, and compliance issues.

  • Insufficient Annual Security Awareness Training: While new employees receive training, the lack of ongoing annual training for all employees means that critical security information may not be reinforced or updated. This can result in a decline in security posture over time as threats evolve.

Recommendations

To mitigate the identified risks and improve the overall cybersecurity posture, the following recommendations are provided:

  • Implement Multi-Factor Authentication (MFA) for Email:

    • Make MFA mandatory for all employee email accounts immediately.

    • Evaluate and implement a robust MFA solution that supports various methods (e.g., authenticator apps, hardware tokens).

    • Provide clear instructions and support to employees during the rollout.

  • Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Draft a comprehensive AUP that clearly outlines employee responsibilities regarding IT asset usage, data classification and handling, internet browsing, software installation, and incident reporting.

    • Ensure all employees read, understand, and formally acknowledge the AUP upon hiring and periodically thereafter.

    • Integrate AUP review into annual security awareness training.

  • Establish Mandatory Annual Security Awareness Training for All Employees:

    • Develop and implement a recurring (at least annual) security awareness training program for all employees, not just new hires.

    • The training should cover current threat landscapes, including advanced phishing techniques, social engineering, ransomware, password best practices, data protection, and incident reporting procedures.

    • Use engaging and interactive training methods to improve retention.

  • Address Data Input Issues for Future Assessments:

    • Ensure all future data inputs for cybersecurity assessments (e.g., network scans, vulnerability lists) are provided in valid and complete formats (e.g., proper JSON structure). This will enable a more thorough and accurate assessment of the organization’s cybersecurity posture.

Conclusion

While Iron Oak Furniture demonstrates some foundational security controls, there are critical gaps that expose the organization to significant cybersecurity risks. The most pressing issues revolve around the lack of MFA for email, the absence of an acceptable use policy, and insufficient ongoing security awareness training. Addressing these areas, as outlined in the recommendations, will substantially enhance the organization’s defense against common attack vectors. A complete cybersecurity assessment, including technical scans and known vulnerability data, is highly recommended once valid input data can be provided.
