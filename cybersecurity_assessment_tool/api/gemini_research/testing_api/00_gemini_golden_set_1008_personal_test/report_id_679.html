Executive Summary

This report details the current cybersecurity posture of Hearth & Home, based on a review of organizational data, security questionnaire responses, and a targeted network scan. The assessment identifies several critical gaps, particularly concerning the internal application of Multi-Factor Authentication (MFA), the absence of a formal Acceptable Use Policy (AUP), and the lack of initial security awareness training for new employees. While email access is protected by MFA and annual training is conducted, these internal control deficiencies significantly elevate the organization’s risk profile. A single open SSH port (22) was also identified, which, while not inherently a vulnerability, warrants review and hardening. Recommendations are provided to address these findings and enhance the overall security posture.

Analysis of Technical Scan

The scan revealed one open port, 22, which is typically used for SSH (Secure Shell). While SSH is a legitimate service for secure remote access, its presence always warrants attention as it is a common target for attackers. Without further information on the specific service, product, and version running, a detailed vulnerability assessment cannot be performed based solely on the scan data. It is crucial to ensure that any SSH service is properly secured with strong authentication (e.g., key-based), rate-limiting, and kept up-to-date.

Recommendations

To address the identified risks and enhance the organization’s cybersecurity posture, the following recommendations are provided:

High Priority Recommendations (Critical/High Risks)

  • Implement Multi-Factor Authentication (MFA) for Computer Login:

    • Mandate MFA for all employee logins to company computers and endpoints.

    • Evaluate and deploy appropriate MFA solutions (e.g., hardware tokens, authenticator apps, biometrics) suitable for the organization’s infrastructure.

  • Implement Multi-Factor Authentication (MFA) for Sensitive Data Systems:

    • Extend MFA requirements to all systems storing or processing sensitive organizational data.

    • Prioritize the implementation for critical systems and data repositories.

  • Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Draft a comprehensive AUP outlining acceptable and unacceptable use of company IT assets, internet, and data.

    • Ensure all employees read, understand, and formally acknowledge the AUP, ideally on an annual basis or upon significant policy changes.

  • Establish Security Awareness Training for New Employees:

    • Integrate mandatory security awareness training as a core part of the onboarding process for all new hires.

    • Training should cover essential topics such as phishing, social engineering, strong password practices, secure data handling, and reporting suspicious activities.

Medium Priority Recommendations (Medium Risks)

  • Secure and Monitor SSH Services:

    • Immediately review the configuration of the open SSH service (port 22) on .

    • Ensure strong authentication methods are enforced, such as disabling password authentication in favor of key-based authentication.

    • Implement IP restrictions (firewall rules) to limit access to trusted source IPs only.

    • Implement rate limiting to mitigate brute-force attacks.

    • Consider changing the default SSH port (22) to a non-standard port, although this is a security-through-obscurity measure and not a primary control.

    • Ensure the SSH server software is running the latest patched version to mitigate known vulnerabilities.

    • Implement robust logging and monitoring for all SSH access attempts and suspicious activities.

Conclusion

While Hearth & Home has implemented some foundational security controls such as MFA for email and annual security training, significant vulnerabilities exist in key areas. The lack of MFA for internal systems and computers, along with the absence of essential policy (AUP) and initial employee training, creates substantial risk. Addressing these critical and high-severity risks through the recommended actions will significantly strengthen the organization’s overall cybersecurity posture and resilience against modern threats. Ongoing vigilance and regular security reviews are essential for maintaining a strong defense.
