6. Recommendations

Based on the current assessment, the following recommendations are provided to improve the cybersecurity posture of Swift Current Labs.

  1. Address Multi-Factor Authentication (MFA) Gaps:

    • Implement mandatory MFA for all email access. Email is a primary vector for phishing and compromise, and MFA significantly reduces this risk.

    • Implement mandatory MFA for all sensitive data systems. This adds a critical layer of security to protect confidential information.

  2. Enhance New Employee Security Awareness Training:

    • Institute mandatory security awareness training for all new employees during their onboarding process. This ensures foundational security knowledge from day one.

  3. Perform a Comprehensive Network Scan:

    • Conduct a successful and thorough external and internal network scan to identify open ports, running services, and potential vulnerabilities. Ensure the scan data is properly collected and accessible for future analysis.

  4. Establish a Vulnerability Management Program:

    • Implement a systematic process for identifying, assessing, and remediating vulnerabilities. Maintain a centralized repository of known risks and their mitigation status. Ensure accurate data is captured for future assessments.
