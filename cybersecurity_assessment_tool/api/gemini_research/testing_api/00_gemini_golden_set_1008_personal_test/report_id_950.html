Executive Summary

This report details the cybersecurity posture assessment for New Era, conducted based on organizational data, network scan results, and pre-existing risk information. The assessment identified several critical areas requiring immediate attention, particularly regarding foundational security controls like Multi-Factor Authentication (MFA) for email, employee security awareness training, and policy enforcement. Furthermore, the network scan revealed the presence of highly outdated and insecure services, posing a significant external threat. Addressing these gaps is crucial to bolstering the organization’s resilience against modern cyber threats.

Risk Assessment

Based on the analysis of organizational controls, technical scan results, and pre-existing vulnerabilities, the following risks have been identified.

Identified Risks and Severities
Severity Risk Name Overview
Critical Outdated FTP Service A highly vulnerable vsftpd 2.3.4 service with anonymous login enabled is exposed on 10.0.0.15. This service is known to contain critical vulnerabilities, including a backdoor, and anonymous access provides an easy entry point for attackers.
High Lack of MFA for Email No Multi-Factor Authentication (MFA) is required for email access, making accounts highly susceptible to phishing, brute-force, and credential stuffing attacks.
High Missing Acceptable Use Policy The absence of an Acceptable Use Policy (AUP) leaves employees without clear guidelines on acceptable use of company IT resources, increasing the risk of misuse, data breaches, and non-compliance.
High Inadequate Security Training A complete lack of security awareness training for new and existing employees significantly increases the organization’s susceptibility to social engineering attacks (e.g., phishing) and makes employees a weak link in the security chain.
High Outdated Windows Policy Computers are running Windows 7, which reached End-of-Life (EOL) in January 2020. This means no further security updates are provided by Microsoft, leaving systems vulnerable to new and evolving threats.

Recommendations

The following recommendations are provided to mitigate the identified risks and enhance the overall cybersecurity posture of New Era.

  1. Immediately Remediate Outdated FTP Service:

    • Disable or remove the FTP service on 10.0.0.15 if it is not essential.

    • If FTP functionality is required, migrate to a secure alternative (e.g., SFTP or FTPS) using a modern, regularly updated server software.

    • Never allow anonymous FTP access unless absolutely necessary and properly secured with strict access controls.

    • Perform a full vulnerability scan and penetration test on the host after remediation.

  2. Implement Multi-Factor Authentication (MFA) for Email:

    • Mandate MFA for all email accounts across the organization. This significantly reduces the risk of successful phishing and credential theft attacks.

    • Educate employees on the importance and use of MFA.

  3. Develop and Enforce an Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that outlines the acceptable use of company IT resources, including internet, email, software, and hardware.

    • Ensure all employees read, understand, and acknowledge the AUP, ideally during onboarding and annually thereafter.

    • Establish clear consequences for AUP violations.

  4. Establish a Robust Security Awareness Training Program:

    • Implement mandatory security awareness training for all new hires as part of their onboarding process.

    • Conduct annual security awareness training for all employees, covering topics like phishing, malware, password hygiene, and data handling.

    • Regularly test employees with simulated phishing campaigns.

  5. Upgrade Outdated Operating Systems:

    • Prioritize the upgrade of all Windows 7 workstations to a currently supported operating system (e.g., Windows 10 or 11).

    • Develop a phased migration plan, ensuring data backup and application compatibility.

    • For systems that cannot be immediately upgraded, isolate them on a segmented network and apply all possible security hardening measures.

Conclusion

The assessment reveals a cybersecurity posture with several critical vulnerabilities and significant gaps in fundamental security controls. While some MFA controls are in place, the lack of MFA for email, coupled with an outdated and insecure FTP service, and the absence of basic policies and training, expose New Era to substantial risk. Proactive and immediate implementation of the recommendations outlined in this report is essential to protect the organization’s assets and data from potential cyber threats.
