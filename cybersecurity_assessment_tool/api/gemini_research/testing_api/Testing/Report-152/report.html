Cybersecurity Assessment Report

Overview

The organization operates a network with at least one host running a web server on port 80. The organization has implemented Multi-Factor Authentication (MFA) for email and sensitive data systems and has an acceptable use policy. New employees receive security awareness training. However, critical gaps exist in the security posture, specifically the lack of MFA for computer logins and the absence of annual security awareness training for all employees.

Identified Risks

Lack of Multi-Factor Authentication for Computer Login - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for employees to log into their computers. This significantly increases the risk of unauthorized access if credentials are stolen or compromised, as a single factor (password) is sufficient for authentication. This was identified from the 'Org Data' indicating 'Do you require MFA to log into computers?':'No'.

Affected Elements:

  • Employee workstations
  • Internal network access

Insufficient Annual Security Awareness Training - Medium

Overview: The organization does not conduct security awareness training for all employees at least once per year. This leaves employees vulnerable to evolving social engineering attacks (phishing, pretexting) and reduces their ability to identify and report security incidents, increasing the overall human risk factor. This was identified from the 'Org Data' indicating 'Does your organization do security awareness training for all employees at least once per year?':'No'.

Affected Elements:

  • All employees
  • Organizational data
  • Network security

Recommendations

  • To mitigate the identified risks, it is strongly recommended that the organization implement Multi-Factor Authentication (MFA) for all computer logins. This will significantly enhance protection against credential theft and unauthorized access. Additionally, a comprehensive annual security awareness training program should be established for all employees to ensure they are continuously updated on current threats and best practices, thereby strengthening the human firewall. Regular vulnerability assessments of the web server on port 80 should also be conducted to identify and address potential software vulnerabilities.

Conclusion

The organization demonstrates a foundational understanding of security with MFA for critical systems and initial employee training. However, the absence of MFA for computer logins and annual security awareness training represents significant vulnerabilities that could be exploited by attackers. Addressing these areas is crucial for improving the overall security readiness and resilience against common cyber threats.
