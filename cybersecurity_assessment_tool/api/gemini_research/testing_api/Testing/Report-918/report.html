Cybersecurity Assessment Report

Overview

The organization demonstrates a strong commitment to multi-factor authentication (MFA), requiring it for email, computer logins, and access to sensitive data systems. An employee acceptable use policy is also in place, and annual security awareness training is conducted for all employees. However, a gap exists in security awareness training for new employees. Network scanning revealed minimal open ports, with port 80 identified as closed. Despite this, a previously identified vulnerability regarding an unencrypted web server on port 80 was noted, suggesting a potential past issue or a discrepancy in scan results that warrants further investigation.

Identified Risks

Unencrypted Web Server - Medium

Overview: Port 80 is open, indicating that an unencrypted HTTP service might be running. This poses a risk of sensitive data being intercepted in transit, compromising confidentiality. The impact could lead to unauthorized disclosure of information. This vulnerability was identified through a security assessment, noting the presence of an unencrypted web service.

Affected Elements:

  • Port 80

Recommendations

  • To mitigate identified risks and enhance the overall security posture, it is recommended to investigate the discrepancy regarding port 80's status and, if an unencrypted web server is indeed active, ensure it is either closed or secured with SSL/TLS encryption. Implementing SSL/TLS for all web services is a crucial long-term fix to protect data in transit. Furthermore, it is strongly advised to establish and implement mandatory security awareness training for all new employees during their onboarding process to ensure they are equipped with essential cybersecurity knowledge from day one. Regularly review and update security policies and training materials to address evolving threats.

Conclusion

The organization has a solid foundation with widespread MFA implementation and an acceptable use policy. Annual security awareness training is a positive step, but the absence of training for new employees represents a significant vulnerability. The identified risk of an unencrypted web server, despite conflicting scan data, highlights the need for thorough verification and immediate remediation if active. Addressing these areas will significantly improve the organization's readiness against common cyber threats and strengthen its overall security posture.
