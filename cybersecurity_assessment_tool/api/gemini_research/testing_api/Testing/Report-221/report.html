Cybersecurity Assessment Report

Overview

The organization has implemented Multi-Factor Authentication (MFA) for email access and computer logins, which are strong security practices. However, MFA is not required for accessing sensitive data systems, posing a significant risk. An acceptable use policy is in place, and annual security awareness training is conducted for all employees. Notably, new employees do not receive security awareness training, which could lead to early security missteps. A network scan identified an open Remote Desktop Protocol (RDP) port (3389), indicating potential external exposure.

Identified Risks

RDP Exposure - Critical

Overview: Remote Desktop Protocol (RDP) is exposed externally, allowing potential attackers to attempt brute-force attacks or exploit RDP vulnerabilities to gain unauthorized access to internal systems. This was identified via a network scan showing port 3389 open.

Affected Elements:

  • External IP address with port 3389 open

Lack of MFA for Sensitive Data Systems - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for accessing sensitive data systems. This significantly increases the risk of unauthorized access to critical information if credentials are compromised, as a single factor (password) is sufficient for authentication. This was identified through the organizational data provided.

Affected Elements:

  • Sensitive data systems

No Security Awareness Training for New Employees - Medium

Overview: New employees do not receive dedicated security awareness training upon onboarding. This leaves them vulnerable to common social engineering attacks, phishing, and other security threats, potentially introducing vulnerabilities into the organization's security posture from day one. This was identified through the organizational data provided.

Affected Elements:

  • New employees
  • Organizational security posture

Recommendations

  • To mitigate the identified risks, it is strongly recommended to immediately close the externally exposed RDP port (3389) and implement a Virtual Private Network (VPN) for secure remote access, ensuring all RDP connections are tunneled through the VPN. Furthermore, Multi-Factor Authentication (MFA) must be mandated for all sensitive data systems to prevent unauthorized access even if primary credentials are compromised. The organization should also establish a comprehensive security awareness training program for all new employees as part of their onboarding process to instill good security practices from the outset. Regular security audits and policy reviews are also advised to maintain a robust security posture.

Conclusion

The organization demonstrates a foundational commitment to security through MFA for email and computer logins, and annual security training. However, critical gaps exist, particularly the external exposure of RDP, the absence of MFA for sensitive data systems, and the lack of security awareness training for new hires. Addressing these vulnerabilities promptly is crucial to significantly enhance the organization's overall cybersecurity readiness and protect against potential breaches.
