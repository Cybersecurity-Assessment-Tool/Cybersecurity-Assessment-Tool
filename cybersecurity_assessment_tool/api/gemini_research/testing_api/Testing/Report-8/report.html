Cybersecurity Assessment Report

Overview

An Nmap scan identified a host with port 80 open. The organization demonstrates strong security practices by requiring MFA for computer logins and sensitive data systems, and conducting regular security awareness training. However, MFA is not enforced for email access, and an employee acceptable use policy is currently absent.

Identified Risks

No specific vulnerabilities were identified.

Recommendations

  • To further enhance the organization's security posture, it is recommended to implement Multi-Factor Authentication (MFA) for all email access. Additionally, developing and enforcing a comprehensive employee acceptable use policy is crucial for establishing clear guidelines and expectations regarding the use of organizational assets and information.

Conclusion

Based on the assessment, the network infrastructure is deemed secure. While the organization has robust security measures in place for computer logins, sensitive data, and employee training, the absence of MFA for email and an acceptable use policy represents areas for improvement in overall organizational readiness. Addressing these points will further strengthen the security posture.
