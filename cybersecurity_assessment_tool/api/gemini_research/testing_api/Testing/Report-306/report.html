Cybersecurity Assessment Report

Overview

The organization's network scan indicates an active host, with port 80 reported as closed. However, a specific vulnerability assessment identified an unencrypted web server, suggesting a potential misconfiguration or a discrepancy in scan results, with port 80 being a concern. The organization has implemented Multi-Factor Authentication (MFA) for computer logins and sensitive data systems, which are positive security measures. However, there is a critical lack of MFA for email access. Furthermore, the organization does not conduct security awareness training for new employees or annually for all employees, which represents a significant human-factor risk. An acceptable use policy is in place.

Identified Risks

Unencrypted Web Server - Medium

Overview: An unencrypted web server was identified, with port 80 potentially open, exposing web traffic to interception. This allows attackers to capture sensitive information transmitted over the network. The presence of an unencrypted service indicates a lack of secure communication protocols.

Affected Elements:

  • Port 80

No MFA for Email Access - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for accessing email. This significantly increases the risk of account compromise through phishing, brute-force attacks, or credential stuffing, as a single compromised password can grant full access to an employee's email.

Affected Elements:

  • Email System
  • Employee Accounts

Inadequate Security Awareness Training - High

Overview: The organization lacks security awareness training for new employees and does not conduct annual training for all employees. This leaves the workforce vulnerable to social engineering attacks, phishing, malware, and other human-factor exploits, as employees may not be equipped to identify and respond to threats.

Affected Elements:

  • All Employees
  • Organizational Data
  • Network Infrastructure

Recommendations

  • To mitigate the identified risks, it is strongly recommended to immediately address the unencrypted web server by ensuring port 80 is closed if not actively used, and implementing SSL/TLS encryption for all web services to protect data in transit. Crucially, Multi-Factor Authentication (MFA) must be implemented for all email access to prevent account compromise. Furthermore, a comprehensive security awareness training program should be established, including mandatory training for all new employees and annual refresher training for the entire workforce, to educate staff on identifying and responding to common cyber threats such as phishing and social engineering. Regular network scans and vulnerability assessments should also be conducted to ensure ongoing security posture.

Conclusion

In conclusion, while the organization has taken positive steps by implementing MFA for computer logins and sensitive data systems, significant vulnerabilities exist. The lack of MFA for email and the complete absence of security awareness training for employees represent critical gaps that expose the organization to a high risk of credential compromise and human-factor attacks. The identified unencrypted web server also poses a risk of data interception. Addressing these areas is paramount to enhancing the organization's overall cybersecurity readiness and reducing its attack surface.
