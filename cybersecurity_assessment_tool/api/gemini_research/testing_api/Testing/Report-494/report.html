Cybersecurity Assessment Report

Overview

The organization's network includes an FTP server running vsftpd 2.3.4. The current security posture reveals significant gaps in identity and access management, particularly concerning Multi-Factor Authentication (MFA) across email, computer logins, and sensitive data systems. Furthermore, the organization lacks an acceptable use policy and does not conduct annual security awareness training for all employees. Several workstations are running outdated operating systems (Windows 7).

Identified Risks

Anonymous FTP Access - High

Overview: The FTP server (vsftpd 2.3.4) on port 21 allows anonymous login. This can lead to unauthorized access to files, data leakage, or even arbitrary file uploads if not properly configured, potentially compromising data confidentiality and integrity.

Affected Elements:

  • FTP Server (vsftpd 2.3.4)
  • Port 21

Lack of MFA for Email - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for email access. This significantly increases the risk of account compromise through phishing or brute-force attacks, as a single compromised password could grant an attacker full access to an employee's email.

Affected Elements:

  • Email system
  • Employee accounts

Lack of MFA for Computer Login - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for computer logins. This makes workstations vulnerable to unauthorized access if passwords are stolen or guessed, potentially leading to lateral movement within the network and access to sensitive data.

Affected Elements:

  • Workstations
  • Employee accounts

Lack of MFA for Sensitive Data Systems - Critical

Overview: The organization does not require Multi-Factor Authentication (MFA) for accessing sensitive data systems. This is a critical vulnerability as it means that a compromised password could directly lead to unauthorized access to highly sensitive information, resulting in severe data breaches.

Affected Elements:

  • Sensitive data systems
  • Employee accounts

No Acceptable Use Policy - Medium

Overview: The organization lacks an employee acceptable use policy. This absence means there are no clear guidelines for employees on the proper and secure use of company IT resources, increasing the risk of misuse, accidental data breaches, and non-compliance with security best practices.

Affected Elements:

  • Employees
  • IT resources

Insufficient Annual Security Training - Medium

Overview: While new employees receive security awareness training, the organization does not conduct annual security awareness training for all employees. This leaves the workforce susceptible to evolving threats like advanced phishing techniques, social engineering, and malware, as their knowledge may become outdated.

Affected Elements:

  • Employees

Outdated Windows Policy - High

Overview: Computers are running Windows 7. This operating system is End-of-Life and no longer receives security updates, making workstations highly vulnerable to known exploits and malware.

Affected Elements:

  • Workstations

Recommendations

  • Immediately disable anonymous FTP access on the vsftpd server and ensure proper authentication and authorization controls are in place. Implement Multi-Factor Authentication (MFA) for all email accounts, computer logins, and access to sensitive data systems to significantly reduce the risk of unauthorized access due to compromised credentials. Develop and implement a comprehensive employee acceptable use policy to provide clear guidelines for the secure and appropriate use of IT resources. Establish a mandatory annual security awareness training program for all employees to keep them informed about current threats and best security practices. Upgrade all workstations running Windows 7 to a currently supported operating system like Windows 10 or 11 to address known vulnerabilities and ensure ongoing security updates. Consider a hardware refresh if existing machines cannot support newer OS versions.

Conclusion

The organization currently faces several critical and high-severity vulnerabilities, primarily stemming from a lack of robust access controls (MFA) and outdated systems/policies. The presence of an anonymous FTP server and workstations running Windows 7, combined with insufficient security policies and training, creates a significant attack surface. Addressing these issues promptly through the recommended actions is crucial to enhance the organization's overall cybersecurity readiness and reduce the risk of data breaches and system compromises.
