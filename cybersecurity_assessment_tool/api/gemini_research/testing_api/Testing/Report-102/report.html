Cybersecurity Assessment Report

Overview

The organization has implemented Multi-Factor Authentication (MFA) for email and computer logins, and maintains an acceptable use policy along with security awareness training for new employees. However, a significant gap exists as MFA is not required for sensitive data systems, and annual security awareness training is not provided to all employees. While network scans indicated port 80 as closed, a reported vulnerability highlights a potential unencrypted web server issue related to this port.

Identified Risks

Unencrypted Web Server - Medium

Overview: Port 80 is open, indicating a potential unencrypted web server. This allows data to be transmitted in plain text, making it vulnerable to interception and eavesdropping. The impact could be the exposure of sensitive information if transmitted over this port. This was identified as a reported vulnerability.

Affected Elements:

  • Port 80

Lack of MFA for Sensitive Data Systems - High

Overview: The organization does not require Multi-Factor Authentication (MFA) to access sensitive data systems. This significantly increases the risk of unauthorized access to critical information if credentials are compromised, as a single factor (password) is sufficient for authentication. This vulnerability was identified through organizational policy review.

Affected Elements:

  • Sensitive data systems
  • User authentication processes

Insufficient Annual Security Awareness Training - Medium

Overview: The organization does not conduct security awareness training for all employees at least once per year. This leaves employees vulnerable to evolving social engineering attacks (e.g., phishing, malware) and reduces their ability to identify and report security threats, increasing the overall human risk factor. This vulnerability was identified through organizational policy review.

Affected Elements:

  • All employees
  • Organizational security posture

Recommendations

  • To mitigate the identified risks, it is recommended to close port 80 if it is not essential, or implement SSL/TLS encryption for all web traffic if a web server is intended to be active on this port. Furthermore, implement mandatory Multi-Factor Authentication (MFA) for all access to sensitive data systems to add an essential layer of security against credential theft. Finally, establish and enforce a policy for mandatory annual security awareness training for all employees, covering current threats like phishing, social engineering, and data handling best practices.

Conclusion

In conclusion, while the organization has established some foundational security controls such as MFA for general access and initial employee training, critical vulnerabilities remain. The absence of MFA for sensitive data systems and the lack of consistent annual security awareness training for all employees represent significant risks that could lead to unauthorized access or successful social engineering attacks. Furthermore, the reported unencrypted web server, if active, poses an additional exposure. Addressing these identified risks is paramount to strengthening the organization's cybersecurity posture and ensuring the protection of its sensitive information and systems.
