Cybersecurity Assessment Report

Overview

The organization's network currently exposes an Nginx web server (version 1.18.0) on port 443 (HTTPS). While an acceptable use policy is in place and new employees receive security awareness training, there are significant gaps in multi-factor authentication (MFA) implementation and ongoing security training. MFA is not required for email access or computer logins, although it is mandated for sensitive data systems. Furthermore, annual security awareness training for all employees is not conducted.

Identified Risks

Lack of MFA for Email Access - High

Overview: Email accounts are a primary target for phishing and credential stuffing attacks. Without multi-factor authentication (MFA), a single compromised password can grant an attacker unauthorized access to sensitive communications, data, and potentially facilitate lateral movement within the organization. This vulnerability was identified by the organization's stated policy of not requiring MFA for email access.

Affected Elements:

  • Email system
  • User accounts

Lack of MFA for Computer Login - High

Overview: Without multi-factor authentication (MFA) for computer logins, a single compromised password can grant an attacker access to an employee's workstation. This could lead to data exfiltration, malware deployment, or further network compromise, as the workstation often serves as an entry point to internal resources. This vulnerability was identified by the organization's stated policy of not requiring MFA for computer logins.

Affected Elements:

  • Employee workstations
  • Network access points

Outdated Nginx Web Server - High

Overview: The Nginx web server running version 1.18.0 is outdated and no longer receives official security updates or support. This means it likely contains known vulnerabilities that could be exploited by attackers to gain unauthorized access, perform denial-of-service attacks, or execute arbitrary code, posing a significant risk to the web service and potentially the underlying infrastructure. This was identified during the network scan.

Affected Elements:

  • Web server (nginx 1.18.0)
  • HTTPS service on port 443

Lack of Annual Security Awareness Training - Medium

Overview: While new employees receive initial training, the absence of annual security awareness training means employees may not be updated on the latest threats, evolving best practices, or changes in organizational policies. This increases the overall risk of successful phishing attacks, social engineering, and other human-factor vulnerabilities that can lead to security incidents. This vulnerability was identified by the organization's stated policy.

Affected Elements:

  • All employees
  • Organizational data
  • Network

Recommendations

  • To significantly enhance the organization's security posture, it is strongly recommended to implement multi-factor authentication (MFA) for all email access and computer logins across the entire organization. This critical step will drastically reduce the risk of unauthorized access due to compromised credentials. Furthermore, the Nginx web server (version 1.18.0) should be immediately updated to a currently supported and patched version to mitigate known vulnerabilities. Regular patching and version upgrades for all software are essential. Finally, a comprehensive annual security awareness training program for all employees should be established and enforced to ensure staff are continuously educated on the latest cyber threats and best security practices, thereby reducing human-factor risks.

Conclusion

The organization demonstrates some foundational security practices, such as an acceptable use policy and initial security awareness training for new hires, and correctly applies MFA to sensitive data systems. However, critical vulnerabilities exist due to the lack of MFA for email and computer logins, the use of an outdated and unsupported web server, and the absence of ongoing, annual security awareness training for all staff. These gaps collectively expose the organization to a high risk of unauthorized access, data breaches, and successful cyberattacks, indicating a moderate level of readiness that requires immediate attention and remediation.
