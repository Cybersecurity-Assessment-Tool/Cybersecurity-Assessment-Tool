Cybersecurity Assessment Report

Overview

The organization demonstrates a mixed security posture. While Multi-Factor Authentication (MFA) is implemented for email and sensitive data systems, a critical vulnerability exists with Remote Desktop Protocol (RDP) exposed directly to the internet. Furthermore, there is a significant gap in internal access controls, as MFA is not required for computer logins. Foundational security practices are also lacking, with no employee acceptable use policy in place and an absence of annual security awareness training for all employees, indicating a potential for human-factor vulnerabilities and inconsistent security behavior across the organization.

Identified Risks

RDP Exposure to Internet - Critical

Overview: Remote Desktop Protocol (RDP) is exposed directly to the internet via an open port 3389. This significantly increases the attack surface, making the organization vulnerable to brute-force attacks, credential stuffing, and exploitation of RDP vulnerabilities, potentially leading to unauthorized access and system compromise. This was identified through network scanning data showing port 3389 as open.

Affected Elements:

  • Port 3389
  • IP address of the host with RDP exposed

Insufficient Authentication for Computer Access - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for computer logins. This leaves employee workstations and potentially internal network access vulnerable to credential theft, phishing attacks, and brute-force attempts, as a single compromised password could grant an attacker full access to internal systems. This was identified from the organizational data indicating 'No' for MFA on computer logins.

Affected Elements:

  • All employee computers
  • Internal network access points

Lack of Employee Acceptable Use Policy - Medium

Overview: The organization lacks an employee acceptable use policy. Without clear guidelines on appropriate use of company IT resources, employees may inadvertently engage in risky behaviors, such as visiting malicious websites, downloading unauthorized software, or mishandling sensitive data, increasing the likelihood of security incidents and making it difficult to enforce security standards. This was identified from the organizational data indicating 'No' for an acceptable use policy.

Affected Elements:

  • All employees
  • Company IT resources

Inadequate Annual Security Awareness Training - Medium

Overview: The organization does not conduct security awareness training for all employees at least once per year. This leads to a degradation of security knowledge over time, making employees more susceptible to social engineering attacks (e.g., phishing, pretexting) and less aware of evolving cyber threats, thereby increasing human-factor risks and weakening the overall security posture. This was identified from the organizational data indicating 'No' for annual security awareness training.

Affected Elements:

  • All employees
  • Organizational security posture

Recommendations

  • To mitigate the identified risks, it is strongly recommended to immediately close the exposed RDP port (3389) to the internet. For necessary remote access, implement a Virtual Private Network (VPN) with strong authentication, ideally including MFA, to secure the connection. Furthermore, implement Multi-Factor Authentication (MFA) for all computer logins to significantly enhance protection against credential theft. Develop and enforce a comprehensive employee acceptable use policy to establish clear guidelines for IT resource usage. Finally, establish a mandatory annual security awareness training program for all employees to ensure continuous education on current cyber threats and best security practices.

Conclusion

The organization currently faces critical and high-severity vulnerabilities primarily due to an exposed RDP service and a lack of MFA for computer logins, which significantly broadens the attack surface. While MFA is present for email and sensitive data, the absence of an acceptable use policy and annual security awareness training indicates a need for improvement in foundational security policies and employee education. The organization's readiness is compromised by these gaps, requiring immediate attention to technical controls and a strategic investment in human-factor security measures to achieve a more robust and resilient cybersecurity posture.
