Cybersecurity Assessment Report

Overview

The organization's network scan revealed an open port 80, indicating potential unencrypted web traffic. Organizational policies show a significant lack of multi-factor authentication for computer logins, absence of an employee acceptable use policy, and no regular security awareness training for employees.

Identified Risks

Unencrypted Web Traffic - Medium

Overview: An open port 80 was detected, typically used for unencrypted HTTP communication. This allows sensitive data transmitted over this port to be intercepted and read by attackers, leading to data breaches or unauthorized access. Identified via Nmap scan.

Affected Elements:

  • Port 80
  • Web Server

Weak Authentication for Computers - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for logging into computers. This significantly increases the risk of unauthorized access if credentials are stolen or guessed, as a single factor (password) is insufficient to protect against modern threats. Identified from organizational data.

Affected Elements:

  • All employee computers
  • User accounts

Absence of Employee Acceptable Use Policy - Medium

Overview: The organization lacks an employee acceptable use policy. This absence can lead to employees engaging in risky online behaviors, misusing company resources, or inadvertently introducing malware, as there are no clear guidelines for appropriate conduct. Identified from organizational data.

Affected Elements:

  • All employees
  • Organizational network resources

Low Security Awareness - Medium

Overview: The organization does not conduct security awareness training for new employees or annually for all employees. This leaves employees vulnerable to social engineering attacks (phishing, pretexting), malware, and other common threats, as they may not recognize or report suspicious activities. Identified from organizational data.

Affected Elements:

  • All employees
  • Organizational data and systems

Recommendations

  • It is strongly recommended to implement HTTPS for all web services running on port 80 to encrypt data in transit. Immediately enforce Multi-Factor Authentication (MFA) for all computer logins to enhance authentication security. Develop and implement a comprehensive employee acceptable use policy to guide appropriate conduct and reduce insider risks. Furthermore, establish mandatory security awareness training for all new hires and conduct annual refresher training for all employees to improve overall human firewall effectiveness.

Conclusion

The organization currently faces significant cybersecurity risks stemming from both technical configurations and policy gaps. Key vulnerabilities include unencrypted web traffic, weak authentication for computer logins due to the lack of MFA, and a general lack of security awareness and policy enforcement among employees. Addressing these areas is critical to improving the organization's overall security posture and readiness against common cyber threats.
