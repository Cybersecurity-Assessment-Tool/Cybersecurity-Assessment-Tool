Cybersecurity Assessment Report

Overview

The organization's network currently operates an FTP server (vsftpd 2.3.4) that permits anonymous login, posing a significant data exposure risk. Internally, there is a critical lack of Multi-Factor Authentication (MFA) across key systems including email, computer logins, and sensitive data access, leaving the organization highly vulnerable to credential-based attacks. Furthermore, workstations are running Windows 7, an end-of-life operating system, which is no longer receiving security updates and is susceptible to known exploits. The absence of an employee acceptable use policy also introduces potential for misuse of IT resources. Positively, the organization does conduct security awareness training for both new and existing employees annually, which is a good foundation for improving human-factor security.

Identified Risks

Anonymous FTP Access - High

Overview: The FTP server (vsftpd version 2.3.4) is configured to allow anonymous login. This configuration can lead to unauthorized access to files, potential data leakage, or even server compromise if write permissions are misconfigured. This vulnerability was identified through an Nmap scan that successfully performed an anonymous FTP login.

Affected Elements:

  • FTP server (vsftpd 2.3.4)
  • Port 21

Lack of Multi-Factor Authentication (MFA) - Critical

Overview: The organization does not require Multi-Factor Authentication (MFA) for accessing email, logging into computers, or accessing sensitive data systems. This significantly increases the risk of unauthorized access in the event of compromised or stolen credentials, as a single factor (password) is the only barrier to entry for critical systems.

Affected Elements:

  • Email systems
  • User login systems
  • Sensitive data systems

Absence of Employee Acceptable Use Policy - Medium

Overview: The organization currently lacks a formal employee acceptable use policy. Without clear guidelines on the appropriate use of company IT resources, employees may inadvertently or intentionally engage in activities that pose security risks, lead to data breaches, or create legal liabilities for the organization.

Affected Elements:

  • All employees
  • IT resources

Outdated Windows Policy - High

Overview: Several workstations within the organization are running Windows 7, an operating system that has reached its end-of-life. This means these systems no longer receive critical security updates, making them highly vulnerable to known exploits and newly discovered threats, significantly increasing the risk of malware infection and system compromise.

Affected Elements:

  • Workstations

Recommendations

  • To mitigate the identified risks, it is strongly recommended to immediately disable anonymous access on the FTP server and implement strong authentication mechanisms, or ideally, transition to a more secure protocol like SFTP or FTPS. A critical priority should be the implementation of Multi-Factor Authentication (MFA) for all email access, computer logins, and sensitive data systems to significantly enhance protection against credential theft. The organization must also develop and enforce a comprehensive employee acceptable use policy to guide appropriate behavior regarding IT resource usage. For the outdated Windows 7 workstations, an urgent upgrade to a currently supported operating system like Windows 10 or 11 is essential to ensure systems receive vital security updates and patches. While security awareness training is already in place, consider incorporating specific modules addressing these new policies and the importance of MFA.

Conclusion

The organization currently faces several critical and high-severity vulnerabilities, primarily stemming from the lack of Multi-Factor Authentication across key systems, the presence of an anonymously accessible FTP server, and the continued use of end-of-life Windows 7 workstations. While security awareness training is a positive aspect of the current posture, the absence of fundamental security controls and policies leaves the organization highly susceptible to unauthorized access, data breaches, and system compromise. Addressing these issues with urgency is paramount to improving the overall cybersecurity readiness and reducing the attack surface.
