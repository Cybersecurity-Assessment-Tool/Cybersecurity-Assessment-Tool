Cybersecurity Assessment Report

Overview

The organization has implemented multi-factor authentication (MFA) for critical access points including email, computer logins, and sensitive data systems, indicating a strong foundational security measure. An acceptable use policy is in place, and annual security awareness training is conducted for all employees. However, a significant vulnerability has been identified: an open Remote Desktop Protocol (RDP) port (3389) on the network, which exposes internal systems to potential external threats. Additionally, there is a gap in security awareness training for new employees.

Identified Risks

RDP Exposure - Critical

Overview: Remote Desktop Protocol (RDP) is exposed to the internet via an open port (3389). This allows external attackers to attempt to gain unauthorized access to internal systems, potentially leading to data breaches, system compromise, or ransomware attacks. This was identified through network scanning which revealed port 3389 as open and accessible.

Affected Elements:

  • IP address with open port 3389

Recommendations

  • Immediately close port 3389 to prevent external RDP access. For legitimate remote access needs, implement a Virtual Private Network (VPN) solution to secure connections, ensuring all RDP traffic is tunneled through the VPN. Furthermore, it is recommended to implement comprehensive security awareness training for all new employees to ensure they are aware of cybersecurity best practices and organizational policies from day one, complementing the existing annual training for all staff.

Conclusion

The organization demonstrates a commendable commitment to security through its widespread use of MFA and consistent annual security awareness training. However, the critical vulnerability of an exposed RDP port presents a significant and immediate risk that requires urgent remediation. Addressing this, along with implementing security awareness training for new hires, will further strengthen the organization's overall cybersecurity posture and readiness against external threats, moving towards a more robust and resilient security environment.
