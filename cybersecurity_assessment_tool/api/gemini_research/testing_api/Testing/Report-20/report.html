Cybersecurity Assessment Report

Overview

The organization operates a network with at least one active host exposing an open HTTP port (80). While an employee acceptable use policy is in place and annual security awareness training is conducted for all employees, critical gaps exist in the overall security posture. Specifically, the absence of Multi-Factor Authentication (MFA) for email and computer logins, and the lack of security awareness training for new employees, represent significant vulnerabilities that could be exploited.

Identified Risks

Lack of Multi-Factor Authentication for Email - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for accessing email. This significantly increases the risk of unauthorized access to employee email accounts through phishing, brute-force attacks, or credential stuffing, potentially leading to data breaches, business email compromise (BEC), and further network penetration. This was identified from the 'Org Data' indicating 'Do you require MFA to access email?': 'No'.

Affected Elements:

  • Email system
  • Employee accounts

Lack of Multi-Factor Authentication for Computer Login - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for logging into computers. This leaves employee workstations vulnerable to unauthorized access if credentials are compromised, potentially allowing attackers to gain a foothold within the internal network, access sensitive data, and deploy malware. This was identified from the 'Org Data' indicating 'Do you require MFA to log into computers?': 'No'.

Affected Elements:

  • Employee workstations
  • Internal network access

Inadequate Onboarding Security Training - Medium

Overview: The organization does not conduct security awareness training for new employees upon joining. This leaves new hires susceptible to social engineering attacks, phishing, and other common threats due to a lack of immediate awareness regarding organizational security policies and best practices. This was identified from the 'Org Data' indicating 'Does your organization do security awareness training for new employees?': 'No'.

Affected Elements:

  • New employees
  • Organizational security posture

Open HTTP Port (80) - Low

Overview: A network scan revealed an open port 80 (HTTP) on at least one host. While not inherently a vulnerability, running services over unencrypted HTTP can expose sensitive information to eavesdropping and man-in-the-middle attacks. It also indicates a potential web server that might have underlying vulnerabilities if not properly secured and patched. This was identified via Nmap scan data.

Affected Elements:

  • Host with port 80 open
  • Web server
  • Network traffic

Recommendations

  • To significantly enhance the organization's security posture, it is strongly recommended to implement Multi-Factor Authentication (MFA) for all email access and computer logins. This critical step will drastically reduce the risk of unauthorized access due to compromised credentials. Furthermore, a mandatory security awareness training program for all new employees should be established as part of their onboarding process to ensure immediate awareness of security best practices. Finally, any services currently running on unencrypted HTTP (port 80) should be migrated to HTTPS (port 443) to protect data in transit, and the underlying web servers must be regularly patched and securely configured.

Conclusion

In conclusion, while the organization demonstrates some foundational security practices such as an acceptable use policy and annual security awareness training, critical vulnerabilities exist that significantly elevate its risk profile. The absence of Multi-Factor Authentication for email and computer logins, coupled with a lack of dedicated security awareness training for new employees, creates substantial entry points for attackers. Additionally, the presence of an open HTTP port warrants attention for secure configuration and encryption. Addressing these key areas is paramount to improving the overall cybersecurity readiness and resilience of the organization.
