Cybersecurity Assessment Report

Overview

The organization's network scan revealed an open HTTP port (80), indicating a potential web service or unencrypted communication channel. Internally, the organization has an acceptable use policy and requires MFA for computer logins. However, critical gaps exist in security posture, specifically the lack of Multi-Factor Authentication (MFA) for email and sensitive data systems. Furthermore, there is no security awareness training for new employees, nor is there annual training for all employees, which significantly increases human-factor risks.

Identified Risks

Lack of MFA for Email Access - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for email access. This significantly increases the risk of unauthorized access to employee email accounts if credentials are compromised through phishing, brute-force attacks, or credential stuffing. Compromised email accounts can lead to further attacks, data breaches, and business email compromise (BEC) scams.

Affected Elements:

  • Email system
  • Employee accounts

Lack of MFA for Sensitive Data Systems - Critical

Overview: The organization does not require Multi-Factor Authentication (MFA) for access to sensitive data systems. This leaves critical data vulnerable to unauthorized access if primary credentials are stolen or guessed. A breach of these systems could result in significant data loss, regulatory fines, reputational damage, and operational disruption.

Affected Elements:

  • Sensitive data systems
  • Databases
  • File shares

Insufficient Security Awareness Training for New Employees - Medium

Overview: The organization does not conduct security awareness training for new employees. New hires may not be aware of the organization's security policies, common cyber threats (e.g., phishing, social engineering), or best practices for data handling. This increases the likelihood of human error leading to security incidents.

Affected Elements:

  • All employees
  • Organizational data
  • Network

Lack of Annual Security Awareness Training - Medium

Overview: The organization does not conduct security awareness training for all employees at least once per year. Regular training is crucial to keep employees updated on evolving threats, reinforce security best practices, and maintain a strong security culture. Without it, employees may become complacent or unaware of new attack vectors.

Affected Elements:

  • All employees
  • Organizational data
  • Network

Open HTTP Port (80) - Medium

Overview: The network scan identified an open HTTP port (80). While HTTP is a standard protocol for web traffic, it transmits data, including potentially sensitive information, in plain text without encryption. This makes it vulnerable to eavesdropping, man-in-the-middle attacks, and session hijacking. If a web server is running on this port, it could also be susceptible to various web application vulnerabilities if not properly secured and patched.

Affected Elements:

  • Network perimeter
  • Web server (if present)
  • Client-server communications over port 80

Recommendations

  • To mitigate the identified risks, it is strongly recommended to implement Multi-Factor Authentication (MFA) for all email accounts and sensitive data systems immediately. This is a critical step to prevent unauthorized access even if credentials are compromised. Additionally, a comprehensive security awareness training program should be established, including mandatory training for all new employees and annual refresher training for the entire workforce. This will help educate employees on current threats and best security practices. For the open HTTP port 80, it is advised to assess its purpose. If it hosts a web service, ensure it is properly secured, regularly patched, and ideally, migrate to HTTPS (port 443) to encrypt all traffic. If the port is not essential, it should be closed or restricted.

Conclusion

The organization currently faces significant cybersecurity risks primarily due to the absence of MFA for key systems and a lack of consistent security awareness training. While MFA is in place for computer logins and an acceptable use policy exists, these measures are insufficient to protect against modern threats. The open HTTP port also presents a potential entry point for attackers. Addressing these vulnerabilities is crucial to enhance the organization's overall security posture and readiness against cyberattacks.
