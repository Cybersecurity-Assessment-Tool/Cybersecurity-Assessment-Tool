Cybersecurity Assessment Report

Overview

The organization operates an Nginx web server on port 443 (HTTPS), indicating a public-facing web presence. While Multi-Factor Authentication (MFA) is implemented for email and sensitive data systems, it is notably absent for computer logins. The organization has an acceptable use policy and provides security awareness training for new employees, but lacks annual security awareness training for all staff.

Identified Risks

Lack of MFA for Computer Logins - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for computer logins. This significantly increases the risk of unauthorized access to employee workstations and the internal network if credentials are compromised through phishing, keylogging, or other attack vectors. Without MFA, a single compromised password can grant an attacker direct access to internal resources, leading to potential data breaches, malware propagation, and lateral movement within the network. This vulnerability was identified through the organizational policy data.

Affected Elements:

  • Employee workstations
  • Internal network
  • User accounts
  • Sensitive data on local machines

Insufficient Annual Security Awareness Training - Medium

Overview: While new employees receive security awareness training, the organization does not conduct annual security awareness training for all employees. This creates a gap in ongoing education, leaving staff vulnerable to evolving cyber threats such as new phishing techniques, social engineering tactics, and malware trends. A lack of regular reinforcement can lead to a decline in security best practices, increasing the likelihood of human error contributing to security incidents, data loss, or system compromise. This vulnerability was identified through the organizational policy data.

Affected Elements:

  • All employees
  • Organizational data
  • Network security
  • Incident response capabilities

Recommendations

  • To mitigate the identified risks, it is strongly recommended that the organization implements Multi-Factor Authentication (MFA) for all computer logins to significantly enhance workstation and network security. Additionally, a mandatory annual security awareness training program should be established for all employees to ensure continuous education on current threats and best practices. Furthermore, regular patching and updates for the Nginx web server (currently version 1.18.0) should be performed to address any known vulnerabilities and maintain a secure public-facing presence.

Conclusion

The organization demonstrates a foundational understanding of security with MFA for critical systems and initial employee training. However, the absence of MFA for computer logins and a comprehensive annual security awareness program for all employees represent significant vulnerabilities that could be exploited by attackers. Addressing these gaps is crucial for enhancing the overall security posture, reducing the risk of unauthorized access, and fostering a more resilient cybersecurity environment.
