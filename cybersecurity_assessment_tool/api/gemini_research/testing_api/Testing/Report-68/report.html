Cybersecurity Assessment Report

Overview

A network scan identified an open HTTP port (80) on at least one host, which could expose web services to potential vulnerabilities. Furthermore, the organization's security posture indicates a lack of Multi-Factor Authentication (MFA) for computer logins, significantly increasing the risk of unauthorized access. New employees are not provided with security awareness training, leaving them susceptible to social engineering and unaware of critical security protocols. While annual security awareness training is conducted for all employees and an acceptable use policy is in place, these gaps represent notable areas for improvement.

Identified Risks

Open HTTP Port - Medium

Overview: An Nmap scan detected port 80 (HTTP) as open on a host within the network. An open HTTP port can expose web services to various attacks, including web application vulnerabilities, unpatched server software exploits, or information disclosure. If not properly secured, monitored, and patched, this could lead to unauthorized access, data breaches, or system compromise.

Affected Elements:

  • Port 80
  • Host(s) with open port 80

Lack of MFA for Computer Logins - High

Overview: The organization does not require Multi-Factor Authentication (MFA) for logging into computers. This significantly weakens the authentication process, making systems highly vulnerable to credential theft, phishing attacks, and brute-force attempts. If an attacker obtains a user's password, they can gain unauthorized access to the computer and potentially the network without an additional layer of security.

Affected Elements:

  • All employee computers
  • User login processes

No Security Awareness Training for New Employees - Medium

Overview: New employees do not receive security awareness training upon onboarding. This leaves them unprepared to identify and respond to common cybersecurity threats such as phishing, malware, and social engineering attacks. A lack of initial training increases the likelihood of human error leading to security incidents, data breaches, or compliance violations.

Affected Elements:

  • New employees
  • Organizational data
  • Email systems

Recommendations

  • To mitigate the identified risks, it is strongly recommended to immediately review the necessity and security configuration of any open port 80, ensuring all web services are patched, hardened, and ideally protected with HTTPS. Implementing mandatory Multi-Factor Authentication (MFA) for all computer logins is critical to significantly enhance authentication security and protect against credential-based attacks. Furthermore, a comprehensive security awareness training program must be integrated into the onboarding process for all new employees to ensure they are equipped with the knowledge to identify and avoid common cyber threats from day one. Regular security audits and vulnerability assessments should also be conducted to proactively identify and address emerging risks.

Conclusion

The organization currently faces several significant cybersecurity risks, primarily stemming from an open HTTP port, the absence of MFA for computer logins, and a gap in security awareness training for new employees. While some foundational security practices are in place, these vulnerabilities present clear pathways for potential unauthorized access and data compromise. Addressing these critical areas will substantially improve the organization's overall security posture and readiness against evolving cyber threats.
