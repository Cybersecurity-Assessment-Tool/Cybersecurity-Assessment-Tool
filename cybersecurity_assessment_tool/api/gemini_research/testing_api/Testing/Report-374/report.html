Cybersecurity Assessment Report

Overview

The organization's network exhibits several critical security vulnerabilities despite some robust security practices. A network scan revealed an active FTP server running vsftpd 2.3.4 on port 21, which allows anonymous login. This version of vsftpd is significantly outdated and known to have critical vulnerabilities, including a backdoor. Furthermore, a significant portion of the workstations are still operating on Windows 7, an end-of-life operating system that no longer receives security updates, leaving them highly susceptible to modern threats. While the organization has implemented Multi-Factor Authentication (MFA) for email, computer logins, and sensitive data systems, and conducts annual security awareness training for all employees, a critical gap exists as new employees do not receive initial security awareness training. An acceptable use policy is in place.

Identified Risks

Anonymous FTP Access and Outdated vsftpd - Critical

Overview: The Nmap scan identified an FTP server (vsftpd 2.3.4) on port 21 that allows anonymous login. This poses a critical risk as unauthorized users can access, upload, or download files, potentially leading to data breaches, malware distribution, or system compromise. The service version (2.3.4) is also extremely old and known to have critical vulnerabilities, including a backdoor (CVE-2011-2523), making the server highly exploitable.

Affected Elements:

  • FTP Server (vsftpd 2.3.4)
  • Port 21

Outdated Windows Operating System - Critical

Overview: The organization's workstations are running Windows 7, an operating system that has reached its end-of-life and no longer receives security updates from Microsoft. This exposes the systems to known and newly discovered vulnerabilities, making them highly susceptible to malware, exploits, and unauthorized access. Continued use of an unsupported OS significantly increases the attack surface and compliance risks.

Affected Elements:

  • Workstations running Windows 7

Lack of Security Awareness Training for New Employees - High

Overview: The organization does not conduct security awareness training for new employees. This leaves new hires vulnerable to social engineering attacks (phishing, vishing, pretexting) and unaware of organizational security policies and best practices from day one. This significantly increases the risk of human error leading to security incidents and compromises the overall security posture.

Affected Elements:

  • All new employees
  • Organizational security posture

Recommendations

  • Immediate action is required to address the critical vulnerabilities identified. The FTP server allowing anonymous access must be secured immediately by disabling anonymous login and either updating vsftpd to a current, supported version or decommissioning the service if it is not essential. All workstations running Windows 7 must be upgraded to a currently supported operating system (e.g., Windows 10 or 11) to ensure they receive vital security updates. Furthermore, a mandatory security awareness training program for all new employees should be implemented to educate them on common threats and organizational security policies from their first day. The organization should continue its practice of requiring MFA for email, computer logins, and sensitive data systems, and maintain its annual security awareness training for all employees, reinforcing a strong security culture.

Conclusion

The organization demonstrates a mixed security posture. While strong controls like MFA and annual security awareness training are in place, critical vulnerabilities such as an anonymously accessible and severely outdated FTP server, and the widespread use of end-of-life Windows 7 workstations, pose immediate and severe risks. The absence of initial security awareness training for new employees also represents a significant human factor vulnerability. Addressing these critical issues is paramount to improving the organization's overall cybersecurity readiness and reducing its exposure to potential breaches and operational disruptions.
