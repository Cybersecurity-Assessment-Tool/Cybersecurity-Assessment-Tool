Cybersecurity Assessment Report

Overview

The organization's network currently has an open MySQL port (3306) running version 5.7.33, which is a significant exposure point. While Multi-Factor Authentication (MFA) is implemented for email and computer logins, it is notably absent for sensitive data systems. The organization has an acceptable use policy and conducts annual security awareness training for all employees, but there is a gap in providing security awareness training for new employees during their onboarding process.

Identified Risks

Database Exposure - High

Overview: MySQL port 3306 is open to the network. This exposes the database to potential unauthorized access, data breaches, and denial-of-service attacks if not properly secured. This was identified via network port scanning.

Affected Elements:

  • Port 3306 (MySQL)

Inadequate Access Control for Sensitive Data - Critical

Overview: The organization does not require Multi-Factor Authentication (MFA) to access sensitive data systems. This significantly increases the risk of unauthorized access to critical information if credentials are compromised, as a single factor (password) is sufficient for authentication. This was identified through organizational policy review.

Affected Elements:

  • Sensitive data systems

Insufficient Onboarding Security Training - Medium

Overview: The organization does not conduct security awareness training for new employees. This leaves new hires vulnerable to social engineering attacks, phishing, and other common threats during their initial employment period, potentially introducing vulnerabilities before they receive annual training. This was identified through organizational policy review.

Affected Elements:

  • New employees
  • Organizational security posture

Recommendations

  • To mitigate the identified risks, it is strongly recommended to immediately implement firewall rules to restrict access to MySQL port 3306, allowing connections only from trusted internal sources or via a secure VPN. Furthermore, Multi-Factor Authentication (MFA) must be enforced for all sensitive data systems to significantly enhance access control and protect critical information. A mandatory security awareness training program should be established for all new employees as part of their onboarding process to ensure they are equipped with essential cybersecurity knowledge from day one. The organization should continue its annual security awareness training for all employees and regularly review and update its acceptable use policy.

Conclusion

The organization demonstrates a foundational commitment to security with MFA for email and computer logins, an acceptable use policy, and annual security awareness training. However, critical vulnerabilities exist, including an exposed MySQL database, the absence of MFA for sensitive data systems, and a lack of initial security training for new hires. Addressing these specific gaps is crucial for significantly improving the overall security posture and reducing the risk of data breaches and unauthorized access.
