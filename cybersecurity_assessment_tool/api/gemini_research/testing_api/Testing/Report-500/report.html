Cybersecurity Assessment Report

Overview

The organization's network shows an open HTTP port (80), which could be an entry point if not properly secured. While multi-factor authentication (MFA) is consistently applied for email, computer logins, and sensitive data systems, significant gaps exist in foundational security practices. Specifically, there is no employee acceptable use policy, and new employees do not receive security awareness training, although annual training is conducted for all staff.

Identified Risks

Lack of Employee Acceptable Use Policy (AUP) - High

Overview: The absence of an Employee Acceptable Use Policy (AUP) means there are no formal guidelines for employees regarding the appropriate use of organizational IT resources. This can lead to misuse of company assets, increased risk of data breaches, and legal liabilities. Identified from organizational policy review.

Affected Elements:

  • All Employees
  • Organizational IT Resources
  • Data Integrity

Inadequate Security Awareness Training for New Employees - High

Overview: New employees are not receiving security awareness training upon onboarding. This leaves them vulnerable to common cyber threats like phishing, social engineering, and malware, potentially compromising organizational security from day one. Identified from organizational training policy review.

Affected Elements:

  • New Employees
  • Organizational Network
  • Data Security

Open HTTP Port (80) - Medium

Overview: An open HTTP port (80) was detected on the network. While not a direct vulnerability, an unencrypted HTTP service can expose sensitive information in transit and may indicate an outdated web server or a lack of HTTPS enforcement. This port could also be a target for various web-based attacks if the underlying service is not properly secured and patched. Identified via Nmap scan.

Affected Elements:

  • Network Perimeter
  • Web Server (Port 80)

Recommendations

  • It is strongly recommended that the organization develop and implement a comprehensive Employee Acceptable Use Policy (AUP) to establish clear guidelines for IT resource usage. Furthermore, immediate action should be taken to integrate mandatory security awareness training into the onboarding process for all new employees. For the open HTTP port 80, it is crucial to ensure that all web services are configured to use HTTPS exclusively, redirecting all HTTP traffic, and that the underlying web server software is fully patched and securely configured. Regular vulnerability scanning and penetration testing should be conducted to identify and remediate further potential weaknesses.

Conclusion

The organization demonstrates a strong commitment to multi-factor authentication across critical access points. However, significant foundational security weaknesses exist, particularly concerning employee policies and initial security training. The presence of an open HTTP port also warrants immediate attention. Addressing these areas is crucial for enhancing the overall security posture and reducing the attack surface.
