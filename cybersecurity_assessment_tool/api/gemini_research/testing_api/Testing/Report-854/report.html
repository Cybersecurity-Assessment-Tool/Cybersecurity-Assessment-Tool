Cybersecurity Assessment Report

Overview

The organization's network currently presents several critical vulnerabilities. A host was identified with an open FTP service (vsftpd 2.3.4) allowing anonymous login, posing a significant data leakage and integrity risk. Furthermore, the organization lacks fundamental security controls, including the absence of Multi-Factor Authentication (MFA) for email, computer logins, and sensitive data systems. There is no formal employee acceptable use policy, and while new employees receive security awareness training, it is not conducted annually for all staff. Additionally, workstations are running an outdated and unsupported operating system (Windows 7), leaving them exposed to known exploits.

Identified Risks

Anonymous FTP Access - High

Overview: The FTP service (vsftpd 2.3.4) on the scanned host allows anonymous login. This configuration enables unauthorized users to access, potentially modify, or exfiltrate files without authentication, severely compromising data confidentiality and integrity. This vulnerability was identified through an Nmap scan utilizing the 'ftp-anon' script.

Affected Elements:

  • FTP Server (vsftpd 2.3.4)
  • Host with port 21 open

Lack of Multi-Factor Authentication (MFA) - Critical

Overview: The organization does not require Multi-Factor Authentication (MFA) for accessing email, logging into computers, or accessing sensitive data systems. This significantly increases the risk of unauthorized access if user credentials are compromised through phishing, brute-force attacks, or other means, as a single factor (password) is the only barrier to entry.

Affected Elements:

  • Email systems
  • User login systems
  • Sensitive data systems

Absence of Employee Acceptable Use Policy (AUP) - Medium

Overview: The organization lacks a formal Employee Acceptable Use Policy (AUP). Without clear guidelines, employees may engage in activities that inadvertently or intentionally compromise security, misuse company resources, or expose the organization to legal and compliance risks. This absence creates an environment of inconsistent security practices and increases the potential for insider threats.

Affected Elements:

  • All employees
  • IT systems
  • Organizational data

Insufficient Security Awareness Training - Medium

Overview: While new employees receive security awareness training, there is no mandatory annual training for all employees. This oversight means that existing staff may not be updated on evolving cyber threats, new company policies, or best practices, leading to a degradation of the 'human firewall' over time and increased susceptibility to social engineering attacks and other human-factor vulnerabilities.

Affected Elements:

  • All employees
  • Organizational security posture

Outdated Operating System (Windows 7) - High

Overview: Workstations within the organization are running Windows 7, an operating system that has reached its end-of-life and no longer receives security updates or patches from Microsoft. This leaves these systems highly vulnerable to known exploits and emerging threats, making them easy targets for malware, ransomware, and other forms of cyberattacks, significantly increasing the risk of system compromise and data breaches.

Affected Elements:

  • Workstations running Windows 7

Recommendations

  • To mitigate the identified risks, immediate action is required. First, disable anonymous FTP access and, if FTP functionality is essential, implement a secure alternative like SFTP or FTPS with strong authentication. Second, deploy and enforce Multi-Factor Authentication (MFA) across all critical access points, including email, computer logins, and sensitive data systems, to significantly enhance account security. Third, develop and implement a comprehensive Employee Acceptable Use Policy (AUP) and ensure all employees acknowledge and adhere to it. Fourth, establish a mandatory annual security awareness training program for all employees to keep them informed about current threats and best practices. Finally, prioritize upgrading all workstations running Windows 7 to a currently supported operating system (e.g., Windows 10 or 11) to ensure they receive vital security updates and patches. Additionally, implement a robust patch management program for all software and operating systems, and consider regular vulnerability scanning and network segmentation to further enhance the security posture.

Conclusion

The organization currently faces a high level of cybersecurity risk due to a combination of critical technical vulnerabilities and significant gaps in foundational security policies and training. The presence of anonymous FTP, the complete absence of MFA, and the use of end-of-life operating systems create easily exploitable entry points for attackers. Furthermore, the lack of an AUP and insufficient ongoing security awareness training for employees indicate a weak security culture and an underdeveloped 'human firewall.' Without prompt and comprehensive remediation efforts addressing both technical and policy-based weaknesses, the organization remains highly susceptible to data breaches, system compromises, and operational disruptions. A strategic and multi-layered approach is essential to rapidly improve the overall security posture and readiness.
