Recommendations

The following actions are recommended to mitigate the identified risks. They are prioritized based on severity and potential impact.

Immediate Priority (Remediate within 24-48 hours)

  • RISK-001 - Remediate FTP Server:

    1. Immediately take the FTP server offline or firewall it from public access.

    2. If the service is business-critical, upgrade the vsftpd software to the latest stable version and disable anonymous access.

    3. If FTP is not required, decommission the service entirely.

    4. Consider replacing FTP with a secure alternative such as SFTP (SSH File Transfer Protocol).

High Priority (Remediate within 30 days)

  • RISK-002 - Implement MFA:

    1. Enable and enforce MFA for all user accounts on the email system.

    2. Identify all systems classified as containing sensitive data and enforce MFA for access.

  • RISK-003 - Establish Security Governance:

    1. Develop and ratify a formal Acceptable Use Policy (AUP) for all employees.

    2. Implement a mandatory security awareness training program for all new and existing employees, to be conducted at least annually.

Medium Priority (Remediate within 90 days)

  • RISK-004 - Upgrade Operating Systems:

    1. Develop a plan to upgrade all workstations from Windows 7 to a modern, supported operating system such as Windows 10 or Windows 11.

    2. Prioritize the upgrade for systems used by employees with access to sensitive data.
