6.0 Recommendations

The following actionable recommendations are provided to mitigate the identified risks. They are prioritized based on severity and exploitability.

Priority 1: Immediate Actions (Within 24-48 Hours)

  1. Mitigate Vulnerable FTP Server (R-01):

    • Immediately take the FTP server offline.

    • If the service is not required for business, decommission it permanently.

    • If the service is required, it must be upgraded to a modern, patched version of FTP software (e.g., a recent version of vsftpd or ProFTPD) and reconfigured to disable anonymous access.

    • Implement strict firewall rules to limit access to trusted IP addresses only.

    • Consider replacing FTP with a more secure file transfer protocol like SFTP (SSH File Transfer Protocol).

Priority 2: High-Impact Actions (Within 1-2 Weeks)

  1. Implement Multi-Factor Authentication (R-02):

    • Enable and enforce MFA across all user accounts for email access (e.g., Office 365, Google Workspace).

    • Identify all systems classified as containing sensitive data and enforce MFA for access.

  2. Develop and Implement an Acceptable Use Policy (R-03):

    • Draft a formal AUP that defines rules for technology use, data handling, and security responsibilities for all employees.

    • Communicate the policy to all staff and require acknowledgment of receipt.

Priority 3: Ongoing/Planned Initiatives

  1. Continue Operating System Upgrades (R-04):

    • Execute the existing plan to upgrade or replace all workstations running Windows 7 to a modern, supported operating system like Windows 10 or 11.
