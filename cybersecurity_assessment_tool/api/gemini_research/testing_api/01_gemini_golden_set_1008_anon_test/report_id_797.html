Recommendations

The following actions are recommended to mitigate the identified risks. They are prioritized based on severity and potential impact.

Immediate Priority (Remediate within 72 hours)

  1. Restrict RDP Access: Immediately close port 3389 on the firewall for traffic originating from the public internet at . If remote administrative access is required, it must be placed behind a Virtual Private Network (VPN) that requires Multi-Factor Authentication.

High Priority (Remediate within 30 days)

  1. Enforce MFA for Email: Implement and enforce MFA for all user access to the organization’s email system (e.g., Office 365, Google Workspace). This is one of the most effective controls to prevent account takeovers.

  2. Deploy MFA for Sensitive Systems: Identify all systems containing sensitive data and deploy MFA for all user access, especially for users with administrative privileges.

  3. Develop an Acceptable Use Policy: Create and implement a formal AUP that clearly defines the rules and expectations for employees when using company technology and data. Require all employees to read and acknowledge the policy.

Informational

  1. Enhance Security Training: Continue the excellent practice of security awareness training. Update the training modules to include specific information on the importance of MFA and the new AUP once it is implemented.
