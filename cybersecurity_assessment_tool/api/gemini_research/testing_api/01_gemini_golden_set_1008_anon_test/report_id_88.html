Recommendations

The following actionable recommendations are provided to address the identified risks. They are prioritized based on severity to guide remediation efforts effectively.

1. Implement Comprehensive Multi-Factor Authentication (Critical)

  • Immediate Action: Enforce MFA for all user access to email systems (e.g., Office 365, Google Workspace). This is the single most effective control to prevent business email compromise.

  • Next Step: Deploy MFA for all remote access solutions (VPN) and for logging into company-managed computers (e.g., via Windows Hello for Business, Duo, or a similar solution).

  • Goal: Ensure that no critical system relies solely on a password for authentication.

2. Develop and Enforce an Acceptable Use Policy (High)

  • Immediate Action: Draft a formal Acceptable Use Policy (AUP) that clearly defines the rules for using company networks, computers, email, and data.

  • Next Step: Require all current employees to read and formally acknowledge the new policy.

  • Goal: Integrate the AUP acknowledgement into the new employee onboarding process to establish clear expectations from day one.

3. Enhance the Security Awareness Training Program (High)

  • Immediate Action: Develop a security awareness training module specifically for new hires. This module should be a mandatory part of the onboarding process.

  • Content to Include: The training should cover key topics such as phishing identification, password security, data handling, and the newly created AUP.

  • Goal: Foster a culture of security throughout the organization, starting from an employee’s first day. Continue with annual refresher training to keep security top-of-mind.
