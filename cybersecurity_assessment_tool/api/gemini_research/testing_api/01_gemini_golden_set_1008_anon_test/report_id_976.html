Recommendations

The following actions are recommended to address the identified risks. Recommendations are prioritized based on severity and potential impact.

ORG-001: Implement MFA for Email (Critical)

  • Action: Immediately procure, configure, and enforce Multi-Factor Authentication (MFA) for all user accounts with access to the organization’s email system.

  • Justification: MFA is the single most effective control for preventing unauthorized account access. It provides a critical second layer of defense that protects against compromised credentials, drastically reducing the likelihood of a successful email-based attack.

  • Priority: Critical

ORG-002: Integrate Security Training into Onboarding (High)

  • Action: Develop a mandatory security awareness training module and integrate it into the formal onboarding process for all new employees and contractors. This training should cover, at a minimum, acceptable use policies, phishing and social engineering awareness, and data handling procedures.

  • Justification: A strong security culture starts on day one. Training new hires ensures a baseline level of security knowledge across the organization, reducing human error and strengthening the human firewall.

  • Priority: High
