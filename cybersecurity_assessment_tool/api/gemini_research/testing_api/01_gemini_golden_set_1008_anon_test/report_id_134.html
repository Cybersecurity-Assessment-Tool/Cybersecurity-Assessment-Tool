Recommendations

Based on the identified risks, the following actions are recommended. They are prioritized to address the most critical threats first.

Immediate Actions (Due within 72 hours)

  1. Remediate FTP Server (R-001, R-002):

    • Immediately take the FTP server offline to prevent exploitation.

    • If FTP is a business necessity, upgrade the vsftpd software to the latest stable version. If not, decommission the service entirely.

    • Disable anonymous FTP login. Enforce strong, unique credentials for all authorized users.

    • Consider replacing FTP with a more secure protocol like SFTP (SSH File Transfer Protocol).

High-Priority Actions (Due within 30 days)

  1. Develop and Implement an Acceptable Use Policy (R-003):

    • Draft a comprehensive AUP that covers data handling, internet usage, password security, and incident reporting.

    • Require all current and new employees to read and formally acknowledge the policy.

  2. Establish a Security Awareness Training Program (R-004):

    • Procure and implement a training solution for all employees.

    • Ensure the program includes modules on phishing, password security, and social engineering.

    • Make training mandatory for all new hires and require annual refresher courses for all staff.

Medium-Priority Actions (Due within 90-180 days)

  1. Plan and Execute OS Upgrades (R-005):

    • Develop a phased plan to upgrade or replace all workstations running Windows 7.

    • Prioritize machines used by employees with access to sensitive data.

    • Ensure all new systems are deployed with a modern, supported operating system.
