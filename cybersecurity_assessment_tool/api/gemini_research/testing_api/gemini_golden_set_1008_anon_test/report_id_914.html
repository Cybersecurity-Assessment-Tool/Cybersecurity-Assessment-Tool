Recommendations

To mitigate the identified risks and enhance the overall cybersecurity posture, the following recommendations are provided. These are categorized by urgency and impact.

Immediate Actions (Critical & High Risks)

  • Decommission/Upgrade Outdated FTP Service:

    • Immediately disable or remove the vsftpd 2.3.4 service on port 21.

    • If FTP functionality is required, replace it with a secure alternative (e.g., SFTP or FTPS) running on a modern, patched server.

    • Ensure no anonymous access is permitted on any file transfer service.

  • Implement Mandatory Annual Security Awareness Training:

    • Establish a program for mandatory, comprehensive security awareness training for all employees, conducted at least annually.

    • Training should cover topics such as phishing, social engineering, password hygiene, data handling, and incident reporting.

Short-Term Actions (Medium Risks)

  • Upgrade Outdated Operating Systems:

    • Initiate a project to upgrade all workstations running Windows 7 to a supported operating system, such as Windows 10 or 11.

    • Develop a robust patch management strategy to ensure all systems are kept up-to-date with the latest security patches.

General Security Enhancements

  • Regular Vulnerability Scanning: Implement a schedule for regular, automated vulnerability scans across the network to proactively identify new risks.

  • Network Segmentation: Consider segmenting the network to limit the blast radius of any potential compromise.

  • Incident Response Plan Review: Ensure an up-to-date incident response plan is in place and regularly tested.

  • Least Privilege Principle: Enforce the principle of least privilege for all user accounts and services.

  • Strong Configuration Management: Implement secure configuration baselines for all systems and services.
