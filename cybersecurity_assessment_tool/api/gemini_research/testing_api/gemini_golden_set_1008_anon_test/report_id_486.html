Recommendations

To mitigate the identified risks and enhance the overall cybersecurity posture, the following actionable recommendations are provided:

  1. Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Create a comprehensive AUP outlining clear guidelines for acceptable use of IT assets, internet, email, data handling, and prohibited activities.

    • Ensure all employees read, understand, and formally acknowledge the AUP upon hiring and periodically thereafter (e.g., annually).

    • Integrate AUP review into employee onboarding and annual refresher training to ensure continuous awareness.

  2. Establish Mandatory Security Awareness Training for New Employees:

    • Develop and implement a mandatory security awareness training program specifically for new employees as part of the onboarding process.

    • This training should cover fundamental topics such as phishing identification, social engineering tactics, password best practices, data handling, incident reporting procedures, and a review of the organization’s AUP.

    • Ensure completion of this training is tracked and enforced.

  3. Review and Address "Unencrypted Web Server" Risk (even if currently mitigated):

    • Investigate the historical context of the "Unencrypted Web Server" risk to understand its origin and confirm sustainable remediation.

    • Establish clear policies and technical controls to ensure that if port 80 (or any other port for web services) were to be opened in the future, it is done so securely, requiring HTTPS/TLS encryption for all web traffic.

    • Implement regular external and internal vulnerability scanning to proactively identify and secure any inadvertently exposed or unencrypted web services.

    • Ensure all publicly accessible services are part of a robust patch management and configuration management program.

  4. Continuous Monitoring and Improvement:

    • Establish a process for regular review and update of all security policies and procedures (e.g., AUP, training modules) to adapt to evolving threats.

    • Conduct periodic internal and external vulnerability assessments and penetration tests to identify potential weaknesses before they can be exploited.

    • Maintain an up-to-date risk register, continuously monitor for new threats, and regularly review and reassess identified risks to ensure ongoing mitigation effectiveness.
