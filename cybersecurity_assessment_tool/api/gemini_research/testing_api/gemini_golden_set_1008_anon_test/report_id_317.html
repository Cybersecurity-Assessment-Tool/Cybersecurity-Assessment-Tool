Recommendations

The following recommendations are provided to address the identified risks and enhance the overall cybersecurity posture of [Organization Name].

  1. Implement Mandatory New Employee Security Awareness Training

    • Develop and implement a mandatory security awareness training program for all new employees. This training should be completed before they are granted access to any organizational systems or sensitive data.

    • Training content should cover essential topics such as identifying phishing attempts, best practices for strong passwords, the dangers of social engineering, proper data handling, and the process for reporting suspicious activities or incidents.

    • Ensure the training is engaging, comprehensive, and includes practical examples relevant to the organization’s operational environment.

  2. Secure Remote Desktop Protocol (RDP) Access

    • Immediate Action: Restrict RDP access to a very limited set of trusted IP addresses (e.g., from the corporate office or specific administrative jump boxes) using firewall rules. If direct internet access is strictly required, enforce Network Level Authentication (NLA) and ensure strong, complex passwords are used, ideally combined with Multi-Factor Authentication (MFA) for RDP logins.

    • Long-Term Solution: Implement a Virtual Private Network (VPN) solution for all remote access to internal resources, including RDP. This will ensure that RDP connections are encapsulated within an encrypted tunnel and not directly exposed to the public internet. Disable direct internet-facing RDP access once the VPN solution is fully operational.

    • Regularly patch and update RDP clients and servers to protect against known vulnerabilities.

    • Implement account lockout policies to prevent brute-force attacks on RDP.

Conclusion
