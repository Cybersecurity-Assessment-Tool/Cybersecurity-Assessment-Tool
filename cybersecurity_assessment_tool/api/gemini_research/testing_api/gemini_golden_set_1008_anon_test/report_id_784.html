5. Recommendations

To mitigate the identified risks and strengthen the organization’s overall cybersecurity posture, the following actionable recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) for All Logins:

    • Extend MFA requirements to all computer logins across the organization.

    • Implement MFA for access to all sensitive data systems. This is a critical control to prevent unauthorized access even if primary credentials are compromised.

  2. Develop and Enforce an Employee Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that clearly outlines acceptable use of all organizational IT resources, including hardware, software, network, and data.

    • Ensure all employees review and sign the AUP upon hiring and annually thereafter.

  3. Establish a Robust Security Awareness Training Program:

    • Implement mandatory security awareness training for all new employees during their onboarding process. This should cover fundamental security principles, threat identification, and reporting procedures.

    • Conduct mandatory annual security awareness training for all employees. This training should be updated regularly to reflect current threat landscapes and organizational policies.

    • Consider implementing phishing simulations to reinforce training and assess employee vigilance.

  4. Regularly Review and Update Security Policies:

    • Establish a schedule for periodic review and update of all security policies and procedures (e.g., AUP, data handling policies, incident response plans) to ensure they remain current and effective.

These recommendations focus on addressing foundational security gaps that, if left unaddressed, pose significant risks to the organization’s data, systems, and overall operational integrity. Prioritizing these controls will substantially improve the organization’s defense against common cyber threats.
