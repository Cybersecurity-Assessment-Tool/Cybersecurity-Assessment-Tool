Recommendations

Based on the assessment findings, the following recommendations are provided to enhance the organization’s cybersecurity posture.

Multi-Factor Authentication (MFA)

  • Implement MFA Across All Critical Access Points: Mandate Multi-Factor Authentication for all email access, computer logins, and sensitive data system access. This significantly reduces the risk of unauthorized access even if credentials are compromised.

Remote Desktop Protocol (RDP) Security

  • Restrict External RDP Access Immediately: Block direct inbound RDP (Port 3389) connections from the internet at the perimeter firewall.

  • Implement Secure Remote Access: For legitimate remote access requirements, deploy a Virtual Private Network (VPN) solution. All RDP connections should traverse the VPN tunnel, adding an essential layer of security.

  • Harden RDP Configuration: If internal RDP is necessary, ensure strong password policies are enforced, network-level authentication (NLA) is enabled, and all systems are regularly patched.

Policy & Training

  • Develop and Implement an Acceptable Use Policy (AUP): Create a formal AUP that outlines acceptable and unacceptable use of organizational IT resources. Communicate this policy clearly to all employees and require acknowledgment.

  • Leverage Existing Security Awareness Training: Continue and potentially expand the existing security awareness training programs to cover new policies and emerging threats. (It is positive that security awareness training is already being conducted for new and all employees annually).
