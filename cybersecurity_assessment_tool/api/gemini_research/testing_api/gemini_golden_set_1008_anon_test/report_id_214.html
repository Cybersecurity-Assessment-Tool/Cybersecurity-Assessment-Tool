Risk Assessment

Based on the security control review and technical analysis, the following risks have been identified.

Identified Risks
ID Risk Name Overview Severity
1 Lack of MFA for Email Access The organization does not require Multi-Factor Authentication (MFA) for accessing email accounts, leaving them highly vulnerable to credential stuffing, phishing, and brute-force attacks. High
2 Insufficient Security Awareness Training for New Employees New employees do not receive security awareness training as part of their onboarding. This leaves them unprepared to identify and respond to common cybersecurity threats, increasing the risk of human-factor-driven breaches. High
3 Insufficient Annual Security Awareness Training for All Employees The organization does not conduct recurring annual security awareness training for all employees. This results in outdated knowledge, increased susceptibility to evolving threats, and a general decline in security posture over time. High

Recommendations

To mitigate the identified risks and improve the overall cybersecurity posture, the following recommendations are provided:

  1. Implement Mandatory MFA for Email Access: Deploy and enforce Multi-Factor Authentication (MFA) for all user accounts accessing email. This is a critical control that significantly reduces the risk of unauthorized access even if user passwords are compromised through phishing or other means.

  2. Establish New Employee Security Awareness Training: Implement a comprehensive security awareness training program for all new employees as a mandatory component of their onboarding process. This training should cover essential topics such as phishing identification, social engineering tactics, password best practices, data handling policies, and incident reporting procedures.

  3. Implement Annual Security Awareness Training for All Employees: Conduct mandatory, recurring security awareness training for all employees at least once per year. This ensures that employees are continuously updated on evolving cyber threats, new security policies, and reinforces best practices, maintaining a strong human firewall against attacks.
