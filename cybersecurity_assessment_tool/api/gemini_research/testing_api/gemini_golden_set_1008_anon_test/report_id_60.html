Risk Assessment

Recommendations

The following recommendations are provided to mitigate the identified risks and enhance the organization’s overall cybersecurity posture.

Policy and Human-Factor Enhancements

  1. Implement MFA for Computer Login: Mandate multi-factor authentication for all computer logins across the organization. This significantly reduces the risk of unauthorized access even if credentials are compromised.

  2. Develop and Implement an Acceptable Use Policy (AUP): Create a comprehensive AUP outlining proper and secure use of company IT resources, data, and internet access. Ensure all employees read, understand, and formally acknowledge this policy upon hire and periodically thereafter.

  3. Mandate Security Awareness Training for New Employees: Implement a mandatory and robust security awareness training program for all new hires as part of their onboarding process. This training should cover essential topics such as phishing, password hygiene, social engineering, and incident reporting.

  4. Establish Regular (Annual) Security Awareness Training for All Employees: Conduct annual (or more frequent) security awareness training for all employees to keep them informed about current threats, evolving attack techniques, and updated best practices. This helps maintain a strong security culture.

Technical Recommendations (Based on Data Gaps)

  1. Conduct a Comprehensive Network Scan: Perform a detailed and professional network scan of all external-facing and internal assets. This scan should identify all open ports, running services, software versions, and potential misconfigurations. The results should then be thoroughly analyzed for vulnerabilities.

  2. Regular Vulnerability Management: Establish a recurring process for vulnerability scanning and penetration testing. Regularly review and address findings to ensure continuous improvement in the technical security posture.
