Recommendations

To address the identified risks and enhance the overall cybersecurity posture, the following recommendations are provided:

Technical Recommendations

  • Restrict Database Access (Port 3306): Implement strict firewall rules to limit access to MySQL port 3306. Ideally, restrict access only to internal, authorized systems or specific IP addresses. For remote access, mandate the use of a VPN.

  • Implement MFA for Computer Logins: Deploy and enforce Multi-Factor Authentication (MFA) for all computer login processes. This significantly reduces the risk of unauthorized access even if credentials are compromised.

Policy and Training Recommendations

  • Develop and Implement an Acceptable Use Policy (AUP): Create a comprehensive AUP that clearly defines employee responsibilities and acceptable conduct when using organizational IT resources. Ensure all employees review and acknowledge the policy annually.

  • Mandate Annual Security Awareness Training: Establish a mandatory annual security awareness training program for all employees. This training should cover current threats (e.g., phishing, social engineering), company policies, and best practices for data protection.
