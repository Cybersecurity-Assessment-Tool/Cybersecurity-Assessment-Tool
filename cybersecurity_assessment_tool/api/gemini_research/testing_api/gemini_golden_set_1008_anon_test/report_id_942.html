Recommendations

To mitigate the identified risks and enhance the overall cybersecurity posture, the following recommendations are provided:

Immediate / Short-Term Recommendations

  1. Implement MFA for Email Access: Immediately deploy and enforce Multi-Factor Authentication for all email accounts. This is a critical control against credential compromise.

  2. Implement MFA for Computer Login: Implement MFA for logging into all organizational computers and workstations.

  3. Develop and Implement an Acceptable Use Policy (AUP): Create a comprehensive AUP that outlines acceptable use of organizational IT resources, and ensure all employees read and acknowledge it.

  4. Initiate Security Awareness Training for New Employees: Establish a mandatory security awareness training program for all new hires as part of their onboarding process.

  5. Conduct Annual Security Awareness Training for All Employees: Implement a recurring (at least annual) security awareness training program for all employees to keep them informed about current threats and best practices.

Long-Term / Strategic Recommendations

  1. Verify and Update Risk Register: Conduct a thorough review of the organization’s existing risk register to ensure all listed vulnerabilities are current and accurately reflect the present environment. Document any remediations or changes in status.

  2. Regular Vulnerability Scanning and Penetration Testing: Establish a schedule for regular internal and external vulnerability scans and periodic penetration tests to proactively identify and address technical weaknesses.

  3. Comprehensive Policy Framework: Develop a robust suite of security policies, including but not limited to, data handling, incident response, password management, and access control.

  4. Continuous Monitoring and Logging: Implement solutions for continuous monitoring of network activity and system logs to detect and respond to suspicious behavior in real-time.
