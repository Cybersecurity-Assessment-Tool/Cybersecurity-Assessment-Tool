Critical Recommendations

RDP Exposure (Port 3389)

  • Immediate Fix: Close port 3389 to external network access. If remote access is required, implement a Virtual Private Network (VPN) solution to restrict RDP access exclusively to authenticated VPN users.

  • Long-Term Fix: If external RDP access is absolutely unavoidable, implement strong security measures:

    • Enforce Network Level Authentication (NLA) for RDP connections.

    • Implement strong, complex password policies and account lockout thresholds.

    • Restrict RDP access to specific trusted source IP addresses via firewall rules.

    • Consider using an RDP gateway or a jump box for added security layers.

High Priority Recommendations

Lack of Multi-Factor Authentication for Computer Login

  • Action: Implement mandatory MFA for all computer logins, especially for administrative accounts and users accessing sensitive systems.

  • Considerations: Evaluate MFA solutions compatible with your operating systems (e.g., Windows Hello for Business, third-party MFA providers) and roll out implementation in phases, starting with high-privilege users.

Insufficient Annual Security Awareness Training

  • Action: Establish and implement a mandatory annual security awareness training program for all employees, including contractors and temporary staff.

  • Content: Training should cover common threats such as phishing, social engineering, ransomware, password hygiene, data handling best practices, and incident reporting procedures.

  • Frequency: Conduct training at least once per year, with additional refresher courses or alerts for emerging threats.

Medium Priority Recommendations

Absence of Employee Acceptable Use Policy (AUP)

  • Action: Develop, formalize, and implement a comprehensive Employee Acceptable Use Policy (AUP) that clearly defines expectations for IT asset usage, internet browsing, email, and data handling.

  • Implementation: Ensure all employees read, understand, and formally acknowledge the AUP upon hiring and annually thereafter. Integrate AUP review into the annual security awareness training.
