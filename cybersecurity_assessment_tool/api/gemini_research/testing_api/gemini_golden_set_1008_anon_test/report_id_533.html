Analysis of Questionnaire Findings

The review of security controls reveals critical weaknesses in two primary areas:

  • Multi-Factor Authentication (MFA): The lack of MFA for email and computer logins represents a severe vulnerability. Credential compromise (e.g., via phishing) could lead to unauthorized access to user accounts and internal systems, potentially bypassing other security measures. While MFA is used for sensitive data systems, the initial access points (email, login) remain unprotected.

  • Security Awareness Training (SAT): The absence of security awareness training for new employees is a significant oversight. New hires are often targets for social engineering and phishing attacks due to unfamiliarity with organizational policies and common threats. This increases the likelihood of human error leading to security incidents.

Recommendations

To mitigate the identified risks and improve the overall cybersecurity posture, the following recommendations are provided. These are prioritized based on their potential impact and severity.

High Priority Recommendations

  1. Secure RDP Access (RDP Exposure):

    • Immediate Fix: Block inbound access to port 3389 from the internet at the perimeter firewall. RDP should generally not be directly exposed.

    • Long-Term Solution: Implement a Virtual Private Network (VPN) or a secure gateway solution (e.g., Azure Bastion, AWS Session Manager) that requires strong authentication (preferably MFA) before users can connect to internal RDP services.

    • Alternative: If RDP is absolutely required externally without a VPN, ensure it is behind a modern RDP gateway with Network Level Authentication (NLA) enabled, strong passwords, and IP restrictions.

  2. Implement MFA for Email Access:

    • Enforce Multi-Factor Authentication for all email accounts. This significantly reduces the risk of account takeover even if passwords are compromised.

    • Educate users on the importance and benefits of MFA.

  3. Implement MFA for Computer Logins:

    • Implement and enforce Multi-Factor Authentication for all computer logins, especially for administrative accounts and critical workstations/servers.

    • This could be achieved via Active Directory integration with MFA solutions or specific endpoint security tools.

Medium Priority Recommendations

  1. Establish Security Awareness Training for New Employees:

    • Develop and implement a mandatory security awareness training program for all new employees during their onboarding process.

    • This training should cover topics such as phishing, social engineering, password best practices, acceptable use policy, and incident reporting procedures.

General Best Practices

  • Regularly review and update security policies and procedures.

  • Conduct periodic vulnerability scans and penetration tests to identify and remediate new exposures.

  • Ensure all systems and applications are regularly patched and updated to protect against known vulnerabilities.

  • Implement robust logging and monitoring to detect and respond to suspicious activities.
