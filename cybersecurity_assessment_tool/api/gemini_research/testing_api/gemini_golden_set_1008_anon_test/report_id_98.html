Recommendations

The following actionable recommendations are provided to mitigate the identified risks and significantly enhance the organization’s cybersecurity posture.

Critical Recommendations

  • Immediately Disable Anonymous FTP Access: Reconfigure the FTP server to strictly disallow anonymous logins. Ensure all FTP access requires strong, unique authentication credentials.

  • Update or Replace FTP Service: Due to its critical vulnerabilities and EOL status, the vsftpd 2.3.4 service must be addressed urgently.

    • If FTP is essential, upgrade vsftpd to the latest stable and supported version, ensuring all security patches are applied.

    • Alternatively, replace FTP with a more secure file transfer protocol such as SFTP (SSH File Transfer Protocol) or SCP (Secure Copy Protocol), or migrate to a secure cloud-based file sharing solution.

    • If FTP functionality is not required, disable and remove the service entirely from the system.

  • Implement MFA for Email Access: Mandate Multi-Factor Authentication for all organizational email accounts. This is a foundational security control to protect against phishing and credential compromise.

High Recommendations

  • Implement MFA for Sensitive Data Systems: Enforce Multi-Factor Authentication for all systems and applications that store, process, or transmit sensitive data. This critical step protects confidential information from unauthorized access.

Medium Recommendations

  • Develop and Implement an Employee Acceptable Use Policy (AUP): Create a clear, comprehensive AUP that outlines acceptable use of all company IT resources, data handling procedures, and security expectations for employees. Ensure all employees review, understand, and acknowledge this policy.

  • Upgrade End-of-Life Operating Systems: Develop and execute a plan to migrate all workstations currently running Windows 7 to a currently supported operating system (e.g., Windows 10 or 11). Ensure all systems are kept up-to-date with the latest security patches and configurations.

General Best Practices

  • Regular Vulnerability Scanning and Penetration Testing: Conduct periodic vulnerability assessments and penetration tests across the entire IT infrastructure to proactively identify and remediate security weaknesses before they can be exploited.

  • Enhanced Security Awareness Training: Continue to reinforce and expand existing security awareness training programs, incorporating advanced topics such as phishing simulations, social engineering tactics, and data privacy best practices.

  • Robust Incident Response Plan: Ensure a well-documented, regularly reviewed, and frequently tested incident response plan is in place to effectively detect, contain, eradicate, and recover from cybersecurity incidents.
