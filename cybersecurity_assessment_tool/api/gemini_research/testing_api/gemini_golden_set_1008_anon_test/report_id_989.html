6. Recommendations

To address the identified risks and improve the overall cybersecurity posture, the following recommendations are provided. These are categorized by the area of concern and prioritized based on their severity.

6.1 Multi-Factor Authentication (MFA)

  • Implement MFA for Computer Logins: Mandate and enforce Multi-Factor Authentication (MFA) for all user logins to company computers and workstations. This significantly reduces the risk of credential theft and unauthorized access.

  • Implement MFA for Sensitive Data Systems: Extend mandatory MFA to all systems that store, process, or transmit sensitive organizational data.

6.2 Policies and Training

  • Develop and Enforce an Acceptable Use Policy (AUP): Create a comprehensive Employee Acceptable Use Policy (AUP) that clearly defines expectations for employee conduct when using company IT resources. Ensure all employees acknowledge and adhere to this policy.

  • Implement Security Awareness Training for New Employees: Establish a mandatory security awareness training program for all new employees as part of their onboarding process.

  • Conduct Annual Security Awareness Training: Implement a mandatory annual security awareness training program for all employees to reinforce best practices, update them on new threats, and ensure ongoing vigilance.

6.3 Network Security and Remote Access

  • Secure or Close Exposed RDP Service:

    • Immediate Fix: Block external access to port 3389 (RDP) at the network perimeter firewall. RDP should ideally never be directly exposed to the internet.

    • Long-term Solution: Implement a robust Virtual Private Network (VPN) solution for all remote access requirements. Users should be required to connect to the VPN before they can access any internal resources, including RDP services. Ensure RDP access is restricted to authenticated VPN users or specific trusted internal networks only. Additionally, ensure strong, unique passwords and consider MFA for internal RDP sessions if direct access is unavoidable.
