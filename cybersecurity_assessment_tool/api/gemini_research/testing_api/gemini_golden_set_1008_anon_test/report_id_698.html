6. Recommendations

Based on the assessment, the following recommendations are provided to improve the cybersecurity posture of [Organization Name]. Prioritization should be given to critical and high-severity risks.

  • Remediate Anonymous FTP Access (Critical):

    • Immediately disable anonymous FTP access on port 21.

    • Decommission the service. If FTP functionality is required, implement a modern, patched, and securely configured file transfer solution (e.g., SFTP, FTPS with strong authentication and access controls).

    • Ensure all public-facing services run current, supported versions and are regularly patched.

  • Implement MFA for Email and Sensitive Data (Critical):

    • Enforce Multi-Factor Authentication (MFA) for all email accounts to significantly reduce the risk of phishing and credential theft.

    • Implement MFA for all sensitive data systems and applications, protecting critical organizational assets.

    • Extend MFA coverage to all user logins where feasible, building upon the existing MFA for computer logins.

  • Upgrade End-of-Life Operating Systems (High):

    • Plan and execute a comprehensive upgrade of all Windows 7 workstations to a currently supported operating system (e.g., Windows 10 or Windows 11).

    • Ensure regular patching and security updates are applied to all operating systems and applications.

  • Develop and Implement an Acceptable Use Policy (High):

    • Create a comprehensive employee acceptable use policy (AUP) that clearly outlines expectations for the use of organizational technology, data, and internet resources.

    • Distribute the AUP to all employees and obtain signed acknowledgments.

  • Establish Robust Security Awareness Training Programs (High):

    • Implement mandatory security awareness training for all new employees as part of their onboarding process.

    • Conduct annual mandatory security awareness training for all existing employees to reinforce best practices and address evolving threats.

    • Training should cover topics such as phishing, social engineering, password best practices, safe browsing, and proper data handling procedures.
