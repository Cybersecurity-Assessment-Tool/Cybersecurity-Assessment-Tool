Recommendations

To address the identified vulnerabilities and enhance the overall cybersecurity posture, the following recommendations are provided:

Immediate Actions (High Priority)

  1. Secure RDP Access (Port 3389):

    • Restrict direct internet access to port 3389. Implement a Virtual Private Network (VPN) for all remote RDP connections.

    • Configure firewall rules to allow RDP access only from trusted IP addresses or the VPN concentrator.

    • Enforce strong, unique passwords and consider implementing MFA for RDP access.

    • Ensure RDP servers are regularly patched and updated.

  2. Implement Multi-Factor Authentication (MFA) for Email:

    • Mandate MFA for all email accounts across the organization. This significantly reduces the risk of account compromise from phishing or stolen credentials.

    • Educate users on the importance and use of MFA.

Strategic Enhancements (Medium Priority)

  1. Establish New Employee Security Awareness Training:

    • Develop and implement a mandatory security awareness training program for all new hires.

    • Cover topics such as phishing, social engineering, password hygiene, and acceptable use policies.

  2. Implement Annual Security Awareness Training:

    • Conduct mandatory annual security awareness training for all employees to reinforce best practices and educate on new threats.

    • Utilize engaging and relevant content to ensure maximum employee participation and retention.
