Recommendations

The following recommendations are critical and should be prioritized to address the severe security deficiencies identified in this assessment.

  1. Implement Multi-Factor Authentication (MFA) Universally:

    • Immediately enforce MFA for all email access.

    • Implement MFA for all computer logins across the organization.

    • Require MFA for access to all sensitive data systems and applications.

  2. Develop and Enforce an Acceptable Use Policy (AUP):

    • Draft a comprehensive AUP outlining acceptable and unacceptable use of company IT resources.

    • Distribute the AUP to all employees and obtain signed acknowledgements.

    • Integrate AUP review into employee onboarding and annual training.

  3. Establish a Comprehensive Security Awareness Training Program:

    • Implement mandatory security awareness training for all new hires during their onboarding process.

    • Conduct annual refresher security awareness training for all employees, covering current threats like phishing, social engineering, and data handling best practices.

    • Implement regular, simulated phishing campaigns to test employee vigilance.

  4. Conduct In-Depth Network Assessment:

    • Perform an authenticated network scan to gain full visibility of internal services and configurations.

    • Conduct a full vulnerability assessment on all internet-facing assets once a target IP is identified, to ensure no unintended services are exposed.

    • If the target [Target IP] was intentionally unreachable, ensure this is by design and not a misconfiguration.

  5. Develop and Implement Core Security Policies:

    • Beyond AUP, establish policies for incident response, data classification, access control, and vendor risk management.

    • Regularly review and update all security policies to adapt to evolving threats and organizational changes.
