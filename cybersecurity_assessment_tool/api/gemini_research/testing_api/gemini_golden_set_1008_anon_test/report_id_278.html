Recommendations

To mitigate the identified risks and improve the overall cybersecurity posture, the following actionable recommendations are provided. Prioritization is given to critical and high-severity risks.

Immediate Actions (Critical & High Risks)

  • Implement MFA for Email: Immediately enable and enforce Multi-Factor Authentication (MFA) for all organizational email accounts. This significantly reduces the risk of email account compromise, even if credentials are stolen.

  • Decommission or Secure FTP Service:

    • If the FTP service is not critically required for business operations, it should be immediately decommissioned and removed from internet exposure.

    • If FTP functionality is necessary, the vsftpd service must be upgraded to the latest stable and supported version. Implement secure configurations, including disabling anonymous access, enforcing strong authentication mechanisms (e.g., user accounts with strong passwords), and restricting access to trusted IP ranges. Consider migrating to SFTP or FTPS for encrypted data transfer.

  • Disable Anonymous FTP Access: As an interim measure, if the FTP service cannot be immediately decommissioned or upgraded, disable anonymous login access to prevent unauthorized access. Ensure all necessary access requires authenticated user accounts.

  • Upgrade Windows Operating Systems: Prioritize and execute a project to upgrade all workstations and servers running Windows 7 to a currently supported operating system (e.g., Windows 10 or 11). Establish and enforce a robust patch management program for all endpoints.

Mid-Term Actions (Medium Risks)

  • Onboarding Security Awareness Training: Implement mandatory cybersecurity awareness training for all new employees as a critical component of their onboarding process. This training should cover organizational security policies, common threat vectors (e.g., phishing, malware), safe browsing habits, and incident reporting procedures.

  • Annual Security Awareness Refresher: While already conducted annually for all employees, continuously review and update the content of this training to address emerging threats and reinforce best practices.

General Security Best Practices

  • Conduct regular vulnerability scans and periodic penetration tests to identify and address security weaknesses proactively.

  • Implement a robust patch management program for all systems, applications, and network devices to ensure timely application of security updates.

  • Regularly review and update cybersecurity policies, procedures, and incident response plans to adapt to the evolving threat landscape.

  • Deploy and configure Endpoint Detection and Response (EDR) solutions on all workstations and servers.

  • Ensure all critical systems are comprehensively logged and monitored for suspicious activity, with alerts configured for potential security incidents.
