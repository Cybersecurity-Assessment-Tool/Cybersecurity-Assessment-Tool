6. Recommendations

The following actionable recommendations are provided to mitigate the identified cybersecurity risks and strengthen [Organization Name]’s overall security posture. These are prioritized based on their assessed severity and potential impact.

  1. Immediate Action: Decommission or Secure Legacy FTP Service:

    • Priority: Critical

    • If the FTP service is not business-critical, it must be immediately decommissioned.

    • If required, urgently upgrade vsftpd to a current, supported, and patched version.

    • Disable anonymous FTP access immediately. Implement strong authentication mechanisms and enforce the use of secure protocols like SFTP or FTPS with TLS encryption for all file transfers.

    • Conduct a thorough forensic investigation of the system hosting the FTP service to check for signs of compromise, given the age and known backdoor vulnerability.

  2. Implement MFA for Sensitive Data Systems:

    • Priority: High

    • Mandate Multi-Factor Authentication (MFA) for all access points to systems and applications containing sensitive organizational data. This should include cloud services, databases, and internal critical business applications.

    • Provide comprehensive user training on the importance and proper usage of MFA.

  3. Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Priority: Medium

    • Create a comprehensive Employee Acceptable Use Policy (AUP) that clearly defines expectations for using company IT assets, handling sensitive information, internet usage, and overall security behavior.

    • Ensure all employees read, understand, and formally acknowledge the AUP upon hiring and annually thereafter.

    • Integrate AUP principles into new employee onboarding and ongoing security awareness training programs.

  4. Upgrade Outdated Operating Systems:

    • Priority: Medium

    • Prioritize and execute a project to upgrade all Windows 7 machines to currently supported operating systems (e.g., Windows 10 or 11).

    • For any systems that cannot be immediately upgraded, implement stringent compensating controls such as network isolation, advanced endpoint detection and response (EDR), and restricted user privileges.
