Recommendations

To address the identified risks and improve the overall cybersecurity posture of [Organization Name], the following recommendations are provided.

Immediate Technical Remediation

  • Decommission or Upgrade FTP Service:

    • If FTP is not absolutely essential, immediately decommission the service.

    • If FTP is required, upgrade to a current, patched, and supported version of an FTP/SFTP server. Consider using SFTP (SSH File Transfer Protocol) for secure file transfers, as plain FTP sends credentials and data in cleartext.

  • Disable Anonymous FTP Access:

    • As an interim measure while evaluating the service, immediately disable anonymous login to the FTP server to prevent unauthorized access.

  • Upgrade End-of-Life Operating Systems:

    • Urgently upgrade all workstations and servers running Windows 7 to a currently supported operating system (e.g., Windows 10 or Windows 11). This is a critical step to ensure systems receive necessary security updates.

Enhance Foundational Security Controls

  • Implement Multi-Factor Authentication (MFA) for Email:

    • Enable and enforce MFA for all email accounts. This significantly reduces the risk of account compromise even if passwords are stolen.

    • Extend MFA implementation to other critical services where applicable, beyond just sensitive data systems and computer logins.

  • Develop and Enforce an Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that clearly outlines employee responsibilities and acceptable behaviors regarding IT assets, data, and internet usage.

    • Ensure all employees read, understand, and formally acknowledge the AUP.

Improve Security Awareness and Training

  • Mandatory Security Awareness Training for New Employees:

    • Implement a mandatory security awareness training program for all new hires as part of their onboarding process.

    • This training should cover topics such as phishing, password hygiene, data handling, and company policies.

  • Conduct Annual Security Awareness Training for All Employees:

    • Establish a recurring annual security awareness training program for all employees.

    • Training content should be updated regularly to reflect current threat landscapes and organizational policies.

    • Consider phishing simulations to test and reinforce learned behaviors.

These recommendations, when implemented, will significantly strengthen the organization’s defenses against common cyber threats and foster a more secure operational environment.
