Recommendations

To address the identified risks and significantly improve the organization’s cybersecurity posture, the following actionable recommendations are provided:

  1. Implement MFA for All Login Points:

    • Mandate Multi-Factor Authentication (MFA) for all computer logins, in addition to email and sensitive data systems.

    • Evaluate and implement MFA for any other critical internal or external applications accessible to employees.

  2. Develop and Enforce an Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that clearly defines the rules and expectations for employee use of company IT assets, networks, and data.

    • Ensure all employees read, understand, and formally acknowledge the AUP upon hiring and periodically thereafter.

  3. Establish a Robust Security Awareness Training Program:

    • Implement mandatory security awareness training for all new employees as part of their onboarding process. This should cover common threats (e.g., phishing), password hygiene, and the AUP.

    • Conduct mandatory annual security awareness training for all employees. This training should be engaging, up-to-date with current threat landscapes, and include simulated phishing exercises.

  4. Upgrade and Maintain Software Versions:

    • Immediately plan and execute an upgrade of the Nginx web server to the latest stable release (e.g., Nginx 1.26.x or newer).

    • Establish a regular patch management program to ensure all operating systems, applications, and network devices are kept up-to-date with security patches and the latest stable versions.

  5. Implement Regular Vulnerability Scanning:

    • Conduct periodic vulnerability scans on all external and internal-facing assets to identify new weaknesses before they can be exploited.

    • Consider engaging in regular penetration testing to simulate real-world attacks and validate security controls.

  6. Develop Incident Response Plan:

    • Create and regularly update an Incident Response Plan (IRP) to ensure the organization can effectively detect, respond to, and recover from cybersecurity incidents.

    • Conduct tabletop exercises to test the IRP and ensure key personnel are familiar with their roles and responsibilities.
