6. Recommendations

To address the identified risks and improve the overall cybersecurity posture, the following actionable recommendations are provided:

  • Implement MFA for Computer Login: Mandate Multi-Factor Authentication (MFA) for all user logins to company computers. This significantly reduces the risk of unauthorized access even if credentials are stolen, forming a crucial layer of defense.

  • Implement MFA for Sensitive Data Systems: Extend MFA requirements to all systems that house or process sensitive organizational data. This adds a critical layer of security to the most valuable assets, protecting against unauthorized access and potential data breaches.

  • Introduce Mandatory Security Awareness Training for New Employees: Establish a mandatory, comprehensive security awareness training program for all new hires as part of their onboarding process. This training should cover common threats like phishing, malware, social engineering, and best practices for data handling.

  • Review and Secure Open Port 22 (SSH):

    • Necessity Assessment: Verify if the open port 22 is legitimate and absolutely necessary for business operations. If not, close it immediately.

    • Secure Configuration: If necessary, ensure SSH is configured securely by:

      • Disabling password authentication, enforcing strong key-based authentication.

      • Using unique, strong SSH keys and protecting them appropriately.

      • Restricting SSH access to specific trusted IP addresses or networks.

      • Implementing a robust logging and monitoring solution for all SSH access attempts and anomalies.

      • Keeping the SSH server software fully up-to-date with the latest security patches to mitigate known vulnerabilities.

      • Consider changing the default SSH port (though this is a minor security enhancement, not a primary defense).

  • Regular Security Audits: Conduct regular internal and external security audits, including comprehensive vulnerability scanning and penetration testing, to continuously identify and remediate potential weaknesses in the environment.

  • Maintain Comprehensive Asset Inventory: Ensure a detailed and up-to-date inventory of all network-connected devices, applications, and services is maintained. This facilitates better management, security oversight, and incident response capabilities.
