Recommendations

Based on the assessment, the following recommendations are provided to enhance the organization’s cybersecurity posture:

RDP Exposure

  • Immediate Action (Easy Fix): Implement firewall rules to restrict access to port 3389 (RDP) to only trusted internal networks or specific, known administrator IP addresses. If remote access is not absolutely essential, close the port entirely.

  • Long-Term Solution:

    • Deploy a Virtual Private Network (VPN) solution that requires authentication for all remote RDP access. RDP should only be accessible once connected to the VPN.

    • Enforce strong, complex passwords for all accounts with RDP access, and implement account lockout policies to deter brute-force attempts.

    • Enable Network Level Authentication (NLA) for RDP connections to add an additional layer of security before a full RDP session is established.

Lack of Multi-Factor Authentication (MFA) for Email

  • Implement Mandatory MFA: Deploy and enforce mandatory MFA for all email accounts across the organization. Prioritize solutions using authenticator apps (e.g., Microsoft Authenticator, Google Authenticator) or hardware security keys over SMS-based MFA.

  • Security Awareness Training: Conduct regular, updated security awareness training for all employees, specifically focusing on the risks of phishing, social engineering, and the importance of MFA.

Absence of Employee Acceptable Use Policy (AUP)

  • Develop and Implement AUP: Create a comprehensive Employee Acceptable Use Policy that clearly outlines employee responsibilities regarding the use of company IT resources, data handling, internet usage, email etiquette, and general cybersecurity best practices.

  • Policy Acknowledgment and Training: Ensure all current and new employees review and formally acknowledge the AUP. Integrate AUP review into annual security awareness training and new employee onboarding processes.

  • Regular Review: Establish a schedule for regular review and updates of the AUP to ensure it remains current with evolving technology and threat landscapes.
