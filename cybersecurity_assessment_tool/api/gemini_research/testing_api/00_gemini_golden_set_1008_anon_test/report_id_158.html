Recommendations

The following recommendations are provided to address the identified risks and enhance the overall cybersecurity posture of [Organization Name]. Recommendations are prioritized based on severity.

  1. Upgrade/Replace Outdated FTP Service:

    • Immediately upgrade the service to a current, patched, and supported version.

    • Alternatively, if FTP functionality is not essential, disable the service entirely.

    • Consider replacing FTP with more secure protocols like SFTP or FTPS for file transfers.

  2. Disable Anonymous FTP Access:

    • Configure the FTP server to disable anonymous logins.

    • Implement strong user authentication and granular access controls for any necessary FTP accounts.

  3. Implement MFA for Sensitive Data Systems:

    • Mandate Multi-Factor Authentication for all access to systems that store, process, or transmit sensitive organizational data. This significantly reduces the risk of credential compromise.

  4. Develop and Enforce an Acceptable Use Policy (AUP):

    • Create a comprehensive Employee Acceptable Use Policy that clearly defines the rules for using company IT resources, data, and communication systems.

    • Ensure all employees read, understand, and formally acknowledge the AUP upon hiring and periodically thereafter.

  5. Mandatory Security Awareness Training for New Employees:

    • Integrate mandatory security awareness training into the onboarding process for all new hires.

    • This training should cover topics such as phishing, social engineering, password hygiene, and safe internet practices.

  6. Upgrade Outdated Operating Systems:

    • Plan and execute a migration strategy to upgrade all workstations running Windows 7 to a currently supported operating system (e.g., Windows 10 or Windows 11).

    • Implement a regular patching and update schedule for all operating systems.
