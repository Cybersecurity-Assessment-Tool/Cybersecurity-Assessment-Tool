6. Recommendations

Based on the assessment findings, the following recommendations are provided to enhance the organization’s cybersecurity posture:

  • Implement Multi-Factor Authentication (MFA):

    • Immediately implement mandatory MFA for all email accounts across the organization. This is a critical step to mitigate phishing and credential compromise risks.

    • Deploy MFA for all computer logins, especially for administrative accounts and users with access to sensitive systems.

  • Develop and Enforce Policies:

    • Develop and formally implement an Employee Acceptable Use Policy (AUP). Ensure all employees acknowledge, understand, and adhere to this policy, providing clear guidelines for IT resource usage.

  • Enhance Security Awareness Training:

    • Introduce mandatory security awareness training for all new employees during their onboarding process to establish a baseline of security knowledge from day one.

    • Continuously reinforce and update annual security awareness training with current threat landscapes and practical examples (e.g., simulated phishing exercises).

  • Investigate Open Port 80:

    • Conduct a thorough investigation of the service running on port 80 on the target. Determine its purpose, necessity, and whether it should be exposed.

    • Ensure the service is patched to the latest version, securely configured, and has appropriate access controls in place.

    • If the service is not required for external access, restrict its accessibility to internal networks or specific trusted IP ranges.

  • Regular Security Audits:

    • Conduct regular vulnerability scans and penetration tests to proactively identify and remediate technical vulnerabilities before they can be exploited.
