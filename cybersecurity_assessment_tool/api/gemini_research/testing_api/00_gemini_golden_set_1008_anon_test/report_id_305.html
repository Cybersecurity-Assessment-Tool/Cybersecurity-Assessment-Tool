Identified Control Gaps:

  • MFA for Sensitive Data Systems: The absence of Multi-Factor Authentication for sensitive data systems significantly elevates the risk of unauthorized access, especially if user credentials are compromised through phishing or other attacks.

  • Employee Acceptable Use Policy (AUP): Without a clear AUP, employees may not understand their responsibilities regarding the use of company IT resources and data, increasing the potential for misuse, data leakage, and compliance violations.

Recommendations

To significantly enhance the organization’s cybersecurity posture and mitigate the identified risks, the following actionable recommendations are provided:

  1. Mitigate RDP Exposure:

    • Immediate Action: Close RDP port 3389 from direct internet access at the perimeter firewall. This is a critical step to remove an immediate high-risk attack vector.

    • Long-term Solution: Implement a Virtual Private Network (VPN) solution to ensure that RDP access is only permissible to authorized users who have first authenticated via the VPN. Alternatively, deploy a secure jump box or bastion host for RDP connections.

    • Implement strong password policies and account lockout thresholds for all accounts with RDP access.

  2. Implement MFA for Sensitive Data Systems:

    • Mandate Multi-Factor Authentication (MFA) for all access to sensitive data systems and applications. This adds a crucial layer of security, significantly reducing the risk of unauthorized access even if credentials are stolen.

  3. Develop and Enforce an Acceptable Use Policy (AUP):

    • Create a comprehensive Acceptable Use Policy that clearly defines appropriate use of organizational IT resources, data, and internet access. The AUP should outline prohibited activities and consequences of violations.

    • Ensure all employees review, understand, and acknowledge the AUP, ideally upon hiring and annually thereafter, to foster a culture of security responsibility.

  4. Establish Annual Security Awareness Training Program:

    • Implement a mandatory, ongoing security awareness training program for all employees, not just new hires. This training should be conducted at least once per year.

    • The curriculum should cover current cybersecurity threats (e.g., advanced phishing techniques, ransomware, social engineering), organizational security policies, and best practices for data handling and secure computing.
