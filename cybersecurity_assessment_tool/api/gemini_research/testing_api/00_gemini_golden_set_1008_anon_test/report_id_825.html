Analysis of Control Gaps

The questionnaire results indicate several significant weaknesses in the organization’s security posture:

  • Insufficient MFA Implementation: MFA is critical for preventing unauthorized access, yet it is not universally enforced for email access or computer logins. This leaves a major vector open for credential-based attacks.

  • Lack of Acceptable Use Policy (AUP): The absence of an AUP means employees may not have clear guidelines regarding the acceptable use of company IT resources, increasing the risk of misuse or security policy violations.

  • Incomplete Security Awareness Training: While annual training is conducted, the lack of dedicated security awareness training for new employees means they may be operating without foundational knowledge of security best practices from day one.

Recommendations

To address the identified risks and strengthen the cybersecurity posture of , the following recommendations are provided:

Immediate Actions (High Priority)

  • Implement MFA for Email Access: Mandate Multi-Factor Authentication for all email accounts. Prioritize administrative and privileged accounts.

  • Implement MFA for Computer Login: Enforce MFA for all user and administrator logins to company computers and network resources.

  • Develop and Implement an Acceptable Use Policy (AUP): Create a clear, comprehensive AUP that outlines proper usage of all company IT assets, software, and networks. Ensure all employees acknowledge and understand this policy.

Short-Term Actions (Medium Priority)

  • Establish Security Awareness Training for New Employees: Implement mandatory security awareness training as part of the onboarding process for all new hires to instill good security practices from the start.

  • Review Port 8080 Configuration: Although deemed "secured" or "false positive" by existing risk data, a periodic review of the application running on Port 8080 () should be conducted to ensure its security posture remains valid and that no new vulnerabilities have emerged. Verify that access controls are robust and that the data handled is appropriately protected.

Long-Term Actions (Ongoing)

  • Regular Policy Review: Periodically review and update all security policies, including the AUP and security awareness training materials, to reflect evolving threats and organizational changes.

  • Continuous Vulnerability Management: Implement a program for regular vulnerability scanning and penetration testing to proactively identify and address technical weaknesses across all systems.

  • Security Culture Development: Foster a strong security culture through ongoing communication, reminders, and making security an integral part of daily operations.
