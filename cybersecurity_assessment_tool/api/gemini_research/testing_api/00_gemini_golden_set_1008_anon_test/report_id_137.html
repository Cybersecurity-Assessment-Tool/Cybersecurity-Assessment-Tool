Recommendations

The following recommendations are provided to mitigate the identified risks and enhance the overall cybersecurity posture. Prioritization should be given to Critical risks.

Critical Risks

  • RDP Exposure:

    • Immediate Fix: Close port 3389 (RDP) on all external interfaces. If remote access to specific systems is essential, restrict access to specific trusted source IP addresses via firewall rules.

    • Long-Term Solution: Implement a Virtual Private Network (VPN) solution for all remote access to internal resources, including RDP. Users should establish a VPN connection first before attempting to access internal RDP servers.

    • Mitigation (if closure/VPN not immediate): Ensure strong, unique passwords are enforced for all RDP accounts, apply account lockout policies, and implement robust logging and monitoring for RDP login attempts to detect suspicious activity. Regularly patch RDP services.

  • Lack of MFA for Email Access:

    • Recommendation: Immediately implement and enforce Multi-Factor Authentication (MFA) for all user accounts accessing organizational email. This should be a mandatory requirement for all users.

  • Lack of MFA for Sensitive Data Systems:

    • Recommendation: Implement and enforce Multi-Factor Authentication (MFA) for all access to systems containing sensitive organizational or client data. Access to these critical assets should always require a second factor of authentication.

High Risks

  • Absence of an Employee Acceptable Use Policy (AUP):

    • Recommendation: Develop and formally implement a comprehensive Acceptable Use Policy (AUP). This policy should clearly outline expectations for employees regarding the proper use of IT resources, data handling, internet usage, software installation, and general security responsibilities. Ensure all employees acknowledge, understand, and comply with the policy.

  • Inadequate Annual Security Awareness Training:

    • Recommendation: Establish and implement a program for mandatory annual security awareness training for all employees, in addition to initial new-hire training. This recurrent training should cover current threats such as sophisticated phishing attacks, social engineering tactics, password best practices, data handling procedures, and incident reporting.
