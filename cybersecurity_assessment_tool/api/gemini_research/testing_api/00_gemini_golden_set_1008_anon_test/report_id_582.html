Recommendations

To mitigate the identified risks and enhance the overall cybersecurity posture, the following actionable recommendations are provided:

Policy and Training Enhancements

  1. Develop and Implement an Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that clearly defines appropriate use of all organizational IT resources, data, and systems.

    • Ensure the AUP is communicated to all employees, acknowledged, and regularly reviewed.

    • Integrate AUP review into new hire onboarding and annual compliance training.

  2. Implement Mandatory Annual Security Awareness Training for All Employees:

    • Develop or procure a robust security awareness training program covering topics such as phishing, social engineering, password hygiene, data handling, and incident reporting.

    • Ensure all employees complete this training at least once per year.

    • Track completion rates and reinforce key messages throughout the year.

Technical Security Enhancements

  1. Verify and Address Unencrypted Web Server Risk:

    • Investigate the pre-existing "Unencrypted Web Server" risk. Confirm if this vulnerability was historical or if an unencrypted web service might be present on other internal or external assets not covered by the current scan.

    • Ensure that no unencrypted web services (HTTP on Port 80) are publicly exposed.

    • For any web services requiring public access, enforce the use of HTTPS with valid, up-to-date SSL/TLS certificates. Redirect all HTTP traffic to HTTPS.

    • Regularly scan and audit all public-facing web servers for proper encryption configuration.
