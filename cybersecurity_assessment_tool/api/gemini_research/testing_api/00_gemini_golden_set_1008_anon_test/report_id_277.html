6. Recommendations

To address the identified risks and enhance the organization’s cybersecurity posture, the following recommendations are provided:

  • Implement Multi-Factor Authentication (MFA):

    • Mandate MFA for all employee computer logins.

    • Implement MFA for access to all sensitive data systems and applications.

  • Develop and Enforce an Acceptable Use Policy (AUP):

    • Create a clear and comprehensive AUP outlining acceptable use of organizational IT assets, data, and internet resources.

    • Distribute the AUP to all employees and require acknowledgment of understanding and compliance.

  • Enhance Security Awareness Training:

    • Establish a mandatory annual security awareness training program for all employees, covering current threats like phishing, social engineering, and data handling best practices.

    • Supplement annual training with regular security bulletins or brief awareness campaigns.

  • Upgrade Nginx Server:

    • Plan and execute an upgrade of the Nginx web server to the latest stable version (e.g., Nginx 1.26.x or the latest mainline version).

    • Ensure proper testing of applications hosted on the Nginx server post-upgrade to prevent service disruption.

    • Implement a regular patching schedule for all web servers and critical infrastructure.
