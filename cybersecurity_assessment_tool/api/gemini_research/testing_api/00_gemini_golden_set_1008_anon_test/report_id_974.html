6. Recommendations

The following recommendations are provided to address the identified risks and improve the organization’s cybersecurity posture. Immediate action is advised for critical and high-priority items.

6.1. Critical Recommendations

  • Decommission or Secure FTP Server Immediately:

    • Evaluate if the FTP server is still required. If not, decommission it immediately.

    • If essential, upgrade to a current, supported version of an FTP server (e.g., modern vsftpd, Pure-FTPd, ProFTPD) and ensure it is securely configured.

    • Disable anonymous FTP access. Implement strong authentication methods and restrict access to only necessary users and IP addresses.

    • Consider alternative secure file transfer protocols like SFTP or SCP.

6.2. High Priority Recommendations

  • Implement MFA for Computer Logins:

    • Deploy and enforce Multi-Factor Authentication (MFA) for all computer logins, especially for administrative accounts and systems.

    • This significantly reduces the risk of credential theft leading to network compromise.

  • Establish Comprehensive Security Awareness Training:

    • Implement mandatory security awareness training for all new employees during their onboarding process.

    • Conduct annual (or more frequent) mandatory security awareness training for all employees, covering topics such as phishing, social engineering, password hygiene, and data handling.

  • Upgrade End-of-Life Operating Systems:

    • Develop and execute a plan to upgrade all computers running Windows 7 to a currently supported operating system (e.g., Windows 10/11, or a supported Linux distribution).

    • Ensure all operating systems and applications are regularly patched and updated.
