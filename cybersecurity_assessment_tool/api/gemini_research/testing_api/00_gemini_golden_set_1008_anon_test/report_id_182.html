Recommendations

The following recommendations are provided to mitigate the identified risks and improve the overall cybersecurity posture. Prioritization is given to critical and high-severity findings.

Immediate Actions (Critical & High Risks)

  • Decommission/Secure FTP Service (Critical):

    • Immediately disable or remove the FTP service on if it is not business-critical.

    • If FTP is required, upgrade to a current, patched version of a secure file transfer protocol (e.g., SFTP, FTPS) and ensure anonymous access is strictly disabled. Implement strong authentication and authorization.

  • Implement MFA for Email (High):

    • Mandate multi-factor authentication for all email accounts across the organization. This significantly reduces the risk of successful phishing attacks and account compromise.

  • Develop and Enforce an Acceptable Use Policy (High):

    • Create a comprehensive Employee Acceptable Use Policy (AUP) outlining the responsible and secure use of organizational IT assets, data, and internet access.

    • Distribute the AUP to all employees, obtain signed acknowledgments, and integrate it into new hire onboarding.

Mid-Term Actions (Medium Risks)

  • Conduct Annual Security Awareness Training (Medium):

    • Establish a program for mandatory annual security awareness training for all employees, covering topics such as phishing, social engineering, data handling, and password best practices.

    • Upgrade Outdated Operating Systems (Medium):

      • Develop a phased plan to upgrade all workstations running Windows 7 (or other End-of-Life operating systems) to a currently supported version (e.g., Windows 10/11) to ensure continued security patching and support. This may involve hardware refresh where necessary.

General Best Practices

  • Conduct regular vulnerability scans and penetration tests to identify and remediate security weaknesses proactively.

  • Implement a robust patch management program for all operating systems, applications, and network devices.

  • Review and harden all internet-facing services to minimize attack surface.
