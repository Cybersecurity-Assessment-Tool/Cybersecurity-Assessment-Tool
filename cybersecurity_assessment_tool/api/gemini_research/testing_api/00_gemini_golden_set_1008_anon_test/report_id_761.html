Recommendations

To address the identified risks and enhance the overall cybersecurity posture, the following recommendations are provided:

  1. Mitigate RDP Exposure:

    • Immediately restrict RDP access (port 3389) to only trusted IP addresses or via a Virtual Private Network (VPN) connection.

    • Implement strong, complex passwords for all RDP accounts and enforce account lockout policies.

    • Consider disabling RDP entirely if it is not strictly necessary for business operations.

    • Ensure all RDP servers are fully patched and up-to-date with the latest security updates.

  2. Implement Multi-Factor Authentication (MFA):

    • Mandate MFA for all user logins to company computers and endpoints.

    • Enforce MFA for access to all sensitive data systems and applications.

  3. Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Create a formal AUP that clearly outlines acceptable and unacceptable use of company IT resources.

    • Disseminate the AUP to all employees and incorporate it into new employee onboarding and annual security awareness training.

  4. Regular Security Audits and Vulnerability Assessments:

    • Conduct periodic external and internal vulnerability scans and penetration tests to identify new weaknesses.

    • Regularly review and update security policies and procedures to adapt to evolving threat landscapes.
