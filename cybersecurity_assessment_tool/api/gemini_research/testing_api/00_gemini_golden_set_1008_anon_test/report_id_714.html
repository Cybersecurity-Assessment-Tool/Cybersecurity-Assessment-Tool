Recommendations

To mitigate the identified risks and improve the overall cybersecurity posture of [Organization Name], the following recommendations are provided:

  1. Prioritize Multi-Factor Authentication (MFA) Implementation:

    • Email Access: Immediately implement and enforce MFA for all email accounts. This is a critical defense against phishing and credential theft.

    • Computer Login: Deploy MFA for all computer logins, especially for devices accessing sensitive organizational data.

  2. Develop and Enforce an Acceptable Use Policy (AUP):

    • Create a comprehensive AUP that outlines acceptable use of organizational IT resources, internet, email, and mobile devices.

    • Ensure all employees review and formally acknowledge the AUP upon hiring and periodically thereafter.

  3. Mandate New Employee Security Awareness Training:

    • Integrate mandatory security awareness training into the onboarding process for all new hires.

    • This training should cover topics such as phishing, password hygiene, data handling, and organizational policies.

  4. Review and Validate Existing Technical Risks:

    • Address the "Unencrypted Web Server" risk. While the recent scan indicates Port 80 is closed, investigate why this risk was previously identified.

    • If a web server is intended to be publicly accessible, ensure it is configured to use HTTPS (SSL/TLS) exclusively on Port 443, and that Port 80 remains closed or redirects securely.

    • If no public web server is intended, ensure Port 80 remains permanently closed and blocked at the firewall level.

  5. Maintain and Enhance Security Awareness Training:

    • Continue with annual security awareness training for all employees, ensuring content is updated to reflect current threat landscapes.

  6. Conduct Regular, Comprehensive Network Scans:

    • Implement a schedule for periodic vulnerability scanning (both internal and external) to proactively identify and address technical weaknesses.

    • Consider authenticated scans for a deeper insight into system configurations.
