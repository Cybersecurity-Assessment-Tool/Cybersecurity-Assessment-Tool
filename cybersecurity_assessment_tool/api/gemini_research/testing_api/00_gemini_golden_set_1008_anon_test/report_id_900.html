Recommendations

To address the identified critical gaps and enhance the overall cybersecurity posture, the following recommendations are provided:

  1. Implement Multi-Factor Authentication (MFA) for All Computer Logins:

    • Action: Deploy and enforce MFA for all user accounts logging into company computers and workstations.

    • Benefit: Significantly reduces the risk of unauthorized access even if passwords are compromised.

  2. Implement Multi-Factor Authentication (MFA) for Sensitive Data Systems:

    • Action: Prioritize and implement MFA for all applications and systems that store, process, or transmit sensitive organizational or customer data.

    • Benefit: Provides a critical layer of defense against data breaches and unauthorized access to high-value assets.

  3. Develop and Implement an Employee Acceptable Use Policy (AUP):

    • Action: Draft, disseminate, and enforce a clear AUP that outlines acceptable use of company IT resources, data handling, and internet browsing.

    • Benefit: Establishes clear expectations for employee conduct, minimizes misuse of resources, and reduces legal and security risks.

  4. Establish a Mandatory Annual Security Awareness Training Program:

    • Action: Develop and implement a program requiring all employees to complete security awareness training at least once per year. Training content should cover current threats such as phishing, social engineering, ransomware, and secure data handling practices.

    • Benefit: Keeps employees informed about evolving threats, reinforces secure behaviors, and strengthens the human firewall against cyberattacks.

  5. Re-evaluate Technical Posture with Valid Data:

    • Action: Conduct a new network scan and ensure the results are provided in a valid, parseable JSON format. Similarly, provide any existing risk register data in a valid JSON format.

    • Benefit: Allows for a comprehensive technical analysis, identification of specific system vulnerabilities, and a complete picture of the organization’s risk landscape.
