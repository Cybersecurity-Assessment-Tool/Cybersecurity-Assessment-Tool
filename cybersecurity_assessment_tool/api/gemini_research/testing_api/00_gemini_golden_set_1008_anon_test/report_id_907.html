6. Recommendations

The following recommendations are provided to address the identified security gaps and risks.

  • Missing MFA for Email: Implement and enforce multi-factor authentication (MFA) for all email accounts across the organization.

  • Lack of Employee Acceptable Use Policy (AUP): Develop and formally implement a comprehensive Employee Acceptable Use Policy (AUP). Ensure all employees review, understand, and acknowledge it regularly.

  • Inadequate Annual Security Awareness Training: Establish and conduct mandatory annual (at least) security awareness training for all employees, covering current threats and best practices.

  • Open SSH Port (22/tcp): Identify the specific SSH service version running on port 22 and ensure it is up-to-date and securely configured. Implement strong password policies or use key-based authentication. Limit SSH access to trusted IP addresses using firewall rules, disable root login, disable unused authentication methods, and implement rate limiting for connection attempts.
