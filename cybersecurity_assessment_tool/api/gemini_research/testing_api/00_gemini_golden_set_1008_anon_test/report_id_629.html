Recommendations

Based on the assessment, the following recommendations are provided to enhance the organization’s cybersecurity posture:

Immediate Actions:

  • Secure RDP Access:

    • Immediately review and restrict external access to RDP (port 3389). Consider disabling direct external access entirely, or placing it behind a VPN.

    • If external RDP access is absolutely necessary, ensure strong, unique passwords are enforced, Network Level Authentication (NLA) is enabled, and IP restrictions are in place.

    • Implement a dedicated jump server or bastion host for all remote administration.

Strategic Actions:

  • Implement Multi-Factor Authentication (MFA):

    • Deploy MFA for all email accounts to significantly reduce the risk of account compromise.

    • Extend MFA implementation to all sensitive data systems and applications to protect critical assets.

  • Enhance Security Awareness Training:

    • Establish a mandatory annual (at minimum) security awareness training program for all employees.

    • Supplement training with regular simulated phishing campaigns to reinforce learned behaviors and identify areas for further education.

  • Review and Update Policies:

    • Ensure the employee acceptable use policy is regularly reviewed and updated to reflect current threats, technologies, and organizational practices.
