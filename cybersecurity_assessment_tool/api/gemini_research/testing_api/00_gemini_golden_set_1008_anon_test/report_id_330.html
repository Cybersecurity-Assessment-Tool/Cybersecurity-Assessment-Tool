Recommendations

Based on the assessment, the following recommendations are provided to significantly enhance [Organization Name]’s cybersecurity posture:

  1. Implement Multi-Factor Authentication (MFA):

    • Action: Extend MFA requirements to all employee computer logins across the organization.

    • Action: Implement MFA for all sensitive data systems to significantly reduce the risk of credential-based attacks and unauthorized access.

    • Benefit: Greatly reduces the impact of stolen or weak passwords.

  2. Develop and Implement an Acceptable Use Policy (AUP):

    • Action: Create a clear, concise AUP outlining appropriate use of organizational IT resources, data, and internet access.

    • Action: Ensure all employees review and acknowledge the AUP upon hiring and annually thereafter.

    • Benefit: Establishes clear guidelines for employees, reducing misuse and legal risks.

  3. Establish Comprehensive Security Awareness Training:

    • Action: Develop and implement mandatory security awareness training for all new employees during their onboarding process.

    • Action: Conduct annual, mandatory security awareness training for all employees, covering topics such as phishing, social engineering, password hygiene, and incident reporting.

    • Benefit: Equips employees with the knowledge to identify and resist cyber threats, turning them into a strong line of defense.

  4. Address Unencrypted Web Server Risk:

    • Action: Investigate the discrepancy between the existing risk report ("Port 80 is open") and the current scan findings (Port 80 closed). Determine if the service was moved, decommissioned, or if the original risk assessment was based on different conditions.

    • Action: If a web service is intended to be available, ensure it is either properly secured and closed off to the public internet, or upgraded to use HTTPS (Port 443) with a valid SSL/TLS certificate to encrypt all traffic.

    • Benefit: Prevents data interception and enhances user trust and data integrity.
