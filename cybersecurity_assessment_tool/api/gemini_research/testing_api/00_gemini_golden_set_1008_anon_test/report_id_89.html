Recommendations

The following recommendations are provided to mitigate the identified risks and improve the overall cybersecurity posture of [Organization Name].

  1. Address RDP Exposure:

    • Immediate Fix: Restrict direct internet access to RDP (port 3389). Implement a Virtual Private Network (VPN) solution as a mandatory gateway for all remote RDP access, ensuring users connect to the VPN first before accessing internal resources.

    • Long-Term Fix: Implement network-level access controls (e.g., firewall rules) to limit RDP access only from trusted IP addresses or through a secure bastion host. Ensure all RDP connections utilize strong, complex passwords and, where possible, integrate with an MFA solution.

    • Maintenance: Regularly patch and update the RDP service and underlying operating system to mitigate known vulnerabilities.

  2. Implement MFA for Computer Logins:

    • Action: Deploy and enforce Multi-Factor Authentication (MFA) for all computer logins, especially for administrative accounts and critical systems.

    • Considerations: Evaluate MFA solutions compatible with the current infrastructure (e.g., Windows Hello for Business, third-party MFA providers, smart cards, FIDO2 security keys). This significantly reduces the risk of unauthorized access even if credentials are compromised.

  3. Establish Security Awareness Training for New Employees:

    • Action: Develop and implement a mandatory security awareness training program specifically for all new employees during their onboarding process.

    • Content: The training should cover essential cybersecurity topics such as phishing identification, secure password hygiene, data handling policies, identifying social engineering tactics, and incident reporting procedures.

    • Benefit: This ensures that all employees are equipped with fundamental security knowledge from day one, reducing human-factor vulnerabilities.
