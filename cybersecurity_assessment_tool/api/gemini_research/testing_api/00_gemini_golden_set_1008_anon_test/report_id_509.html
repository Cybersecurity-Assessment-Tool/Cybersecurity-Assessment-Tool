Recommendations

The following recommendations are provided to address the identified security gaps and mitigate the associated risks. These are actionable steps designed to enhance the organization’s cybersecurity posture.

1. Secure Remote Desktop Protocol (RDP) Access

  • Immediate Action:

    1. Restrict RDP access to trusted IP addresses only using firewall rules.

    2. Implement a Virtual Private Network (VPN) solution and require all RDP connections to originate from a VPN-secured session.

    3. Enforce strong, multi-factor authentication (MFA) for all RDP connections, even when accessed via VPN.

  • Long-Term Strategy:

    1. Evaluate the necessity of external RDP access. If internal access is sufficient, disable external exposure entirely.

    2. Consider implementing secure alternatives such as a bastion host or a Just-In-Time (JIT) access solution to further minimize exposure.

2. Implement and Enforce Multi-Factor Authentication (MFA)

  • Immediate Action:

    1. Deploy and enforce MFA for all organizational email accounts.

    2. Implement and mandate MFA for all user and administrative computer logins.

  • Review and Expansion:

    1. Conduct a comprehensive review to ensure MFA is consistently enforced for all sensitive data systems and applications, validating the existing "Yes" answer for sensitive systems and identifying any gaps.

    2. Explore extending MFA to other critical systems and cloud services where applicable.

3. Enhance Security Awareness Training

  • Immediate Action:

    1. Develop and implement a mandatory, comprehensive annual security awareness training program for all employees. This training should cover current threats such as phishing, ransomware, social engineering tactics, and safe internet practices.

  • Ongoing Strategy:

    1. Supplement annual training with regular, brief security reminders (e.g., monthly newsletters, short video clips).

    2. Conduct periodic simulated phishing campaigns to test employee vigilance and provide targeted remediation training.

    3. Ensure new employees receive initial security awareness training during their onboarding process.
