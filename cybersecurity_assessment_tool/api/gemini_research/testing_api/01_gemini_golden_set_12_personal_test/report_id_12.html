Recommendations

  • Rationale: A valid and repeatable data collection process is the foundation of any security program. Without reliable data, risk cannot be accurately measured or managed.

  • Action: While the data collection issue is being resolved, manually verify that MFA is enforced for all users on the primary email system ([Domain]).

Priority 3: Re-initiate Security Assessment (High)

  • Action: Once the data integrity issues are resolved, conduct a new, full assessment using valid data.

  • Rationale: To gain a true understanding of the organization’s security posture, a complete analysis based on accurate network, control, and risk data is required.
